# Prompt 输入消融实验结果

| 变体 | 平均有效率 | 高约束 API | 中约束 API | 低约束 API | 唯一程序数 |
|------|------------|------------|------------|------------|------------|
| A. 无文档输入 | 93.1% | 86.6% | 94.7% | 99.0% | 1249 |
| B. 原始文档输入 | 89.6% | 77.7% | 93.7% | 98.6% | 1628 |
| C. 结构化提示蒸馏 | 90.2% | 78.2% | 94.8% | 99.0% | 1593 |

> 难度分组: 高约束 = fuzz4all.txt 的前 35 个 API; 中约束 = 第 36–70; 低约束 = 第 71–末尾。
> 平均有效率 = 各 API (filter+success 通过样本数 / 总采样数) 的算术平均。
> 唯一程序数 = 各 API 在该变体下 valid 程序经 ast unparse 归一化后去重数量, 跨 API 求和。
