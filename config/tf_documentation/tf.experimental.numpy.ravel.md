# tf.experimental.numpy.ravel

TensorFlow variant of NumPy's `ravel`.

```
tf.experimental.numpy.ravel(
    a
)
```

Unsupported arguments: `order`.

See the NumPy documentation for [`numpy.ravel`](https://numpy.org/doc/stable/reference/generated/numpy.ravel.html).
