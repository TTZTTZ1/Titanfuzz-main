TensorFlow variant of NumPy's `sort`.

```
tf.experimental.numpy.sort(
    a, axis=-1, kind='quicksort', order=None
)
```

See the NumPy documentation for [`numpy.sort`](https://numpy.org/doc/stable/reference/generated/numpy.sort.html).
