# tf.experimental.numpy.nextafter

**Source URL:** [https://tensorflow.google.cn/api_docs/python/tf/experimental/numpy/nextafter](https://tensorflow.google.cn/api_docs/python/tf/experimental/numpy/nextafter)

---

[View source on GitHub](https://github.com/tensorflow/tensorflow/blob/v2.16.1/tensorflow/python/ops/numpy_ops/np_math_ops.py#L408-L411) |

TensorFlow variant of NumPy's `nextafter`.

```
tf.experimental.numpy.nextafter(
    x1, x2
)
```

Unsupported arguments: `out`, `where`, `casting`, `order`, `dtype`, `subok`, `signature`, `extobj`.

See the NumPy documentation for [`numpy.nextafter`](https://numpy.org/doc/stable/reference/generated/numpy.nextafter.html).
