TensorFlow variant of NumPy's `concatenate`.

```
tf.experimental.numpy.concatenate(
    arys, axis=0
)
```

See the NumPy documentation for [`numpy.concatenate`](https://numpy.org/doc/stable/reference/generated/numpy.concatenate.html).
