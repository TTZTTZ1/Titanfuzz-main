TensorFlow variant of NumPy's [`random.randint`](https://tensorflow.google.cn/api_docs/python/tf/keras/random/randint).

```
tf.experimental.numpy.random.randint(
    low,
    high=None,
    size=None,
    dtype=tf.experimental.numpy.int64
)

tf.experimental.numpy.int64
```

See the NumPy documentation for [`numpy.random.randint`](https://numpy.org/doc/stable/reference/generated/numpy.random.randint.html).
