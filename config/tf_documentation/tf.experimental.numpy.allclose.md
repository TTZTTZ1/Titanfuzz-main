# tf.experimental.numpy.allclose

**Source URL:** [https://tensorflow.google.cn/api_docs/python/tf/experimental/numpy/allclose](https://tensorflow.google.cn/api_docs/python/tf/experimental/numpy/allclose)

---

[View source on GitHub](https://github.com/tensorflow/tensorflow/blob/v2.16.1/tensorflow/python/ops/numpy_ops/np_math_ops.py#L542-L547) |

TensorFlow variant of NumPy's `allclose`.

```
tf.experimental.numpy.allclose(
    a, b, rtol=1e-05, atol=1e-08, equal_nan=False
)
```

See the NumPy documentation for [`numpy.allclose`](https://numpy.org/doc/stable/reference/generated/numpy.allclose.html).
