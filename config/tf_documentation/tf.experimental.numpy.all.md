# tf.experimental.numpy.all

**Source URL:** [https://tensorflow.google.cn/api_docs/python/tf/experimental/numpy/all](https://tensorflow.google.cn/api_docs/python/tf/experimental/numpy/all)

---

[View source on GitHub](https://github.com/tensorflow/tensorflow/blob/v2.16.1/tensorflow/python/ops/numpy_ops/np_array_ops.py#L420-L424) |

TensorFlow variant of NumPy's `all`.

```
tf.experimental.numpy.all(
    a, axis=None, keepdims=None
)
```

Unsupported arguments: `out`, `where`.

See the NumPy documentation for [`numpy.all`](https://numpy.org/doc/stable/reference/generated/numpy.all.html).
