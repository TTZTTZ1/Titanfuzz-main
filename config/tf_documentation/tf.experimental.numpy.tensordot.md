TensorFlow variant of NumPy's `tensordot`.

```
tf.experimental.numpy.tensordot(
    a, b, axes=2
)
```

See the NumPy documentation for [`numpy.tensordot`](https://numpy.org/doc/stable/reference/generated/numpy.tensordot.html).
