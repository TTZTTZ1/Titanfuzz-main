# tf.experimental.numpy.array_equal

**Source URL:** [https://tensorflow.google.cn/api_docs/python/tf/experimental/numpy/array_equal](https://tensorflow.google.cn/api_docs/python/tf/experimental/numpy/array_equal)

---

[View source on GitHub](https://github.com/tensorflow/tensorflow/blob/v2.16.1/tensorflow/python/ops/numpy_ops/np_math_ops.py#L1155-L1171) |

TensorFlow variant of NumPy's `array_equal`.

```
tf.experimental.numpy.array_equal(
    a1, a2
)
```

Unsupported arguments: `equal_nan`.

See the NumPy documentation for [`numpy.array_equal`](https://numpy.org/doc/stable/reference/generated/numpy.array_equal.html).
