TensorFlow variant of NumPy's `isneginf`.

```
tf.experimental.numpy.isneginf(
    x
)
```

See the NumPy documentation for [`numpy.isneginf`](https://numpy.org/doc/stable/reference/generated/numpy.isneginf.html).
