# tf.experimental.numpy.stack

**Source URL:** [https://tensorflow.google.cn/api_docs/python/tf/experimental/numpy/stack](https://tensorflow.google.cn/api_docs/python/tf/experimental/numpy/stack)

---

[View source on GitHub](https://github.com/tensorflow/tensorflow/blob/v2.16.1/tensorflow/python/ops/numpy_ops/np_array_ops.py#L1198-L1211) |

TensorFlow variant of NumPy's `stack`.

```
tf.experimental.numpy.stack(
    arrays, axis=0
)
```

Unsupported arguments: `out`, `dtype`, `casting`.

See the NumPy documentation for [`numpy.stack`](https://numpy.org/doc/stable/reference/generated/numpy.stack.html).
