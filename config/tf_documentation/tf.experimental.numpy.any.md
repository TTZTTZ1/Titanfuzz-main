# tf.experimental.numpy.any

**Source URL:** [https://tensorflow.google.cn/api_docs/python/tf/experimental/numpy/any](https://tensorflow.google.cn/api_docs/python/tf/experimental/numpy/any)

---

[View source on GitHub](https://github.com/tensorflow/tensorflow/blob/v2.16.1/tensorflow/python/ops/numpy_ops/np_array_ops.py#L427-L431) |

TensorFlow variant of NumPy's `any`.

```
tf.experimental.numpy.any(
    a, axis=None, keepdims=None
)
```

Unsupported arguments: `out`, `where`.

See the NumPy documentation for [`numpy.any`](https://numpy.org/doc/stable/reference/generated/numpy.any.html).
