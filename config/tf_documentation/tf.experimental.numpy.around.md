# tf.experimental.numpy.around

**Source URL:** [https://tensorflow.google.cn/api_docs/python/tf/experimental/numpy/around](https://tensorflow.google.cn/api_docs/python/tf/experimental/numpy/around)

---

[View source on GitHub](https://github.com/tensorflow/tensorflow/blob/v2.16.1/tensorflow/python/ops/numpy_ops/np_array_ops.py#L820-L837) |

TensorFlow variant of NumPy's `around`.

```
tf.experimental.numpy.around(
    a, decimals=0
)
```

Unsupported arguments: `out`.

See the NumPy documentation for [`numpy.around`](https://numpy.org/doc/stable/reference/generated/numpy.around.html).
