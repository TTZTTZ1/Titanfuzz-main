# tf.experimental.numpy.isnan

**Source URL:** [https://tensorflow.google.cn/api_docs/python/tf/experimental/numpy/isnan](https://tensorflow.google.cn/api_docs/python/tf/experimental/numpy/isnan)

---

[View source on GitHub](https://github.com/tensorflow/tensorflow/blob/v2.16.1/tensorflow/python/ops/numpy_ops/np_math_ops.py#L934-L937) |

TensorFlow variant of NumPy's `isnan`.

```
tf.experimental.numpy.isnan(
    x
)
```

Unsupported arguments: `out`, `where`, `casting`, `order`, `dtype`, `subok`, `signature`, `extobj`.

See the NumPy documentation for [`numpy.isnan`](https://numpy.org/doc/stable/reference/generated/numpy.isnan.html).
