# tf.experimental.numpy.hypot

**Source URL:** [https://tensorflow.google.cn/api_docs/python/tf/experimental/numpy/hypot](https://tensorflow.google.cn/api_docs/python/tf/experimental/numpy/hypot)

---

[View source on GitHub](https://github.com/tensorflow/tensorflow/blob/v2.16.1/tensorflow/python/ops/numpy_ops/np_math_ops.py#L430-L433) |

TensorFlow variant of NumPy's `hypot`.

```
tf.experimental.numpy.hypot(
    x1, x2
)
```

Unsupported arguments: `out`, `where`, `casting`, `order`, `dtype`, `subok`, `signature`, `extobj`.

See the NumPy documentation for [`numpy.hypot`](https://numpy.org/doc/stable/reference/generated/numpy.hypot.html).
