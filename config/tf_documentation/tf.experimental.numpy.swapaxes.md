TensorFlow variant of NumPy's `swapaxes`.

```
tf.experimental.numpy.swapaxes(
    a, axis1, axis2
)
```

See the NumPy documentation for [`numpy.swapaxes`](https://numpy.org/doc/stable/reference/generated/numpy.swapaxes.html).
