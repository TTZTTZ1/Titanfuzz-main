# tf.experimental.numpy.random.uniform

[View source on GitHub](https://github.com/tensorflow/tensorflow/blob/v2.16.1/tensorflow/python/ops/numpy_ops/np_random.py#L81-L91) |

TensorFlow variant of NumPy's [`random.uniform`](https://tensorflow.google.cn/api_docs/python/tf/keras/random/uniform).

```
tf.experimental.numpy.random.uniform(
    low=0.0, high=1.0, size=None
)
```

See the NumPy documentation for [`numpy.random.uniform`](https://numpy.org/doc/stable/reference/generated/numpy.random.uniform.html).
