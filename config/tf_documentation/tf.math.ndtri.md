```
tf.math.ndtri(
    x, name=None
)
```

| Args | |

|  |  |
| --- | --- |
| `x` | `Tensor` with type `float` or `double`. |
| `name` | A name for the operation (optional). |

| Returns | |
| Inverse error function of `x`. | |
