# tf.experimental.numpy.diff

**Source URL:** [https://tensorflow.google.cn/api_docs/python/tf/experimental/numpy/diff](https://tensorflow.google.cn/api_docs/python/tf/experimental/numpy/diff)

---

[View source on GitHub](https://github.com/tensorflow/tensorflow/blob/v2.16.1/tensorflow/python/ops/numpy_ops/np_math_ops.py#L1054-L1086) |

TensorFlow variant of NumPy's `diff`.

```
tf.experimental.numpy.diff(
    a, n=1, axis=-1
)
```

Unsupported arguments: `prepend`, `append`.

See the NumPy documentation for [`numpy.diff`](https://numpy.org/doc/stable/reference/generated/numpy.diff.html).
