# tf.experimental.numpy.cumsum

**Source URL:** [https://tensorflow.google.cn/api_docs/python/tf/experimental/numpy/cumsum](https://tensorflow.google.cn/api_docs/python/tf/experimental/numpy/cumsum)

---

[View source on GitHub](https://github.com/tensorflow/tensorflow/blob/v2.16.1/tensorflow/python/ops/numpy_ops/np_array_ops.py#L500-L514) |

TensorFlow variant of NumPy's `cumsum`.

```
tf.experimental.numpy.cumsum(
    a, axis=None, dtype=None
)
```

Unsupported arguments: `out`.

See the NumPy documentation for [`numpy.cumsum`](https://numpy.org/doc/stable/reference/generated/numpy.cumsum.html).
