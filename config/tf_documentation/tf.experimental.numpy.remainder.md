# tf.experimental.numpy.remainder

**Source URL:** [https://tensorflow.google.cn/api_docs/python/tf/experimental/numpy/remainder](https://tensorflow.google.cn/api_docs/python/tf/experimental/numpy/remainder)

---

[View source on GitHub](https://github.com/tensorflow/tensorflow/blob/v2.16.1/tensorflow/python/ops/numpy_ops/np_math_ops.py#L176-L179) |

TensorFlow variant of NumPy's `remainder`.

```
tf.experimental.numpy.remainder(
    x1, x2
)
```

Unsupported arguments: `out`, `where`, `casting`, `order`, `dtype`, `subok`, `signature`, `extobj`.

See the NumPy documentation for [`numpy.remainder`](https://numpy.org/doc/stable/reference/generated/numpy.remainder.html).
