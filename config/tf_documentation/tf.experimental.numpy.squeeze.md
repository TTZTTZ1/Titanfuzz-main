TensorFlow variant of NumPy's `squeeze`.

```
tf.experimental.numpy.squeeze(
    a, axis=None
)
```

See the NumPy documentation for [`numpy.squeeze`](https://numpy.org/doc/stable/reference/generated/numpy.squeeze.html).
