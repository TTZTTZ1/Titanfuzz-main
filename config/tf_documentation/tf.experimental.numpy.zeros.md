TensorFlow variant of NumPy's `zeros`.

```
tf.experimental.numpy.zeros(
    shape, dtype=float
)
```

See the NumPy documentation for [`numpy.zeros`](https://numpy.org/doc/stable/reference/generated/numpy.zeros.html).
