# tf.experimental.numpy.divmod

**Source URL:** [https://tensorflow.google.cn/api_docs/python/tf/experimental/numpy/divmod](https://tensorflow.google.cn/api_docs/python/tf/experimental/numpy/divmod)

---

[View source on GitHub](https://github.com/tensorflow/tensorflow/blob/v2.16.1/tensorflow/python/ops/numpy_ops/np_math_ops.py#L182-L185) |

TensorFlow variant of NumPy's `divmod`.

```
tf.experimental.numpy.divmod(
    x1, x2
)
```

Unsupported arguments: `out1`, `out2`, `out`, `where`, `casting`, `order`, `dtype`, `subok`, `signature`, `extobj`.

See the NumPy documentation for [`numpy.divmod`](https://numpy.org/doc/stable/reference/generated/numpy.divmod.html).
