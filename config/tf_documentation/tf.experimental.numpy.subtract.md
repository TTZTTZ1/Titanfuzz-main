# tf.experimental.numpy.subtract

**Source URL:** [https://tensorflow.google.cn/api_docs/python/tf/experimental/numpy/subtract](https://tensorflow.google.cn/api_docs/python/tf/experimental/numpy/subtract)

---

[View source on GitHub](https://github.com/tensorflow/tensorflow/blob/v2.16.1/tensorflow/python/ops/numpy_ops/np_math_ops.py#L102-L105) |

TensorFlow variant of NumPy's `subtract`.

```
tf.experimental.numpy.subtract(
    x1, x2
)
```

Unsupported arguments: `out`, `where`, `casting`, `order`, `dtype`, `subok`, `signature`, `extobj`.

See the NumPy documentation for [`numpy.subtract`](https://numpy.org/doc/stable/reference/generated/numpy.subtract.html).
