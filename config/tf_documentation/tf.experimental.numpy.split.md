TensorFlow variant of NumPy's `split`.

```
tf.experimental.numpy.split(
    ary, indices_or_sections, axis=0
)
```

See the NumPy documentation for [`numpy.split`](https://numpy.org/doc/stable/reference/generated/numpy.split.html).
