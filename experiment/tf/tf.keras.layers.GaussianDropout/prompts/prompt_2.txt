**tf.keras.layers.GaussianDropout** is a regularization layer that applies multiplicative Gaussian noise during training to help prevent overfitting.

**Key Features:**
- **Functionality:** Multiplies the input by random noise drawn from a Gaussian (normal) distribution centered at 1.0.
- **Active State:** Only active during training (`training=True`); does nothing during inference.
- **Noise Standard Deviation:** Controlled by the `rate` parameter, where `stddev = sqrt(rate / (1 - rate))`.

**Parameters:**
- `rate` (Float): Drop probability. Determines the noise magnitude.
- `seed` (Integer, optional): Random seed for reproducible behavior.

**Usage:**
```python
# Example: Apply Gaussian dropout with a 0.2 rate
layer = tf.keras.layers.GaussianDropout(rate=0.2)
output = layer(inputs, training=True)  # Noise applied only during training
```

**Note:** Unlike standard `Dropout` (which sets values to zero), this layer scales inputs with continuous random noise, which can be beneficial for certain network architectures.
