`tf.image.resize` is a TensorFlow function for resizing images to a specified target size using various interpolation methods. It supports both 4-D (batch) and 3-D (single image) input tensors.

**Key features:**
- **Methods:** Supports multiple resizing algorithms (`bilinear`, `nearest`, `bicubic`, `area`, `lanczos3`, `lanczos5`, `gaussian`, `mitchellcubic`), with `bilinear` as default.
- **Aspect ratio:** By default, the aspect ratio is not preserved (can cause distortion). Use `preserve_aspect_ratio=True` to maintain it, scaling to fit within the target dimensions.
- **Anti-aliasing:** Optional anti-aliasing filter for downsampling (`antialias=True`).
- **Data type:** Returns `float32` except when using `nearest` neighbor interpolation, which retains the input dtype.

**Usage example:**
```python
resized = tf.image.resize(image, [new_height, new_width], method='bilinear')
```

**Note:** To avoid distortion without cropping, consider `tf.image.resize_with_pad`.
