4. **Dynamic Shapes:** Enable dynamic shape support and build profiles using an input function.

**Core Methods:**

- `build()`: Executes the converted graph with sample input to build and cache TensorRT engines.
- `save()`: Saves the converted SavedModel, optionally including GPU-specific engines.
- `summary()`: Provides a detailed report of the conversion results.

**Important Notes:**
- Conversion is platform-dependent (requires compatible GPU environment).
- INT8 calibration requires representative input data.
- Dynamic shape mode requires building profiles with an input function.
- Windows support is experimental.
