`tf.cond` is a TensorFlow function that conditionally executes one of two callable functions (`true_fn` or `false_fn`) based on a boolean predicate `pred`. It returns the output of the selected function.

**Key Features:**
- **Automatic Conversion:** In `tf.function`, Python `if` statements with tensor conditions are automatically converted to `tf.cond` (unless `autograph=False`).
- **Graph-Based Execution:** Both `true_fn` and `false_fn` are called exactly once during graph construction, not during runtime execution. The function stitches the appropriate branch into the graph based on `pred`.
- **Output Consistency:** The two functions must return the same number and type of outputs (allowing nested structures like lists/tuples, which are implicitly unpacked if singletons).
- **Eager Execution:** Operations outside the branches are always executed, regardless of which branch is taken.

**Important Notes:**
- Do not use tensors created inside a branch outside of `tf.cond` directly; instead, return them as outputs.
- A scalar boolean tensor is required for `pred`.

**Example:**
```python
result = tf.cond(x < y, lambda: x * 7, lambda: y + 3)
```
If `x < y` is true, `x * 7` is computed; otherwise, `y + 3` is computed.
