`tf.switch_case` is an integer-indexed conditional operation that selects and executes one callable from a set based on an index tensor. It is more efficient than `tf.case` when exactly one branch is chosen, similar to a C++ switch/case statement.

**Key Features:**
- **Inputs:**  
  - `branch_index`: An integer tensor specifying which branch to execute.  
  - `branch_fns`: A dictionary mapping integers to callables, a list of (int, callable) pairs, or a list of callables (where the index serves as the key).  
  - `default`: Optional callable executed if no key matches (if omitted, the callable with the maximum key is used).  
- **Behavior:**  
  Selects the callable corresponding to `branch_index`; falls back to `default` if provided, otherwise uses the branch with the highest key.  
- **Requirements:**  
  All callables must return the same nested structure of tensors (as supported by `tf.nest`). Keys must form a contiguous range from `0` to `len(branch_fns)-1`.  
- **Errors:** Raises `TypeError` for invalid `branch_fns` or non-callable entries.

**Example:**
```python
r = tf.switch_case(branch_index, branch_fns={0: f1, 1: f2}, default=f3)
```
