`tf.switch_case` is a TensorFlow operation that implements an integer-indexed conditional, similar to a C++ switch/case statement. It selects and executes one callable from a set based on an integer tensor input.

**Key Features:**
- More efficient than `tf.case` when exactly one branch is selected.
- Accepts `branch_fns` as a dictionary, list of (int, callable) pairs, or a simple list of callables (where index serves as the key).
- Requires keys to form a contiguous set from `0` to `len(branch_fns) - 1`.
- Returns the output of the selected callable; falls back to `default` if no key matches, or to the max-keyed branch if no default is provided.
- Supports nested structures via `tf.nest`—all callables must return the same structure.

**Example:**
```python
r = tf.switch_case(branch_index, branch_fns={0: f1, 1: f2}, default=f3)
```
This executes `f1` if `branch_index` is 0, `f2` if 1, otherwise `f3`.
