**Key Features:**
- Accepts `branch_fns` as a dictionary, list of (int, callable) pairs, or a list of callables (with implicit integer keys starting from 0).
- Uses `branch_index` (an int tensor) to choose the matching callable.
- Falls back to `default` if no key matches; if no default is provided, it executes the branch with the maximum key.
- All callables must return identical nested tensor structures (as supported by `tf.nest`).

**Example:**
```python
r = tf.switch_case(branch_index, branch_fns={0: f1, 1: f2}, default=f3)
```
