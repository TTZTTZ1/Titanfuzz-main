`tf.random.uniform` generates random numbers from a uniform distribution within a specified range.

**Key Features:**
- Outputs random values uniformly distributed in `[minval, maxval)` (inclusive of `minval`, exclusive of `maxval`).
- Default range for floats is `[0, 1)`; for integers, `maxval` must be explicitly set.
- Supports float and integer data types (`float16`, `float32`, `float64`, `int32`, `int64`).
- Allows seeding for reproducible sequences when combined with `tf.random.set_seed`.

**Parameters:**
- `shape`: Output tensor shape.
- `minval` (default 0): Lower bound (inclusive).
- `maxval` (default 1 for floats): Upper bound (exclusive).
- `dtype` (default `float32`): Output data type.
- `seed`: Optional seed for reproducibility.
- `name`: Operation name.

**Example:**
```python
tf.random.uniform(shape=[2, 3], minval=-1, maxval=1, dtype=tf.float32)
```

**Note:** Integer outputs may have slight bias unless `maxval - minval` is a power of two.
