The `tf.keras.metrics.MeanIoU` class computes the **mean Intersection-over-Union (IoU)** for semantic segmentation tasks. It calculates IoU per class and then averages the results.

**Key Features:**
- **Inputs:** Requires `num_classes` to allocate a confusion matrix.
- **Handles:** Integer or dense label encodings via `sparse_y_true` and `sparse_y_pred`.
- **Flexibility:** Supports ignoring a specific class (e.g., void class) and sample weighting.
- **Usage:** Can be integrated directly into Keras model training via `compile()`.

**Calculation:**
- IoU for each class = `true_positives / (true_positives + false_positives + false_negatives)`
- Final metric = average of IoUs across all classes.

**Example:**
```python

m.update_state([0, 0, 1, 1], [0, 1, 0, 1])

```

This metric is essential for evaluating segmentation models by measuring pixel-wise classification accuracy across classes.
