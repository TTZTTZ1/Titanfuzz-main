The `tf.keras.metrics.MeanIoU` class computes the mean Intersection-over-Union (IoU) metric, commonly used for evaluating semantic segmentation models. It calculates IoU per class and then averages the results.

**Key Features:**
- **Formula:** IoU = true_positives / (true_positives + false_positives + false_negatives)
- **Requires** `num_classes` to allocate a confusion matrix.
- Supports optional **class ignoring** (`ignore_class`) for void/background labels.
- Handles both **sparse** (integer-encoded) and **dense** (one-hot/logits) label formats via `sparse_y_true` and `sparse_y_pred`.
- Accepts **sample weights** to mask or weight specific predictions.

**Basic Usage:**
```python

metric.update_state(y_true, y_pred)
mean_iou = metric.result()  # Returns the mean IoU across classes
```

```python

```
