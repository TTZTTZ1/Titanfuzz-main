`tf.keras.metrics.MeanIoU` computes the **mean Intersection-over-Union (IoU)** for semantic segmentation tasks. It calculates IoU per class and then averages them, using a confusion matrix to track true positives, false positives, and false negatives.

### Key Features:
- **Inputs**: Requires `num_classes` to allocate a confusion matrix.
- **Handles sparse or dense labels/predictions** via `sparse_y_true` and `sparse_y_pred`.
- **Optional class ignoring** (`ignore_class`) for void/background labels.
- **Supports sample weighting** for flexible metric computation.


### Integration:


### Main Methods:
- `update_state()`: Accumulates confusion matrix stats.
- `result()`: Computes mean IoU from the confusion matrix.
- `reset_state()`: Clears accumulated state between epochs.

This metric is essential for evaluating segmentation models, providing a per-class and overall measure of prediction overlap with ground truth.
