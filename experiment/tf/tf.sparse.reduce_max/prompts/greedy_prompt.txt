**Key features:**
- Reduces a `SparseTensor` along given axes; reduces all dimensions if `axis=None`.
- Undefined (missing) values do not participate in the reduction—unlike dense reductions, this can yield negative results for sparse axes.
- If no values exist along a reduced axis, the result defaults to 0.
- Supports `keepdims` to retain reduced dimensions with size 1.
- **Note:** No gradient is defined, so it cannot be used in trainable models.

**Example:**
