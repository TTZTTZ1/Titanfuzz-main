`tf.keras.layers.Conv1DTranspose` is a 1D transposed convolution layer used for upsampling or generating outputs with a shape that reverses a regular convolution. It expands the spatial dimensions (steps) while transforming channels via learned filters.

**Key Parameters:**
- `filters`: Number of output channels.
- `kernel_size`: Size of the convolution window.
- `strides`: Upsampling factor (default 1).
- `padding`: `"valid"` (no padding) or `"same"` (output matches input size).
- `data_format`: `"channels_last"` (batch, steps, channels) or `"channels_first"` (batch, channels, steps).

**Input/Output Shapes:**
- Input: 3D tensor `(batch, steps, channels)` (channels_last) or `(batch, channels, steps)` (channels_first).
- Output: 3D tensor with upsampled `steps` dimension and channels changed to `filters`.

**Restriction:** `strides > 1` cannot be used with `dilation_rate > 1`.

**Example:**
```python
layer = Conv1DTranspose(filters=32, kernel_size=3, strides=2, padding='same')
# Input shape: (4, 10, 128) → Output shape: (4, 20, 32)
```

Commonly used in autoencoders, generative models, and signal processing for learned upsampling.
