The `tf.keras.layers.Conv1D` layer performs 1D convolution, primarily used for temporal or sequential data. It applies a convolution kernel over a single spatial/temporal dimension to extract features, optionally adding bias and applying an activation function.

**Key Parameters:**
- `filters`: Number of output channels/filters.
- `kernel_size`: Size of the convolution window.
- `strides`: Step size for the convolution (must be 1 if `dilation_rate` > 1).
- `padding`: `"valid"` (no padding), `"same"` (output size matches input when strides=1), or `"causal"` (for temporal order preservation).
- `data_format`: `"channels_last"` (default, shape: `[batch, steps, channels]`) or `"channels_first"` (shape: `[batch, channels, steps]`).
- `dilation_rate`: Spacing between kernel elements for dilated convolution.
- `activation`: Activation function applied to outputs.

**Input/Output Shapes:**
- **Input:** 3D tensor `(batch, steps, channels)` for `"channels_last"`, or `(batch, channels, steps)` for `"channels_first"`.
- **Output:** 3D tensor with `steps` dimension adjusted based on `padding` and `strides`, and channels replaced by `filters`.

**Example:**
```python
x = np.random.rand(4, 10, 128)  # 4 samples, 10 timesteps, 128 features
y = keras.layers.Conv1D(32, 3, activation='relu')(x)  # Output shape: (4, 8, 32)
```

This layer is commonly used in time-series analysis, audio processing, and text-related tasks for local pattern extraction.
