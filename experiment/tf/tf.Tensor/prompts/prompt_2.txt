- **Properties**: Includes `dtype` (data type), `shape` (dimensions), and `name`.

### Important Methods:
- **`shape` / `get_shape()`**: Return the tensor's shape as a `tf.TensorShape` (may be partially known in graph mode).

- **`ref()`**: Provides a hashable reference for use in sets/dictionaries (since tensors are not directly hashable).
- **`numpy()`**: Converts the tensor to a NumPy array (in eager execution).

### Operator Overloading:
Supports Python operators (e.g., `+`, `-`, `@` for matrix multiplication) and comparisons, enabling intuitive syntax similar to NumPy.

### Notes:
- Tensors are immutable; operations create new tensors.
- Avoid using tensors as booleans (e.g., in `if` statements) to prevent errors.
- When created from NumPy arrays, data may be shared (changes to the array can affect the tensor).

For more details, refer to the [TensorFlow Guide](https://tensorflow.org/guide/tensor).
