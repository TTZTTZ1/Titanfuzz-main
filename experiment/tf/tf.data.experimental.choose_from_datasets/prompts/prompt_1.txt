`tf.data.experimental.choose_from_datasets` creates a dataset that selects elements from a list of input datasets based on indices provided by a separate `choice_dataset`. It deterministically interleaves data, allowing custom control over the order in which elements are drawn from each source.

**Key Points:**
- **Inputs**:  
  - `datasets`: A non-empty list of datasets with compatible structures.  
  - `choice_dataset`: A dataset of scalar `tf.int64` values (0 to `len(datasets)-1`) specifying which dataset to pick from at each step.
- **Behavior**:  
  - For each index in `choice_dataset`, the output dataset yields the next element from the corresponding dataset in `datasets`.
  - If `stop_on_empty_dataset=False` (default), it skips exhausted datasets and continues; if `True`, it stops when any dataset is empty.
- **Deprecated**: Use `tf.data.Dataset.choose_from_datasets` instead, noting that the non-experimental version defaults `stop_on_empty_dataset=True`.
- **Example**: With three datasets `["foo", "bar", "baz"]` repeated and `choice_dataset = [0, 1, 2, 0, 1, 2, ...]`, the output cycles through `"foo", "bar", "baz"` accordingly.
