- **max_output_size_per_class**: Maximum boxes to retain per class after NMS.
- **max_total_size**: Overall maximum boxes retained across all classes (clips/pads outputs).
- **iou_threshold**: IOU threshold for overlap suppression.
- **score_threshold**: Minimum score threshold to consider a box.

**Optional Flags:**
- `pad_per_class`: If `True`, pads outputs per class; otherwise clips/pads to `max_total_size`.
- `clip_boxes`: If `True`, clips box coordinates to [0, 1]; otherwise outputs raw coordinates.

**Returns:** A tuple of four tensors:
1. `nmsed_boxes`: Selected bounding boxes.
2. `nmsed_scores`: Corresponding scores.
3. `nmsed_classes`: Corresponding class labels.
4. `valid_detections`: Number of valid detections per batch item.

This operation is coordinate-system invariant (robust to translations/reflections) and helps reduce redundant detections in object detection pipelines.
