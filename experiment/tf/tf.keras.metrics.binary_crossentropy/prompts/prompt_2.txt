**Key parameters:**
- `y_true`: Ground truth values (shape: `[batch_size, d0, ... dN]`).
- `y_pred`: Predicted values (same shape as `y_true`).
- `from_logits`: If `True`, treats `y_pred` as logits; otherwise, assumes it represents probabilities (default: `False`).
- `label_smoothing`: Smooths labels by adjusting values toward 0.5 (range: `[0, 1]`).
- `axis`: Axis along which to compute the mean (default: `-1`).

**Returns:**
