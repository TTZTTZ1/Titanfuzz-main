- **Purpose**: Finds the best specificity given a minimum sensitivity constraint.
- **Core Calculation**: Uses true positives, true negatives, false positives, and false negatives across multiple thresholds to determine the optimal specificity.
- **Parameters**:
  - `sensitivity`: Target sensitivity value (between 0 and 1).
  - `num_thresholds`: Number of thresholds to evaluate (default 200).
  - `class_id`: Optional class ID for multi-class classification.
- **Usage**: Works with binary or multi-class classification via `class_id`.
- **Integration**: Can be used directly in `model.compile()` as a metric.
