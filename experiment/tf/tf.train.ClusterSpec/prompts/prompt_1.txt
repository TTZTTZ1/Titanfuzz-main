`tf.train.ClusterSpec` represents a TensorFlow cluster composed of multiple **jobs** (e.g., "worker", "ps"), each containing one or more **tasks** (processes) identified by network addresses.

**Key Features:**
- **Construction**: Can be created from a dictionary mapping job names to either:
  - A list of addresses (dense task indices).
  - A dictionary mapping sparse task indices to addresses.
- **Core Methods**:
  - `as_dict()` / `as_cluster_def()`: Export cluster definition.
  - `task_address(job_name, task_index)`: Get address of a specific task.
  - `num_tasks(job_name)`: Count tasks in a job.
  - `job_tasks(job_name)`: List all task addresses for a job.
- **Use Case**: Essential for configuring distributed TensorFlow servers (`tf.distribute.Server`) in multi‑worker or parameter‑server training setups.

**Example:**
```python
cluster = tf.train.ClusterSpec({
    "worker": ["worker0:2222", "worker1:2222"],
    "ps": ["ps0:2222", "ps1:2222"]
})
```
