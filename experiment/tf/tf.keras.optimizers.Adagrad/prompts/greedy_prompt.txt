- **Per-parameter learning rates:** Accumulates squared gradients to scale learning rates individually.
- **Default settings:** Learning rate = 0.001, initial accumulator = 0.1, epsilon = 1e-07.
- **Advanced options:** Supports weight decay, gradient clipping, exponential moving average (EMA), loss scaling, and gradient accumulation.
- **Usage:** Suitable for sparse data and problems where parameters require different learning rates.

**Common Use Cases:**
- Training recommendation systems and deep retrieval models.
- Federated learning with sparse updates.
- Scenarios benefiting from adaptive learning rates without manual tuning.

**Example:**
