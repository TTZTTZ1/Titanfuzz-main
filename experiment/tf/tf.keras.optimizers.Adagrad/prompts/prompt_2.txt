`tf.keras.optimizers.Adagrad` is an optimizer that adapts learning rates per parameter based on update frequency, reducing them for frequently updated parameters. It's suitable for sparse data and benefits from higher initial learning rates.

**Key Parameters:**
- `learning_rate`: Default 0.001 (higher values often better)
- `initial_accumulator_value`: Starting accumulator value (≥0)
- `epsilon`: Numerical stability constant
- Optional: Weight decay, gradient clipping, EMA, loss scaling, gradient accumulation

**Core Mechanism:**  
Maintains a per-parameter accumulator that sums squared gradients, automatically decreasing learning rates over time.

**Typical Use:**  
```python
optimizer = tf.keras.optimizers.Adagrad(learning_rate=0.01)
model.compile(optimizer=optimizer, ...)
```

**Advantages:**  
- Automatic learning rate adaptation  
- Well-suited for sparse features/imbalanced data

**Considerations:**  
- Accumulators can cause excessively small updates over time  
- Often works better with higher initial learning rates than other optimizers
