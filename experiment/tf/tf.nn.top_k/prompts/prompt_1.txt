**Key features:**
- Works for vectors, matrices, or higher-rank tensors.
- Returns a tuple of `(values, indices)` where `values` contains the top `k` elements, and `indices` holds their positions.
- By default, results are sorted in descending order (`sorted=True`).
- If values are equal, the element with the lower index appears first.
- Supports different `index_type` (default: `tf.int32`).

**Example usage:**
```python

```
The output shapes match the input shape except the last dimension, which becomes `k`.
