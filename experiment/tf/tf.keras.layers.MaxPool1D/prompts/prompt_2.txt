`tf.keras.layers.MaxPool1D` performs 1D max pooling to downsample temporal data by taking the maximum value over a sliding window.

**Key Parameters:**
- `pool_size`: Size of the pooling window (default: 2).
- `strides`: Step size for sliding the window (defaults to `pool_size`).
- `padding`: `"valid"` (no padding) or `"same"` (output length matches input length).
- `data_format`: `"channels_last"` (batch, steps, features) or `"channels_first"` (batch, features, steps).

**Input Shape:**  
3D tensor: `(batch_size, steps, features)` for `channels_last`, or `(batch_size, features, steps)` for `channels_first`.

**Output Shape:**  
Same as input shape but with `steps` dimension reduced based on `pool_size`, `strides`, and `padding`.

**Example:**
```python
x = np.array([1., 2., 3., 4., 5.]).reshape(1, 5, 1)
max_pool = keras.layers.MaxPool1D(pool_size=2, strides=1, padding="valid")
max_pool(x)  # Output shape: (1, 4, 1)
```

This layer reduces the temporal dimension while preserving feature depth, commonly used in CNNs for feature extraction and dimensionality reduction.
