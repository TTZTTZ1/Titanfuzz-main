**Key Parameters:**
- `filters`: Number of output filters (pointwise convolution).
- `kernel_size`: Size of the depthwise convolution window.
- `strides`: Convolution stride (default 1).
- `padding`: `"valid"` (no padding) or `"same"` (padding to preserve spatial dimensions when `strides=1`).
- `data_format`: `"channels_last"` (default) or `"channels_first"`.
- `depth_multiplier`: Number of depthwise output channels per input channel (default 1).
- `activation`: Optional activation function applied to the output.

**Input/Output Shape:**
- **Input:** 4D tensor `(batch, height, width, channels)` if `data_format="channels_last"`, or `(batch, channels, height, width)` if `"channels_first"`.
- **Output:** 4D tensor with spatial dimensions adjusted by convolution and `filters` as the last/first dimension depending on `data_format`.

**Example:**
```python
layer = SeparableConv2D(filters=3, kernel_size=4, strides=2, activation='relu')
output = layer(input_tensor)  # Output shape depends on input and parameters.
```

**Use Case:** Efficient feature extraction in models like MobileNet, reducing computational cost while maintaining performance.
