**Key Parameters:**
- `filters`: Number of output channels.
- `kernel_size`: Size of the 3D convolution window (int or tuple of 3 ints).
- `strides`: Step size for convolution (default: `(1,1,1)`).
- `padding`: `"valid"` (no padding) or `"same"` (padding to preserve spatial dimensions when `strides=1`).
- `data_format`: `"channels_last"` (default) or `"channels_first"` for dimension ordering.
- `dilation_rate`: Spacing between kernel elements (default: `(1,1,1)`).
- `activation`: Optional activation function applied to outputs.

**Input Shape:**
- 5D tensor: `(batch, spatial_dim1, spatial_dim2, spatial_dim3, channels)` for `channels_last`.
- 5D tensor: `(batch, channels, spatial_dim1, spatial_dim2, spatial_dim3)` for `channels_first`.

**Output Shape:**
- Same 5D structure as input, with spatial dimensions adjusted based on padding/strides, and last dimension changed to `filters`.

**Example:**
```python

y = keras.layers.Conv3D(32, 3, activation='relu')(x)  # Output shape: (4, 8, 8, 8, 32)
```

**Note:** Incompatible to set both `strides > 1` and `dilation_rate > 1`.
