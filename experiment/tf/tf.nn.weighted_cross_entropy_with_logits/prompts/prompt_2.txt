- **Purpose**: Weighted version of `sigmoid_cross_entropy_with_logits`.
- **Formula**:  
  `loss = labels * -log(sigmoid(logits)) * pos_weight + (1 - labels) * -log(1 - sigmoid(logits))`
- **Effect**:
  - `pos_weight > 1`: Increases recall (reduces false negatives).
  - `pos_weight < 1`: Increases precision (reduces false positives).
- **Inputs**:
  - `labels`: Values in [0, 1], same shape as `logits`.
  - `logits`: Real numbers (`float32`/`float64`).
  - `pos_weight`: Non-negative weight for positive examples.
- **Output**: Tensor of loss values, same shape as `logits`.
- **Error**: Raises `ValueError` if `logits` and `labels` shapes mismatch.

**Example**:
```python
loss = tf.nn.weighted_cross_entropy_with_logits(
    labels=[1., 0.], 
    logits=[1.5, -0.1], 
    pos_weight=2.0
)
```
