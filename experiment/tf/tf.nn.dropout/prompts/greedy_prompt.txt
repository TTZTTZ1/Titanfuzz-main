`tf.nn.dropout` is a regularization technique that randomly sets elements of an input tensor to zero during training to prevent overfitting. It scales the remaining elements to preserve the expected value. Key features include:

- **Parameters**:  
  - `x`: Floating-point input tensor.  
  - `rate`: Dropout probability (scalar between 0 and 1).  
  - `noise_shape`: Optional shape for controlling dropout patterns (e.g., dropping entire channels).  
  - `seed`: For reproducible randomness.  

- **Behavior**:  
  - Each element is independently dropped with probability `rate` and scaled by `1/(1 - rate)`.  
  - If `rate=0`, the tensor remains unchanged.  

- **Note**:  
  - Prefer `tf.nn.experimental.stateless_dropout` for better RNG control.  
  - Use named arguments for consistency between TensorFlow versions.  

- **Example**:  
  ```python
  x = tf.ones([3,5])
  tf.nn.dropout(x, rate=0.5, seed=1)  # Randomly zeros ~50% of elements
  ```
