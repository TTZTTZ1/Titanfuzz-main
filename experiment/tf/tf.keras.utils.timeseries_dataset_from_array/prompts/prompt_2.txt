`tf.keras.utils.timeseries_dataset_from_array` creates a TensorFlow dataset of sliding windows from a time series array, suitable for sequence-based modeling like forecasting or temporal regression.

**Key parameters:**
- `data`: Input time series array (time dimension on axis 0).
- `targets`: Corresponding targets for each window (can be `None`).
- `sequence_length`: Length of each output sequence.
- `sequence_stride`: Step between consecutive sequences.
- `sampling_rate`: Interval between timesteps within a sequence.
- `batch_size`: Number of sequences per batch.
- `shuffle`: Whether to shuffle samples.
- `start_index`/`end_index`: Bounds for selecting data.

**Returns:**  
A `tf.data.Dataset` yielding either `(sequences, targets)` tuples or just `sequences` if targets are `None`.

**Common use cases:**
1. **Predict next step:** Use past `sequence_length` steps to predict the immediate future.
2. **Many-to-many prediction:** Align separate input and target sequences (e.g., for seq2seq models).
3. **Non-overlapping windows:** Set `sequence_stride = sequence_length` for disjoint sequences.

The function efficiently handles indexing and batching, making it ideal for transforming raw time series into supervised learning formats.
