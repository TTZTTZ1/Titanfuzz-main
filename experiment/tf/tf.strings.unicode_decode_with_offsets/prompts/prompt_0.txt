- **Output:** A tuple `(codepoints, start_offsets)`, where:

- **Encoding:** Requires specifying the input encoding (e.g., `'UTF-8'`).
- **Error Handling:** Supports `'strict'`, `'replace'`, or `'ignore'` for invalid substrings.

```python

# offsets:    [[0, 1, 3, 5, 6, 7, 8, 9, 10], [0]]
```
