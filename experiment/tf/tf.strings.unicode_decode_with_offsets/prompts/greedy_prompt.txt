- **Output:** A tuple `(codepoints, start_offsets)`:
  - `codepoints`: int32 tensor of decoded Unicode code points.

- **Parameters:**

  - `errors`: How to handle invalid bytes (`'strict'`, `'replace'`, `'ignore'`).  
  - `replacement_char`: Replacement code point for invalid/control characters.  
  - `replace_control_characters`: Whether to replace C0 control characters.

**Example:**
```python

# codepoints: [[71, 246, 246, 100, 110, 105, 103, 104, 116], [128522]]
# offsets:    [[0, 1, 3, 5, 6, 7, 8, 9, 10], [0]]
```
