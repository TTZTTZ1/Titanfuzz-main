The `tf.keras.initializers.glorot_normal` (also called Xavier normal initializer) is a weight initializer that draws samples from a truncated normal distribution centered at 0. It is designed to maintain stable variance across layers in neural networks, especially when using activation functions like sigmoid or tanh.

**Key Features:**
- **Standard Deviation:** `stddev = sqrt(2 / (fan_in + fan_out))`, where `fan_in` is the number of input units and `fan_out` is the number of output units.
- **Purpose:** Helps mitigate vanishing/exploding gradients during training.
- **Usage:** Can be used standalone or as a kernel initializer in Keras layers (e.g., `Dense`, `Conv2D`).
- **Seed Control:** Accepts a seed for deterministic behavior; using a `SeedGenerator` allows different random values across calls.

**Example:**
```python
initializer = tf.keras.initializers.GlorotNormal(seed=42)
layer = tf.keras.layers.Dense(10, kernel_initializer=initializer)
```
