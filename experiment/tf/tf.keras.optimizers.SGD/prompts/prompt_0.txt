The `tf.keras.optimizers.SGD` class implements the **Stochastic Gradient Descent (SGD) optimizer**, optionally with **momentum** and **Nesterov acceleration**. It is a fundamental optimizer for training neural networks by iteratively updating model parameters to minimize loss.

### **Core Functionality:**
- **Basic SGD Update (momentum=0):**  
  `w = w - learning_rate * g`
- **SGD with Momentum (momentum > 0):**  
  `velocity = momentum * velocity - learning_rate * g`  
  `w = w + velocity`
- **Nesterov Momentum (nesterov=True):**  
  Adjusts the update to use a "look-ahead" velocity for potentially faster convergence.

### **Key Parameters:**
- `learning_rate`: Step size for updates (can be float, schedule, or callable).
- `momentum`: Accelerates descent and reduces oscillations (≥ 0).
- `nesterov`: Enables Nesterov momentum for improved convergence.
- `weight_decay`: Optional L2 regularization (weight decay).
- Gradient clipping (`clipnorm`, `clipvalue`, `global_clipnorm`): Controls gradient magnitude.
- Exponential Moving Average (`use_ema`): Smooths weights over training for better final models.
- `loss_scale_factor`: Prevents underflow in mixed-precision training.
- `gradient_accumulation_steps`: Averages gradients over multiple steps for small batch sizes.

### **Main Methods:**
- `apply(grads, trainable_variables)`: Updates variables using computed gradients.
- `build(variables)`: Initializes optimizer state (e.g., momentum buffers).
- `update_step(gradient, variable, learning_rate)`: Performs the core parameter update.
- `finalize_variable_values(var_list)`: Applies EMA to variables if enabled.
- `get_config()` / `from_config()`: Serialization and deserialization.

### **Typical Use:**
```python
optimizer = tf.keras.optimizers.SGD(learning_rate=0.01, momentum=0.9, nesterov=True)
model.compile(optimizer=optimizer, ...)
```
SGD is widely used in both basic and advanced training loops, supporting distributed training, custom schedules, and stability enhancements like gradient clipping and EMA.
