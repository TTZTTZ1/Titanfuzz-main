The `tf.keras.optimizers.SGD` class implements the **Stochastic Gradient Descent** optimizer, optionally with **momentum** and **Nesterov acceleration**. It is a fundamental optimization algorithm used to update model parameters during training by minimizing the loss function.

### **Key Features:**
- **Basic SGD**: Updates parameters by subtracting the gradient scaled by the learning rate.
- **Momentum**: Accelerates convergence and reduces oscillations by accumulating a velocity term from past gradients.
- **Nesterov Momentum**: A variant that adjusts the update rule for potentially faster convergence.
- **Additional Enhancements**: Supports weight decay, gradient clipping, exponential moving averages (EMA), loss scaling, and gradient accumulation.

### **Primary Use Cases:**
- Training deep learning models, especially when fine-tuning or when a simple, stable optimizer is preferred.
- Scenarios requiring customizable learning rate schedules or noise reduction via gradient accumulation.
- Mixed precision training with loss scaling to prevent underflow.

### **Common Parameters:**
- `learning_rate`: Step size for updates (can be dynamic via schedules).
- `momentum`: Strength of the momentum term (≥ 0).
- `nesterov`: Enables Nesterov momentum if `True`.
- `weight_decay`: Applies L2 regularization to parameters.
- `clipnorm`/`clipvalue`: Gradient clipping to prevent exploding gradients.
- `use_ema`: Smooths model weights over time for better generalization.

### **Update Rules:**
- **Without momentum**:  
  `w = w - learning_rate * g`
- **With momentum**:  
  `velocity = momentum * velocity - learning_rate * g`  
  `w = w + velocity`
- **With Nesterov**:  
  Adjusts velocity before updating weights for improved convergence.

SGD is widely used due to its simplicity, efficiency, and effectiveness, especially when combined with momentum and learning rate scheduling.
