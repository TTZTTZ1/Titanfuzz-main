The `tf.keras.optimizers.RMSprop` optimizer implements the RMSprop algorithm, which maintains a moving average of squared gradients and divides the gradient by the root of this average to adjust learning rates adaptively per parameter.

**Key Features:**
- **Adaptive Learning Rates:** Adjusts step sizes based on recent gradient magnitudes.
- **Momentum Support:** Optional plain momentum (not Nesterov) for accelerated convergence.
- **Centered Option:** Can normalize gradients by estimated variance (centered=True) for potentially better training stability.
- **EMA Integration:** Supports exponential moving averages of weights for smoother updates.
- **Gradient Clipping:** Includes options for individual or global gradient norm clipping.
- **Weight Decay:** Optional L2 regularization via weight decay.

**Common Use Cases:**
- Recurrent neural networks (RNNs) and non-stationary objectives.
- Scenarios requiring per-parameter adaptive learning rates.
- Training with small batch sizes (can use gradient accumulation).

**Example Usage:**
```python
opt = keras.optimizers.RMSprop(learning_rate=0.01, rho=0.9)
model.compile(optimizer=opt, loss='mse')
```

**Parameters Summary:**
- `learning_rate`: Base step size (default 0.001)
- `rho`: Discount factor for gradient history (default 0.9)
- `momentum`: Momentum coefficient (default 0.0)
- `epsilon`: Numerical stability constant (default 1e-7)
- `centered`: Whether to use centered RMSprop (default False)

The optimizer includes standard features like weight decay, gradient clipping, and EMA support, making it suitable for various deep learning tasks where adaptive learning rates are beneficial.
