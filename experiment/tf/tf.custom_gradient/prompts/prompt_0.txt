- **Handles Multiple Inputs**: The gradient function must return gradients for each input tensor.
- **Supports Variables**: If the forward function uses trainable variables, the gradient function must accept an additional `variables` argument and return gradients for those variables.
- **Chain Rule Integration**: The custom gradient automatically integrates with TensorFlow's automatic differentiation system via the chain rule.

- Nested custom gradients require careful handling as they don't automatically compute higher-order derivatives.
- Use `tf.stop_gradient` when operations without registered gradients are used within the function to avoid `LookupError`.
