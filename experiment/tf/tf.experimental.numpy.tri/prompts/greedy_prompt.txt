- **M** (int, optional): Number of columns. Defaults to `N` if not provided.
- **k** (int, optional): Diagonal offset.  
  - `k = 0` → main diagonal and below are ones.  
  - `k > 0` → includes diagonals above the main diagonal.  
  - `k < 0` → excludes diagonals below the main diagonal.
- **dtype** (optional): Data type of the output (e.g., `tf.float32`). Default is `tf.float64`.

**Returns:**  
A TensorFlow tensor of shape `(N, M)` with the lower‑triangle pattern.

**Example:**
```python

# [[0 0 0 0]
```
