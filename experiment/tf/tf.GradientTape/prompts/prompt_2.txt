`tf.GradientTape` is a TensorFlow context manager that records operations for automatic differentiation. It enables gradient computation by tracing operations on watched tensors, supporting both first-order and higher-order derivatives.

**Key Features:**
- **Automatic Variable Tracking:** By default, it watches all trainable variables (e.g., `tf.Variable` objects) accessed within its context.
- **Manual Tensor Watching:** Use `watch(tensor)` to explicitly track non-variable tensors.
- **Persistent Mode:** Set `persistent=True` to compute multiple gradients from the same tape (released via garbage collection).
- **Fine-Grained Control:** Disable automatic tracking with `watch_accessed_variables=False` to manually specify watched tensors.

**Core Methods:**
- `gradient(target, sources)`: Computes gradients of `target` with respect to `sources`.
- `jacobian(target, sources)`: Computes the Jacobian matrix for vector-valued outputs.
- `batch_jacobian(target, source)`: Efficiently computes per-example Jacobians under independence assumptions.
- `reset()`: Clears all recorded operations without exiting the context.
- `stop_recording()`: Temporarily suspends operation recording to reduce memory usage.

**Usage Example:**
```python
x = tf.constant(3.0)
with tf.GradientTape() as tape:
    tape.watch(x)
    y = x * x
dy_dx = tape.gradient(y, x)  # Returns 6.0
```

**Notes:**
- Non-persistent tapes can only compute gradients once.
- Supports nested tapes for higher-order derivatives.
- For better performance, wrap tape operations in `@tf.function`.
