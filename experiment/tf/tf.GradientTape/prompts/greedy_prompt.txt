`tf.GradientTape` is a TensorFlow context manager for automatic differentiation. It records operations performed within its scope to compute gradients of outputs with respect to inputs or variables.

**Key Features:**
- **Automatic Tracking:** By default, it watches all trainable variables (e.g., `tf.Variable`). You can manually watch tensors using `watch()`.
- **Gradient Computation:** Use `gradient(target, sources)` to compute derivatives.
- **Higher-Order Derivatives:** Nested tapes enable computation of higher-order gradients.
- **Persistent Mode:** Set `persistent=True` to compute multiple gradients from the same tape (requires manual cleanup).
- **Fine-Grained Control:** Disable automatic variable tracking with `watch_accessed_variables=False` for explicit control.

**Common Methods:**
- `gradient()`: Computes gradients of a target with respect to sources.
- `jacobian()`: Computes the Jacobian matrix.
- `batch_jacobian()`: Efficiently computes per-example Jacobians for batched data.
- `reset()`: Clears all recorded operations.
- `stop_recording()`: Temporarily pauses recording to reduce memory usage.

**Example:**
```python
x = tf.constant(3.0)
with tf.GradientTape() as tape:
    tape.watch(x)
    y = x * x
dy_dx = tape.gradient(y, x)  # Returns 6.0
```

Use `tf.GradientTape` for custom training loops, gradient-based optimizations, and advanced differentiation tasks in TensorFlow.
