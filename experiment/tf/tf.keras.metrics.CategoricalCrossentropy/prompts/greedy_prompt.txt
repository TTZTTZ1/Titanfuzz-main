`tf.keras.metrics.CategoricalCrossentropy` is a metric for evaluating classification models with multiple label classes, where labels are expected to be one-hot encoded. It computes the average cross-entropy between true labels and predicted probabilities.

**Key Parameters:**
- `from_logits`: Set to `True` if predictions are raw logits (default is `False`).
- `label_smoothing`: Smooths labels by redistributing a small value from true to other classes (range [0, 1]).
- `axis`: Axis along which to compute entropy (default is `-1`, the last dimension).

**Usage Example:**
