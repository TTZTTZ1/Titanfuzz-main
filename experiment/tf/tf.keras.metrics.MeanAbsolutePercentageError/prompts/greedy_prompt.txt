The `tf.keras.metrics.MeanAbsolutePercentageError` metric calculates the mean absolute percentage error (MAPE) between true and predicted values. It is defined as:

**Formula:**  
`loss = 100 * mean(abs((y_true - y_pred) / y_true))`

**Key Features:**
- Inherits from `MeanMetricWrapper`, making it a streaming metric that accumulates results over batches.
- Automatically handles sample weighting.
- Commonly used in regression tasks to evaluate prediction accuracy as a percentage.

**Usage:**
- Can be added directly to `model.compile()` as a metric.
- Use `update_state()` to accumulate error statistics and `result()` to compute the final MAPE.
- Reset state with `reset_state()` between epochs or evaluation runs.

**Example:**
```python
m = keras.metrics.MeanAbsolutePercentageError()
m.update_state(y_true, y_pred)
print(m.result())  # Returns the MAPE as a scalar tensor
```

**Note:**  
MAPE can be sensitive to zero or near-zero true values, as division by small numbers may produce large errors.
