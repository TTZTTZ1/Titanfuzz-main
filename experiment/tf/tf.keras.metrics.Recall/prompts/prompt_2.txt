`tf.keras.metrics.Recall` is a metric that calculates the recall (true positive rate) for binary or multi-class classification tasks. It tracks true positives and false negatives to compute:

\[
\text{Recall} = \frac{\text{True Positives}}{\text{True Positives} + \text{False Negatives}}
\]

**Key Features:**
- **Thresholds:** Default is 0.5, but customizable for probability-based predictions.
- **Top-k:** Optionally compute recall based on top-k predicted classes.
- **Class-specific:** Can target a specific class ID for binary evaluation in multi-class settings.
- **Sample weighting:** Supports masking or weighting examples.

**Usage:**
- Integrated via `model.compile()` for training/evaluation.
- Manually updated with `update_state()` for custom loops.

**Example:**
```python
metric = tf.keras.metrics.Recall()
metric.update_state([0, 1, 1, 1], [1, 0, 1, 1])  # y_true, y_pred
print(metric.result().numpy())  # Output: 0.6666667
```

Useful for imbalanced datasets where detecting positive instances is critical.
