`tf.nn.sampled_softmax_loss` is a TensorFlow function designed to efficiently train a softmax classifier over a very large number of classes. Instead of computing the loss over all classes, it estimates the loss by sampling a subset of negative classes, making it significantly faster for tasks with huge vocabularies (e.g., language modeling). It is intended only for training, as it generally underestimates the full softmax loss. For evaluation, the full softmax loss should be used.

- `inputs`: Forward activations from the network.
- `num_sampled`: Number of negative classes to sample per batch.
- `num_classes`: Total number of possible classes.
- `sampled_values`: Optional pre-sampled candidates; defaults to `log_uniform_candidate_sampler`.

**Common Usage:**
- **Training:** Use `sampled_softmax_loss` for speed.
- **Evaluation/Inference:** Compute the full softmax loss using `tf.nn.softmax_cross_entropy_with_logits`.
