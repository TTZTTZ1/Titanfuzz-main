The `tf.keras.metrics.AUC` class approximates the Area Under the Curve (AUC) for ROC or Precision-Recall curves, evaluating binary classifier performance across all operational thresholds. It discretizes predictions into buckets to compute AUC via Riemann summation.

**Key Parameters:**
- `num_thresholds`: Number of thresholds for discretization (default 200).
- `curve`: Type of curve—'ROC' (default) or 'PR' (Precision-Recall).
- `summation_method`: Riemann sum method ('interpolation', 'minoring', or 'majoring').
- `thresholds`: Optional custom threshold list (overrides `num_thresholds`).
- `multi_label`: If True, computes AUC per label and averages; if False, flattens multilabel data.
- `from_logits`: Indicates whether predictions are logits (True) or probabilities (False).

**Usage Example:**
```python
metric = tf.keras.metrics.AUC(num_thresholds=3)
metric.update_state([0, 0, 1, 1], [0, 0.5, 0.3, 0.9])
print(metric.result())  # Output: 0.75
```

**Integration with `compile()`:**
```python
model.compile(optimizer='sgd',
              loss=tf.keras.losses.BinaryCrossentropy(),
              metrics=[tf.keras.metrics.AUC()])
```

**Key Methods:**
- `update_state()`: Accumulates prediction statistics.
- `result()`: Computes and returns the current AUC value.
- `reset_state()`: Clears accumulated metric state.

The metric supports sample weighting and provides configurable approximation methods for AUC calculation.
