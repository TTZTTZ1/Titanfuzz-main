\[
\text{MAPE} = 100 \times \text{mean}\left( \frac{|y_{\text{true}} - y_{\text{pred}}|}{\max(y_{\text{true}}, \epsilon)} \right)
\]

where \(\epsilon\) (default `1e-7`) prevents division by zero.

**Key points:**
- **Inputs:** `y_true` and `y_pred` must have the same shape.
- **Output:** Returns loss values with the same shape as the inputs but without the last dimension.
- **Use case:** Commonly used in regression tasks to interpret error as a percentage.
- **Note:** Sensitive to zero values in `y_true` due to division; the \(\epsilon\) term mitigates this issue.

**Example:**
```python
loss = tf.keras.losses.MAPE(y_true, y_pred)
```
