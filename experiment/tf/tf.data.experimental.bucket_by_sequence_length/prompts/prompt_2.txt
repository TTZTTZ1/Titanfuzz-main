- Pads sequences within each bucket to a uniform size and batches them.
- Reduces computational waste by limiting padding within batches.

**Main Parameters:**
- `element_length_func`: Function to compute each element's length.
- `bucket_boundaries`: List defining bucket boundaries (e.g., `[3, 5]` creates buckets: `<3`, `3-5`, `>5`).
- `bucket_batch_sizes`: Batch sizes for each bucket (must have `len(bucket_boundaries) + 1` entries).
- `padding_values` (optional): Values used for padding (defaults to 0).
- `pad_to_bucket_boundary`: If `True`, pads to the bucket's maximum length.
- `drop_remainder`: Drops incomplete batches if `True`.

**Example Usage:**
