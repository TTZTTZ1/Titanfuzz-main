  1. `query` (shape: `(batch_size, Tq, dim)`).

- `dropout`: Dropout rate applied to attention scores.

**Additional Options:**
- Supports masking for both queries and values.
- Can return attention scores via `return_attention_scores=True`.
- Includes a `use_causal_mask` option for preventing attention to future positions (useful in decoder layers).

This layer is commonly used in sequence-to-sequence tasks, such as neural machine translation, to allow dynamic alignment between sequences.
