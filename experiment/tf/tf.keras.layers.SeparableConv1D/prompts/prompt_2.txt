- `depth_multiplier`: Number of depthwise output channels per input channel (default 1).
- `activation`: Activation function applied to the output (e.g., `'relu'`).

**Input/Output Shapes:**
- Input: 3D tensor with shape depending on `data_format`.
- Output: 3D tensor with same format, where `steps` dimension changes based on `strides`/`padding`, and the channel dimension becomes `filters`.

**Example:**
```python

```
This layer is efficient for sequence data processing (e.g., time-series, text) and supports customization of initializers, regularizers, and constraints for both depthwise and pointwise components.
