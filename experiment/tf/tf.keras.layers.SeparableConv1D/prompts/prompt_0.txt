- `kernel_size`: Size of the depthwise convolution window.
- `strides`: Stride length (default 1).
- `padding`: `"valid"` (no padding) or `"same"` (padding to preserve input size when `strides=1`).
- `data_format`: `"channels_last"` (default, shape: `(batch, steps, channels)`) or `"channels_first"` (shape: `(batch, channels, steps)`).
- `depth_multiplier`: Number of depthwise output channels per input channel (default 1).

**Input Shape:**  
3D tensor: `(batch, steps, channels)` if `data_format="channels_last"`, or `(batch, channels, steps)` if `"channels_first"`.

**Output Shape:**  
3D tensor with same data format, where `steps` may change based on padding/strides, and the channel dimension becomes `filters`.

**Example:**
```python

```

This layer is useful for reducing computational cost while maintaining performance in sequence or temporal data processing.
