`tf.linalg.pinv` computes the **Moore-Penrose pseudo-inverse** of one or more matrices using singular value decomposition (SVD). It is useful for solving least-squares problems when a matrix is not invertible.

**Key features:**
- **Input:** A float-type tensor with at least two dimensions.
- **Method:** Uses SVD, zeroing out singular values below a threshold (`rcond`).
- **Default `rcond`:** `10. * max(num_rows, num_cols) * np.finfo(dtype).eps` (differs from NumPy’s default).
- **Output:** A tensor with the same shape as the input, except the last two dimensions are transposed.

**Example use case:**  
Given a matrix `A`, `tf.linalg.pinv(A)` returns `A_pinv` such that `A_pinv @ A` approximates the identity matrix (for full-rank matrices) or solves the least-squares problem `A @ x = b`.

**Raises errors for non-float inputs or fewer than 2 dimensions.**
