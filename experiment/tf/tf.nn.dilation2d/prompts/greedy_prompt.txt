`tf.nn.dilation2d` performs grayscale morphological dilation on a 4-D input tensor using a 3-D filter. It computes a max-sum correlation where each output element is the maximum over a sliding window of the sum of input values and corresponding filter values.

**Key Parameters:**
- **input**: 4-D tensor of shape `[batch, height, width, depth]`
- **filters**: 3-D tensor of shape `[filter_height, filter_width, depth]`
- **strides**: Must be `[1, stride_height, stride_width, 1]`
- **dilations**: Must be `[1, rate_height, rate_width, 1]`
- **padding**: `"SAME"` or `"VALID"`
- **data_format**: Currently only supports `"NHWC"`

**Operation:**
```
output[b, y, x, c] = max_{dy, dx} (input[b, strides[1]*y + rates[1]*dy, 
                                          strides[2]*x + rates[2]*dx, c] 
                                   + filters[dy, dx, c])
```

**Note:** Max-pooling is a special case when the filter contains all zeros. The function supports various numeric types and has a duality relationship with erosion operations.
