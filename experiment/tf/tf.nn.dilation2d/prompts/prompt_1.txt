# First, here is a general summary of the API's functionality and usage:
---GENERAL SUMMARY START---
`tf.nn.dilation2d` performs grayscale morphological dilation on a 4-D input tensor using a 3-D filter. It computes the maximum sum correlation between the input and filter across spatial dimensions, independently for each channel.

**Key Parameters:**
- **input**: 4-D tensor of shape `[batch, height, width, depth]`
- **filters**: 3-D tensor of shape `[filter_height, filter_width, depth]`
- **strides**: Must be `[1, stride_height, stride_width, 1]`
- **padding**: `"SAME"` or `"VALID"`
- **dilations**: Must be `[1, rate_height, rate_width, 1]`
- **data_format**: Currently only supports `"NHWC"`

**Operation:**  
For each position `(y, x)` and channel `c`, the output is:
```
output[b, y, x, c] = max_{dy, dx} (input[b, strides[1]*y + rates[1]*dy, strides[2]*x + rates[2]*dx, c] + filters[dy, dx, c])
```
This is equivalent to a max-sum correlation (unmirrored filter). Max-pooling is a special case when the filter contains all zeros.

**Note:** Dilation of `input` by `filters` equals the negation of the erosion of `-input` by the reflected `filters`.
---GENERAL SUMMARY END---
