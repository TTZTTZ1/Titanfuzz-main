- **Equation format:** Uses shorthand like `'ij,jk->ik'` for matrix multiplication, following NumPy's `einsum` conventions.
- **Automatic summation:** Repeated indices not in the output are summed over (e.g., `'ij,jk'` implicitly sums over `j`).
- **Common operations:** Supports matrix multiplication, dot products, outer products, transpose, diagonal extraction, trace, and batch operations.
- **Ellipsis support:** Allows broadcasting over unspecified dimensions using `'...'` (e.g., `'...ij,...jk->...ik'`).
- **No broadcasting on named axes:** Matching labels must have the same length; use `tf.squeeze` or `tf.reshape` for length-1 axes.

**Example uses:**
- Matrix multiplication: `tf.einsum('ij,jk->ik', A, B)`
- Dot product: `tf.einsum('i,i->', u, v)`
- Batch matrix multiplication: `tf.einsum('bij,bjk->bik', S, T)`

**Arguments:**
- `equation`: String describing the contraction.
- `*inputs`: Tensors to contract.
- `optimize`: Optional contraction path optimization strategy.
- `name`: Optional operation name.

**Returns:** Contracted tensor with shape determined by the equation.
