- **Equation format:** Uses labels like `'ij,jk->ik'` for matrix multiplication.
- **Summation:** Repeated indices not in the output are summed (e.g., `'ij,jk'` sums over `j`).
- **Common operations:** Supports matrix multiplication, dot products, transposes, traces, batch operations, etc.
- **Ellipsis:** Allows broadcasting over unspecified dimensions (e.g., `'...ij,...jk->...ik'`).
- **No broadcasting on named axes:** Matching labels must have the same length.

**Example:**

- `equation`: String describing the operation (similar to NumPy's `einsum`).
- `*inputs`: Tensors to contract.
- `optimize`: Optional contraction path optimization strategy.
- `name`: Optional operation name.
