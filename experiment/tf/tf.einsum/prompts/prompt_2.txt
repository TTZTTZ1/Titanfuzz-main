**Key features:**
- Supports operations like matrix multiplication, dot products, outer products, transposition, diagonal extraction, trace, and batch operations
- Uses index-based notation (e.g., `'ij,jk->ik'` for matrix multiplication)
- Automatically sums over repeated indices if output indices aren't specified
- Supports ellipsis (`...`) for broadcasting and handling variable numbers of dimensions
- Does not support broadcasting on named axes (matching labels must have same length)

**Example uses:**
- Matrix multiplication: `tf.einsum('ij,jk->ik', A, B)`
- Dot product: `tf.einsum('i,i->', u, v)`
- Batch operations: `tf.einsum('bij,bjk->bik', s, t)`
- Broadcasting with ellipsis: `tf.einsum('...ij,...jk->...ik', s, t)`

**Parameters:**
- `equation`: String describing the operation in NumPy einsum format
- `*inputs`: Tensors to contract
- `optimize`: Optional contraction path optimization strategy
- `name`: Optional operation name
