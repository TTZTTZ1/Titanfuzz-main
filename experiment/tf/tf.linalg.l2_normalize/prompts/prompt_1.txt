`tf.linalg.l2_normalize` (also available as `tf.math.l2_normalize` or `tf.nn.l2_normalize`) performs L2 normalization on a tensor along a specified axis. It divides each element by the L2 norm (Euclidean norm) of the slice along that axis, ensuring the resulting vector has unit length (norm = 1), with a small epsilon value to avoid division by zero.

**Key Parameters:**
- `x`: Input tensor.
- `axis`: Dimension(s) along which to normalize.
- `epsilon`: Lower bound for the norm (default 1e-12).
- `name`: Optional operation name.
- `dim`: Deprecated; use `axis` instead.

**Functionality:**
For a 1-D tensor with `axis=0`, it computes:  
`output = x / sqrt(max(sum(x**2), epsilon))`  
For higher dimensions, each 1-D slice along the specified axis is normalized independently.

**Example:**
```python
x = tf.constant([3.0, 4.0])
tf.math.l2_normalize(x).numpy()  # Output: [0.6, 0.8]
```
