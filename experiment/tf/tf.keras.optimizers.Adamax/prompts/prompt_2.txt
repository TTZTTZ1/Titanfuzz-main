`tf.keras.optimizers.Adamax` is a variant of the Adam optimizer based on the infinity norm, designed for efficient gradient-based optimization, particularly useful for time-variant data like speech. It adjusts the learning rate adaptively using exponentially weighted moments.

**Key Features:**
- **Adaptive Learning Rate:** Uses two moving averages: the first moment (momentum) and the exponentially weighted infinity norm to scale gradients.
- **Default Parameters:** `learning_rate=0.001`, `beta_1=0.9`, `beta_2=0.999`, `epsilon=1e-07`.
- **Additional Options:** Supports weight decay, gradient clipping (per-weight or global), exponential moving average (EMA), loss scaling, and gradient accumulation.

**Update Rule:**
1. Update timestep: `t += 1`
2. Update first moment: `m = beta_1 * m + (1 - beta_1) * g`
3. Update infinity norm: `u = max(beta_2 * u, |g|)`
4. Compute bias-corrected learning rate: `current_lr = learning_rate / (1 - beta_1 ** t)`
5. Update parameter: `w = w - (current_lr * m) / (u + epsilon)`

**Typical Usage:**
```python
optimizer = tf.keras.optimizers.Adamax(learning_rate=0.01)
model.compile(optimizer=optimizer, loss='mse')
```
