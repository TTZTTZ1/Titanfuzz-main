`tf.sigmoid` (also available as `tf.math.sigmoid`) computes the sigmoid activation function element-wise on a tensor. It transforms input values into a range between 0 and 1 using the formula \( y = 1 / (1 + e^{-x}) \). This function is commonly used in machine learning for logistic regression and as an activation in neural networks.

**Key features:**
- **Input:** A tensor of type `float16`, `float32`, `float64`, `complex64`, or `complex128`.
- **Output:** A tensor with the same shape and dtype as the input, where each element is mapped to \((0, 1)\).
- **Behavior:** Large positive inputs approach 1, large negative inputs approach 0, and an input of 0 yields 0.5.
- **Compatibility:** Equivalent to `scipy.special.expit`.

**Example:**
```python
tf.sigmoid([-100.0, 0.0, 100.0])  # Output: ~[0.0, 0.5, 1.0]
```
