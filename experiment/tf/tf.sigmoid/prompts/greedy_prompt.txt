`tf.sigmoid` (also accessible as `tf.math.sigmoid`) computes the sigmoid activation function element-wise on a tensor. It transforms input values into a range between 0 and 1 using the formula \( y = 1 / (1 + e^{-x}) \).

**Key Features:**
- **Input:** A tensor of type `float16`, `float32`, `float64`, `complex64`, or `complex128`.
- **Output:** A tensor with the same shape and dtype as the input, where each element is mapped to \((0, 1)\).
- **Behavior:** Large positive inputs approach 1; large negative inputs approach 0.
- **Aliases:** `tf.nn.sigmoid`; migration aliases are available under `tf.compat.v1`.
- **SciPy Equivalent:** `scipy.special.expit`.

**Example:**
```python
x = tf.constant([-100.0, 0.0, 100.0])
tf.sigmoid(x)  # Outputs approximately [0.0, 0.5, 1.0]
```

Commonly used in logistic regression, neural network activation layers, and probability estimation.
