**Key features:**
- **Input:** A tensor of type `float16`, `float32`, `float64`, `complex64`, or `complex128`.
- **Output:** A tensor of the same type and shape as the input, with each element transformed via the sigmoid function.
- **Behavior:** For large positive inputs, outputs approach 1; for large negative inputs, outputs approach 0.
- **Common use:** Often used in machine learning for binary classification, probability estimation, and as an activation function in neural networks.
- **Compatibility:** Equivalent to `scipy.special.expit`.
