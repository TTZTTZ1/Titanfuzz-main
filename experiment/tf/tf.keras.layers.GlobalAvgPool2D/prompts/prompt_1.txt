`tf.keras.layers.GlobalAveragePooling2D` (also aliased as `GlobalAvgPool2D`) performs global average pooling on 2D spatial data, reducing each feature map to a single average value per channel.

**Key Functionality:**
- **Operation:** Computes the average of all values in each channel across the entire height and width dimensions.
- **Purpose:** Reduces spatial dimensions to 1×1, commonly used to flatten spatial features before a fully connected layer in CNNs.

**Parameters:**
- `data_format`: `"channels_last"` (default, shape: `(batch, height, width, channels)`) or `"channels_first"` (shape: `(batch, channels, height, width)`).
- `keepdims`: If `True`, retains spatial dimensions as size 1 (e.g., `(batch, 1, 1, channels)`); if `False` (default), outputs `(batch, channels)`.

**Input/Output Shapes:**
- **Input:** 4D tensor (batch, height, width, channels).
- **Output:** 
  - With `keepdims=False`: 2D tensor `(batch, channels)`.
  - With `keepdims=True`: 4D tensor with spatial dimensions as 1 (shape depends on `data_format`).

**Example:**
```python
x = np.random.rand(2, 4, 5, 3)  # 2 samples, 4x5 feature maps, 3 channels
y = keras.layers.GlobalAveragePooling2D()(x)  # Output shape: (2, 3)
```

**Use Cases:** Often applied at the end of convolutional networks (e.g., in models like ResNet or MobileNet) to aggregate spatial information before classification.
