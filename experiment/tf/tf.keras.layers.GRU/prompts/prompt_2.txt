The `tf.keras.layers.GRU` is a gated recurrent unit (GRU) layer for sequence modeling, implementing the GRU architecture from Cho et al. (2014). It is a type of recurrent neural network (RNN) layer that processes sequential data and supports both CPU and GPU-accelerated (cuDNN) implementations for optimized performance.

**Key Features:**
- **CuDNN Optimization:** Automatically uses a faster cuDNN implementation on GPUs when specific conditions are met (e.g., `activation='tanh'`, `recurrent_activation='sigmoid'`, no dropout, `reset_after=True`).
- **Two Variants:** Supports two GRU variants based on the order of operations (reset gate applied before or after matrix multiplication). The default (`reset_after=True`) is cuDNN-compatible and allows CPU inference.
- **Flexible Outputs:** Can return either the last output step or the full sequence (`return_sequences`), and optionally the final state (`return_state`).
- **Bidirectional Processing:** Supports processing sequences in reverse (`go_backwards=True`).
- **Stateful Mode:** Allows carrying over states between batches for continuous sequences (`stateful=True`).

**Main Parameters:**
- `units`: Dimensionality of the output space.
- `activation`/`recurrent_activation`: Activation functions for the output and recurrent steps (default: `tanh` and `sigmoid`).
- `dropout`/`recurrent_dropout`: Fraction of units to drop for input and recurrent transformations.
- `return_sequences`: Whether to return all time steps or just the last.
- `return_state`: Whether to return the final state in addition to the output.
- `reset_after`: GRU convention (reset gate before or after matrix multiplication; default `True` for cuDNN compatibility).

**Input/Output:**
- **Input:** 3D tensor of shape `(batch_size, timesteps, features)`.
- **Output:** 
  - If `return_sequences=False`: 2D tensor `(batch_size, units)`.
  - If `return_sequences=True`: 3D tensor `(batch_size, timesteps, units)`.
  - If `return_state=True`: Returns a tuple (output, final_state).

**Common Use Cases:** Sequence modeling tasks such as machine translation, text generation, time series forecasting, and video classification.
