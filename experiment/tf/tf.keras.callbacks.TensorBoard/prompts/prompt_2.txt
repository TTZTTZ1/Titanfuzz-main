- **Metrics & Histograms**: Logs metrics (e.g., loss, accuracy) and optionally computes weight histograms (`histogram_freq`) and embeddings (`embeddings_freq`).

- **Profiling**: Supports batch profiling with `profile_batch`.
- **Custom Summaries**: Allows batch-level custom summaries using `tf.summary` in model code.

**Basic Usage**:
```python

model.fit(x_train, y_train, callbacks=[tensorboard_callback])
```
After training, launch TensorBoard via:
```bash

```

This callback integrates seamlessly with Keras training loops, providing real-time visualization of model performance and structure.
