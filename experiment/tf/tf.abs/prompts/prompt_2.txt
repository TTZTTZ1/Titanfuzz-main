`tf.abs` (also `tf.math.abs`) computes the absolute value of each element in a tensor.  

**Key points:**  
- **Input types:** Supports integer, floating-point, and complex tensors (including `SparseTensor`).  
- **For real numbers:** Returns the same dtype with each element’s absolute value.  
- **For complex numbers:** Returns `float32` or `float64` with magnitude \(\sqrt{a^2 + b^2}\) per element.  
- **SparseTensor handling:** Returns a `SparseTensor` with absolute values applied to its non-zero values.  

**Example:**  
```python
x = tf.constant([-2.25, 3.25])
tf.abs(x)  # [2.25, 3.25]
```
