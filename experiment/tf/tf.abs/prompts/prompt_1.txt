`tf.abs` (or `tf.math.abs`) computes the absolute value of each element in a tensor.  

**Key features:**  
- Supports integer, floating-point, and complex number tensors.  
- For real numbers: returns the absolute value directly.  
- For complex numbers \(a + bj\): returns \(\sqrt{a^2 + b^2}\) as a `float32` or `float64` tensor.

**Example:**  
```python

tf.abs([-2.25+4.75j])  # [5.25594901] (float64)
```
