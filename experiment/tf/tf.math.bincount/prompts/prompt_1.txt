`tf.math.bincount` counts occurrences of integer values in an array, optionally with weights or per-axis binning.

**Key Features:**
- Counts frequency of each integer value in the input array.
- Returns a vector where each index corresponds to a value and its value is the count (or weighted sum).
- Supports `weights` to sum corresponding weights instead of counting.
- With `axis=-1`, performs independent bin counts per row for 2D inputs.
- `binary_output=True` outputs 1 for presence instead of counts.
- Works with Tensor, RaggedTensor, and SparseTensor (missing zeros in sparse tensors are not counted).

**Parameters:**
- `arr`: Input tensor (rank 2 if `axis=-1`).
- `weights`: Optional same-shape tensor for weighted bin increments.
- `minlength`/`maxlength`: Control output vector length.
- `axis`: Flattening axis (`0`, `-1`, or `None`).
- `binary_output`: Output binary presence indicators.

**Returns:**  
A vector (or matrix with `axis=-1`) of counts/weighted sums.

**Note:** On GPU, weighted bin counting requires XLA; otherwise, use `tf.math.unsorted_segment_sum`.
---GENERAL SUMMARY END---
