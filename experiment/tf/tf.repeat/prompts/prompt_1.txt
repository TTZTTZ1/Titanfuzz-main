`tf.repeat` duplicates elements of a tensor along a specified axis, with customizable repetition counts for each element.

**Key parameters:**
- `input`: An N-dimensional tensor.
- `repeats`: A 1-D integer tensor specifying repetitions per element. Must match the length of the specified axis.
- `axis`: The axis along which to repeat (default is `None`, which flattens the tensor first).

**Returns:** A tensor with the same shape as `input`, except expanded along the given axis (or flattened if `axis=None`).

**Examples:**
- `tf.repeat(['a', 'b', 'c'], repeats=[3, 0, 2], axis=0)` → `[b'a', b'a', b'a', b'c', b'c']`
- `tf.repeat([[1, 2], [3, 4]], repeats=[2, 3], axis=0)` repeats rows.
- `tf.repeat([[1, 2], [3, 4]], repeats=[2, 3], axis=1)` repeats columns.
- With `axis=None`, the tensor is flattened before repeating.
---GENERAL SUMMARY END---
