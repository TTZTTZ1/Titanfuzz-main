`tf.while_loop` is a TensorFlow operation that repeatedly executes a loop body as long as a specified condition is true. It is commonly used to implement dynamic loops in TensorFlow graphs, especially within `tf.function` where Python loops are automatically converted.

**Key Features:**
- **Condition and Body:** Requires two callables: `cond` (returns a boolean tensor) and `body` (returns updated loop variables).
- **Loop Variables:** Accepts a structure of tensors (e.g., tuple, list) passed to both `cond` and `body`.
- **Shape Invariants:** Enforces shape consistency across iterations; optional `shape_invariants` allows variable shapes to change within specified bounds.
- **Parallel Execution:** Supports non-strict semantics with `parallel_iterations` to control memory and execution order.
- **Memory Management:** `swap_memory` enables GPU-CPU memory swapping to handle large models.
- **AutoGraph Integration:** Automatically used inside `tf.function` to convert Python `while` loops.

**Common Use Cases:**
- Implementing iterative algorithms (e.g., accumulators, search loops).
- Dynamic sequence processing in RNNs.
- Loops with variable iteration counts.

**Example:**
```python
i = tf.constant(0)
cond = lambda i: i < 10
body = lambda i: (i + 1,)
result = tf.while_loop(cond, body, [i])[0]  # Returns 10
```
