- `filters`: Number of output filters (pointwise convolution).

- `strides`: Convolution stride (default `(1, 1)`).

- `data_format`: `"channels_last"` (default, shape `(batch, height, width, channels)`) or `"channels_first"`.
- `depth_multiplier`: Number of depthwise output channels per input channel (default `1`).
- `activation`: Activation function applied to the output (optional).
- `use_bias`: Whether to add a bias vector.

**Input/Output Shape:**
- **Input:** 4D tensor with shape depending on `data_format`.
- **Output:** 4D tensor with spatial dimensions adjusted by `strides` and `padding`, and channel dimension equal to `filters`.

**Example:**
```python
# Applies separable convolution with 3 filters, 4x4 kernel, stride 3, padding 2, and ReLU activation.
y = keras.layers.SeparableConv2D(3, 4, strides=3, padding=2, activation='relu')(x)
```

This layer is useful for building lightweight convolutional networks, such as MobileNet, by reducing computational cost compared to standard convolutions.
