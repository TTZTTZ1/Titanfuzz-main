  - Options: `return_sequences`, `return_state`, `stateful`, `go_backwards`.

### Common Use Cases:
- Video analysis (action recognition, motion prediction).
- Volumetric medical image sequences.
- Any task requiring joint spatial and temporal feature learning.

### Notes:
- Similar to LSTM but uses convolutions instead of dense transformations.
- Supports masking, dropout, regularization, and constraints.
- Reference: Based on the ConvLSTM architecture by Shi et al., 2015.
