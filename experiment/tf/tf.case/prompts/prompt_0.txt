It takes a list of `(predicate, callable)` pairs, evaluates the predicates, and returns the tensors from the first callable whose predicate is `True`. If no predicate is `True`, it returns the result of an optional `default` callable. Key features include:

- **Exclusive mode**: If `exclusive=True`, it ensures at most one predicate is `True`, raising an error if multiple are.
- **Strict mode**: If `strict=True`, it disables automatic unpacking of singleton lists/tuples in return values.
- **Nested structures**: Supports nested outputs via `tf.nest`, requiring all callables to return the same structure.
- **V2 compatibility**: In TensorFlow 2.x, `pred_fn_pairs` must be a list or tuple (not a dictionary, as tensors are no longer hashable).

**Example**:
