`tf.argsort` returns the indices that would sort a tensor along a specified axis, allowing you to reorder elements without directly sorting the values.  

**Key features:**  
- **Input:** A numeric tensor (1D or higher).  
- **Axis:** Sorts along a given axis (default: last axis).  
- **Direction:** `'ASCENDING'` (default) or `'DESCENDING'`.  
- **Stability:** Optionally stable sort (equal elements keep original order).  
- **Output:** An int32 tensor of indices with the same shape as the input.  

**Common use:**  
Use `tf.gather` with the returned indices to obtain the sorted tensor.  

**Example:**  
```python
values = [1, 10, 26.9, 2.8, 166.32, 62.3]
indices = tf.argsort(values)  # [0, 3, 1, 2, 5, 4]
sorted_values = tf.gather(values, indices)  # Sorted order
```
