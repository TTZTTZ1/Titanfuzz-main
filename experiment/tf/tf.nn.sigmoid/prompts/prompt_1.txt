`tf.nn.sigmoid` (also available as `tf.math.sigmoid` or `tf.sigmoid`) computes the element-wise sigmoid activation function for a given tensor. It transforms each input value \( x \) into a value between 0 and 1 using the formula:

\[
\text{sigmoid}(x) = \frac{1}{1 + e^{-x}}
\]

**Key Features:**
- **Input:** A tensor of type `float16`, `float32`, `float64`, `complex64`, or `complex128`.
- **Output:** A tensor of the same shape and type as the input, with each element mapped to the range (0, 1).
- **Behavior:** For large positive inputs, the output approaches 1; for large negative inputs, it approaches 0.
- **Common Use:** Often used in binary classification and as an activation function in neural networks.
- **Compatibility:** Equivalent to `scipy.special.expit`.

**Example:**
```python
x = tf.constant([-10.0, 0.0, 10.0])
tf.math.sigmoid(x)  # Outputs values close to [0., 0.5, 1.]
```
