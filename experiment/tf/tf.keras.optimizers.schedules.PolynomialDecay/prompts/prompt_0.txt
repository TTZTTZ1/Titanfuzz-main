`tf.keras.optimizers.schedules.PolynomialDecay` implements a polynomial learning rate decay schedule, where the learning rate decreases from an initial value to a specified end value over a given number of steps, following a polynomial curve.

**Key Features:**
- **Decay Formula:**  
  Learning rate = `((initial_learning_rate - end_learning_rate) * (1 - step / decay_steps) ^ power) + end_learning_rate`
- **Cycle Mode (Optional):**  
  If `cycle=True`, the schedule can extend beyond `decay_steps` by restarting the decay cycle.
- **Usage:**  
  Pass the schedule directly to a Keras optimizer as the `learning_rate` argument.
- **Serialization:**  
  Supports saving/loading via `serialize` and `deserialize`.

**Parameters:**
- `initial_learning_rate`: Starting learning rate (float).
- `decay_steps`: Number of steps over which decay occurs (positive integer).
- `end_learning_rate`: Final learning rate (float, default 0.0001).
- `power`: Exponent of the polynomial (float, default 1.0 → linear decay).
- `cycle`: Whether to restart decay cycles (bool, default False).
- `name`: Operation name (string).

**Example:**
```python
lr_schedule = PolynomialDecay(
    initial_learning_rate=0.1,
    decay_steps=10000,
    end_learning_rate=0.01,
    power=0.5  # Square root decay
)

```
