The `tf.keras.layers.Normalization` layer is a preprocessing layer that normalizes continuous features by centering them around 0 with a standard deviation of 1. It applies the transformation `(input - mean) / sqrt(variance)`.

**Key Features:**
- **Adaptation:** You can either predefine the `mean` and `variance` during construction or learn them from your data using the `adapt()` method, which must be called before model training or evaluation.
- **Axis Control:** The `axis` parameter determines which dimensions are normalized separately (default is `-1`, normalizing the last axis per index). Use `axis=None` to normalize all elements globally.
- **Inversion:** Setting `invert=True` allows the layer to reverse the normalization, transforming normalized data back to its original scale.

**Usage:**
1.  **With `adapt()`:** Call `adapt()` on your training data to compute and store the mean and variance.
2.  **Direct Parameters:** Provide known `mean` and `variance` values at layer creation.

**Example Use Cases:** Normalizing features in structured data, audio preprocessing, or any scenario requiring standardized input distributions for improved model training.
