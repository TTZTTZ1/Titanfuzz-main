The `tf.keras.layers.Normalization` layer normalizes continuous input features by shifting and scaling them to have a mean of 0 and standard deviation of 1. It computes `(input - mean) / sqrt(variance)` at runtime.

**Key Features:**
- **Adaptation:** You can either precompute statistics via `adapt()` on a dataset or directly provide `mean` and `variance` during initialization.
- **Axis Control:** Use the `axis` parameter to specify which axes to normalize separately (default is `-1`, normalizing per feature index).
- **Inversion:** Setting `invert=True` allows the layer to reverse the normalization, restoring original values.

**Usage:**
1. **With `adapt()`:** Call `adapt()` on training data before using the layer in `fit()`, `evaluate()`, or `predict()`.
2. **Direct Parameters:** Provide known `mean` and `variance` values at initialization.

**Example:**
```python
layer = keras.layers.Normalization(axis=-1)
layer.adapt(training_data)  # Learn statistics
normalized = layer(input_data)  # Apply normalization
```

This layer is essential for standardizing input data, improving model training stability and performance.
