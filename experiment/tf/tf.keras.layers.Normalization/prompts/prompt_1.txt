`tf.keras.layers.Normalization` is a preprocessing layer that normalizes input features by shifting and scaling them to have a mean of 0 and a standard deviation of 1. It computes `(input - mean) / sqrt(variance)` at runtime.

**Key Features:**
- **Adaptation:** You can call `adapt(data)` to automatically compute and store the mean and variance from your dataset. This must be done before training or inference if `mean` and `variance` are not provided manually.
- **Manual Parameters:** Alternatively, you can directly specify `mean` and `variance` during layer initialization.
- **Axis Control:** The `axis` argument determines which axes are normalized separately (default is `-1`, normalizing each feature independently). Use `axis=None` for global normalization.
- **Inversion:** Setting `invert=True` allows the layer to reverse the normalization, transforming normalized data back to its original scale.

**Common Use Cases:**
- Normalizing continuous features in structured data.
- Preprocessing inputs for models to improve training stability and performance.
- Feature scaling in regression or classification tasks.

**Example:**
```python
layer = tf.keras.layers.Normalization(axis=-1)
layer.adapt(training_data)  # Compute statistics

```

This layer is essential for standardizing data distributions in machine learning workflows.
