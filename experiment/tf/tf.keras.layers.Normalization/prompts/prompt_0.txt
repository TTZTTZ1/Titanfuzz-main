The `tf.keras.layers.Normalization` layer normalizes continuous input features by shifting and scaling them to have zero mean and unit variance. It operates using the formula `(input - mean) / sqrt(variance)`.

**Key Features:**
- **Adaptation:** You can either predefine `mean` and `variance` during initialization or learn them from data using the `adapt()` method, which must be called before training or inference.
- **Axis Control:** The `axis` parameter determines how normalization is applied—per index along specified axes (default is `-1`, the last axis) or globally (`axis=None`).
- **Inversion:** Setting `invert=True` allows the layer to reverse the normalization, restoring inputs to their original scale.

**Usage:**
- Use `adapt()` with a dataset to compute and store mean/variance automatically.
- Alternatively, provide fixed `mean` and `variance` values during layer creation.
- The layer integrates seamlessly into Keras models for preprocessing.

**Example Use Cases:** Normalizing numerical features in structured data, audio signals, or any continuous input to improve model training stability and performance.
