The `tf.keras.layers.ReLU` layer implements the Rectified Linear Unit activation function with configurable parameters. It applies an element-wise activation defined by:

- **Default behavior:** `f(x) = max(x, 0)`
- **Extended formula:**  
  - `f(x) = max_value` if `x >= max_value`  
  - `f(x) = x` if `threshold <= x < max_value`  
  - `f(x) = negative_slope * (x - threshold)` otherwise

**Key parameters:**
- `max_value` (float ≥ 0): Caps activation output (default `None`, no cap).
- `negative_slope` (float ≥ 0): Slope for negative values (default `0.0`, standard ReLU).
- `threshold` (float ≥ 0): Activation threshold (default `0.0`).

**Example usage:**
```python
relu_layer = tf.keras.layers.ReLU(max_value=10, negative_slope=0.5, threshold=0)
result = relu_layer([-10, -5, 0, 5, 10])  # Output: [-5., -2.5, 0., 5., 10.]
```

This layer is commonly used in neural networks to introduce non-linearity and can be customized for variants like Leaky ReLU or clipped ReLU.
