`tf.pad` pads a tensor along each dimension with specified values.  

**Key parameters:**  
- `tensor`: Input tensor to pad.  
- `paddings`: Integer tensor of shape `[n, 2]` where `n` is the rank of `tensor`. For each dimension `D`, `paddings[D, 0]` is the padding before the content and `paddings[D, 1]` is the padding after.  
- `mode`: Padding mode — `"CONSTANT"` (default), `"REFLECT"`, or `"SYMMETRIC"`.  
- `constant_values`: Fill value for `"CONSTANT"` mode (default 0).  

**Output size for dimension D:**  
`paddings[D, 0] + tensor.dim_size(D) + paddings[D, 1]`  

**Mode restrictions:**  
- `"REFLECT"`: Padding size ≤ `tensor.dim_size(D) - 1`  
- `"SYMMETRIC"`: Padding size ≤ `tensor.dim_size(D)`  

**Example:**  
For a 2×3 tensor, with `paddings = [[1,1], [2,2]]`, the padded tensor will be 4×7.
