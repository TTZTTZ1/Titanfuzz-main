`tf.pad` pads a tensor by adding values around its edges according to specified padding widths.  

**Key parameters:**  
- `tensor`: Input tensor to pad.  
- `paddings`: Integer tensor of shape `[n, 2]` where `n` is the rank of `tensor`. For each dimension `D`, `paddings[D, 0]` is the number of values to add before the tensor, and `paddings[D, 1]` is the number to add after.  
- `mode`: Padding mode:  
  - `"CONSTANT"`: Pads with a constant value (default 0).  
  - `"REFLECT"`: Pads with mirror reflections of the tensor (padding size ≤ dimension size − 1).  
  - `"SYMMETRIC"`: Pads with symmetric reflections (padding size ≤ dimension size).  
- `constant_values`: Scalar value used for constant padding (default 0).  

**Output size for dimension D:**  
`paddings[D, 0] + tensor.dim_size(D) + paddings[D, 1]`  

**Example:**  
For a 2D tensor `[[1, 2, 3], [4, 5, 6]]` with `paddings = [[1, 1], [2, 2]]`, the padded tensor will have shape `(4, 7)` after adding 1 row above/below and 2 columns left/right.
