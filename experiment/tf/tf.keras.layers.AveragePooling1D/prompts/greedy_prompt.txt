- `strides`: Step size for moving the window (defaults to `pool_size`).
- `padding`: `"valid"` (no padding) or `"same"` (output length matches input length when strides=1).
- `data_format`: `"channels_last"` (default, shape: `[batch, steps, features]`) or `"channels_first"` (shape: `[batch, features, steps]`).

**Functionality:**
- Reduces the temporal dimension (steps) by averaging values within each window.
- Output shape depends on padding:  
  - `"valid"`: `output_length = (input_length - pool_size) // strides + 1`  
  - `"same"`: `output_length = ceil(input_length / strides)`

This layer is commonly used in sequence models to reduce dimensionality while preserving feature information.
