- `input`: 4D tensor with shape determined by `data_format`.
- `ksize`: Size of the pooling window (int or list of length 1, 2, or 4).
- `strides`: Stride of the sliding window (int or list of length 1, 2, or 4).
- `padding`: Either `"VALID"` (no padding) or `"SAME"` (padding to preserve size), or an explicit padding list.
- `data_format`: Layout of the input (`"NHWC"`, `"NCHW"`, or `"NCHW_VECT_C"`).

**Functionality:**  
It slides a window of size `ksize` across the input, moving by `strides` steps, and outputs the maximum value in each window. This reduces spatial dimensions while retaining the most prominent features.

**Example usage:**
