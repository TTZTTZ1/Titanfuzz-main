The `tf.keras.layers.AveragePooling2D` layer performs 2D average pooling to downsample spatial dimensions (height and width) by computing the average value within a sliding window for each input channel.

**Key Parameters:**
- `pool_size`: Size of the pooling window (int or tuple of 2 ints).
- `strides`: Step size for sliding the window (defaults to `pool_size`).
- `padding`: `"valid"` (no padding) or `"same"` (padding to preserve input dimensions).
- `data_format`: `"channels_last"` (default, shape: `[batch, height, width, channels]`) or `"channels_first"` (shape: `[batch, channels, height, width]`).

**Functionality:**
- Reduces spatial dimensions while retaining depth (channels).
- Output size depends on `padding`:
  - `"valid"`: `output_size = floor((input_size - pool_size) / strides) + 1`
  - `"same"`: `output_size = floor((input_size - 1) / strides) + 1`

**Use Case:** Commonly used in convolutional neural networks to progressively reduce feature map size, decrease computational load, and extract dominant features.
