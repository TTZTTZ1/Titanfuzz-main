`tf.keras.layers.Embedding` is a Keras layer that converts integer indices into dense, fixed-size vector representations. It is commonly used for representing categorical data, such as words in natural language processing.

**Key Features:**
- **Input:** 2D integer tensor of shape `(batch_size, sequence_length)`.
- **Output:** 3D tensor of shape `(batch_size, sequence_length, output_dim)`.
- **Vocabulary Size:** Defined by `input_dim`, which must be ≥ the maximum integer index + 1.
- **Masking:** Optionally mask zero values with `mask_zero=True` (requires index 0 to be reserved for padding).

**Main Parameters:**
- `input_dim`: Vocabulary size.
- `output_dim`: Dimensionality of the embedding vectors.
- `embeddings_initializer/regularizer/constraint`: Control initialization and regularization of the embedding matrix.
- `lora_rank`: Enables Low-Rank Adaptation (LoRA) for efficient fine-tuning by decomposing the embedding matrix into lower-rank trainable matrices.

**Example:**
```python
embedding_layer = tf.keras.layers.Embedding(input_dim=1000, output_dim=64)
# Input: (batch, 10) → Output: (batch, 10, 64)
```

**Use Cases:** Word embeddings, categorical feature encoding, and any task requiring dense representations of discrete tokens.
