`tf.keras.layers.Embedding` is a layer that converts integer indices into dense, fixed-size vectors. It is commonly used for representing categorical data (like words in text) in neural networks.

**Key Features:**
- **Input:** 2D integer tensor `(batch_size, sequence_length)` with values in range `[0, input_dim-1]`.
- **Output:** 3D tensor `(batch_size, sequence_length, output_dim)` where each integer is replaced by its embedding vector.
- **Parameters:**
  - `input_dim`: Vocabulary size (max index + 1).
  - `output_dim`: Dimension of the embedding vectors.
  - `mask_zero`: If `True`, treats index 0 as padding and masks it (requires `input_dim = vocabulary size + 1`).
  - `embeddings_initializer/regularizer/constraint`: Control initialization, regularization, and constraints on the embedding matrix.
  - `lora_rank`: Enables Low-Rank Adaptation (LoRA) for efficient fine-tuning by decomposing the embedding matrix into lower-rank trainable matrices.

**Example:**
```python
model = keras.Sequential()
model.add(keras.layers.Embedding(1000, 64))  # 1000-word vocabulary → 64-dim vectors
```

**Use Cases:** Word embeddings, categorical feature representation, and any task involving mapping discrete tokens to continuous vectors.
