The `tf.keras.layers.Embedding` layer converts integer indices into dense, fixed-size vector representations. It is commonly used for mapping discrete tokens (like words) to continuous vectors in natural language processing tasks.

**Key Parameters:**
- `input_dim`: Vocabulary size (max integer index + 1).
- `output_dim`: Dimensionality of the embedding vectors.
- `mask_zero`: If `True`, treats index 0 as padding and masks it (requires `input_dim = vocabulary size + 1`).
- `lora_rank`: Enables Low-Rank Adaptation (LoRA) for efficient fine-tuning by decomposing the embedding matrix into lower-rank trainable matrices.

**Input/Output:**
- **Input:** 2D tensor of shape `(batch_size, sequence_length)`.
- **Output:** 3D tensor of shape `(batch_size, sequence_length, output_dim)`.

**Example:**
```python
model = keras.Sequential()
model.add(keras.layers.Embedding(1000, 64))  # Maps indices 0-999 to 64D vectors
```

**Additional Features:**
- Supports custom initializers, regularizers, and constraints for the embedding matrix.
- Includes `enable_lora()` method to apply LoRA after layer creation.
- Compatible with quantization for optimized inference.
