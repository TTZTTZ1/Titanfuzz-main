`tf.nn.experimental.stateless_dropout` is a deterministic dropout function that randomly sets input elements to zero with probability `rate` to prevent overfitting, while scaling the remaining elements to preserve the expected value. Unlike the stateful `tf.nn.dropout`, it guarantees reproducible results for the same `seed` across multiple calls. Key features include:

- **Deterministic output**: Same `seed` always produces the same dropout mask.
- **Parameters**: 
  - `x`: Floating-point input tensor.
  - `rate`: Drop probability (scalar, 0 ≤ rate < 1).
  - `seed`: Integer tensor of shape `[2]` for RNG control.
  - `noise_shape`: Optional tensor to broadcast dropout mask (e.g., drop entire channels).
  - `rng_alg`: RNG algorithm (defaults to `"auto_select"`).
- **Use case**: Preferred over stateful dropout for reproducibility in distributed training or controlled randomness.

Example:
```python
x = tf.ones([3,5])
output = tf.nn.experimental.stateless_dropout(x, rate=0.5, seed=[1, 0])
```
Raises `ValueError` if `rate` is invalid or `x` is non-floating-point.
