`tf.nn.experimental.stateless_dropout` is a deterministic dropout function that randomly sets input elements to zero to prevent overfitting, while preserving the expected value by scaling the remaining elements. Unlike the stateful `tf.nn.dropout`, it guarantees reproducible results for the same `seed` input.

**Key Features:**
- **Stateless & Reproducible:** Identical `seed` always produces the same dropout mask; different seeds yield different masks.
- **Rate-based Dropout:** Each element is independently dropped with probability `rate` (scalar between 0 and 1), and kept elements are scaled by `1/(1-rate)`.
- **Customizable Mask Shape:** Optional `noise_shape` allows broadcasting the dropout mask (e.g., to drop entire channels uniformly).
- **Controlled RNG:** Uses `seed` (a 2-element integer tensor) and optional `rng_alg` to govern random number generation.

**Arguments:**
- `x`: Floating-point input tensor.
- `rate`: Drop probability (0 ≤ rate < 1).
- `seed`: 2-integer tensor determining the random mask.
- `rng_alg` (optional): RNG algorithm.
- `noise_shape` (optional): Broadcastable shape for the dropout mask.
- `name` (optional): Operation name.

**Returns:** A tensor of the same shape and dtype as `x`.

**Use Case:** Preferred over stateful dropout when reproducibility across runs is required, such as in distributed training or debugging.
