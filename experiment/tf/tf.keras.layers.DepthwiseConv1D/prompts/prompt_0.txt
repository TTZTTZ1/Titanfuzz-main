`tf.keras.layers.DepthwiseConv1D` performs a 1D depthwise convolution, where each input channel is convolved separately with its own set of filters. Unlike standard convolution, it does not mix information across input channels. This layer is often used as the first step in depthwise separable convolutions.

**Key Parameters:**
- `kernel_size`: Size of the convolution window.
- `strides`: Stride length; incompatible with `dilation_rate > 1`.
- `padding`: `"valid"` (no padding) or `"same"` (padding to maintain size when `strides=1`).
- `depth_multiplier`: Number of output channels generated per input channel. Total output channels = `input_channels * depth_multiplier`.
- `data_format`: `"channels_last"` (default, shape: `[batch, steps, channels]`) or `"channels_first"` (shape: `[batch, channels, steps]`).

**Input/Output Shapes:**
- **Input:** 3D tensor with shape depending on `data_format`.
- **Output:** 3D tensor where the channel dimension expands by `depth_multiplier`.

**Example:**
```python
x = np.random.rand(4, 10, 12)  # 4 samples, 10 steps, 12 channels
y = keras.layers.DepthwiseConv1D(3, strides=2, depth_multiplier=3)(x)
print(y.shape)  # (4, 4, 36)  # 36 = 12 channels * 3
```

This layer is efficient for channel-wise feature extraction while keeping computational costs lower than standard convolutions.
