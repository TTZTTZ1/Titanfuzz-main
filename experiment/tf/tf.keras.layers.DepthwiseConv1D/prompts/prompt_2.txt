The `tf.keras.layers.DepthwiseConv1D` layer performs a 1D depthwise convolution, where each input channel is convolved with a separate kernel (depthwise kernel). This is the first step in depthwise separable convolution and does not mix information across input channels.

**Key Features:**
- **Depthwise Operation:** Each input channel is processed independently with its own kernel.
- **Depth Multiplier:** The `depth_multiplier` parameter controls how many output channels are generated per input channel, resulting in `input_channels * depth_multiplier` total output channels.
- **No Cross-Channel Mixing:** Unlike standard convolution, it does not combine information from different input channels.

**Main Parameters:**
- `kernel_size`: Size of the convolution window.
- `strides`: Stride length (must be 1 if `dilation_rate > 1`).
- `padding`: `"valid"` (no padding) or `"same"` (padding to maintain size when `strides=1`).
- `depth_multiplier`: Number of output channels per input channel.
- `data_format`: `"channels_last"` (default, shape: `[batch, steps, channels]`) or `"channels_first"` (shape: `[batch, channels, steps]`).
- `dilation_rate`: Dilation rate for dilated convolution.

**Input/Output Shapes:**
- **Input:** 3D tensor with shape depending on `data_format`.
- **Output:** 3D tensor with channels multiplied by `depth_multiplier`.

**Example:**
```python
x = np.random.rand(4, 10, 12)  # 4 samples, 10 steps, 12 channels
y = keras.layers.DepthwiseConv1D(kernel_size=3, strides=2, depth_multiplier=3)(x)
print(y.shape)  # (4, 4, 36)  # 36 channels = 12 * 3
```

This layer is useful for lightweight, channel-wise feature extraction in 1D sequences like audio or time-series data.
