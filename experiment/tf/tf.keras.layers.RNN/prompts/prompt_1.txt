The `tf.keras.layers.RNN` class is the base layer for implementing recurrent neural networks in TensorFlow. It wraps one or more RNN cells to process sequential input data.

**Key Features:**
- **Cell-based architecture**: Accepts a single RNN cell or a list of cells (stacked RNN).
- **Flexible output control**: Can return either the final output or the full sequence (`return_sequences`), and optionally the final state (`return_state`).
- **Bidirectional processing**: Supports processing sequences backwards (`go_backwards`).
- **State management**: Enables stateful RNNs for batch-to-batch state persistence.
- **Masking support**: Compatible with masked timesteps in variable-length sequences.

**Input/Output:**
- **Input shape**: 3D tensor `(batch_size, timesteps, features)`
- **Output shape**: 
  - Default: `(batch_size, output_size)`
  - With `return_sequences`: `(batch_size, timesteps, output_size)`
  - With `return_state`: List containing output tensor plus state tensors

**Common Use Cases:**
- Time series forecasting
- Sequence classification/generation
- Stacked RNN architectures

The layer provides both symbolic and imperative execution modes, with options for unrolling (for short sequences) and custom initial states.
