`tf.data.experimental.make_csv_dataset` reads CSV files into a batched TensorFlow dataset, where each batch is returned as a tuple `(features, labels)`. It automatically handles parsing, batching, shuffling, and type inference, making it suitable for machine learning workflows.

**Key features:**
- **Automatic batching:** Groups rows into batches of size `batch_size`.
- **Feature/label separation:** If `label_name` is specified, that column is separated from the features dictionary.
- **Flexible input:** Supports multiple files via glob patterns and optional column selection.
- **Shuffling and repetition:** Configurable shuffling and epoch control.
- **Type inference:** Infers column types from data if defaults are not provided.

**Common use cases:**
- Loading CSV data for training models.
- Preprocessing tabular data in TensorFlow pipelines.

**Example:**
```python
dataset = tf.data.experimental.make_csv_dataset(
    file_pattern="data.csv",
    batch_size=32,
    label_name="target"

```
This creates a dataset where each batch contains feature tensors (all columns except "target") and label tensors (the "target" column).
