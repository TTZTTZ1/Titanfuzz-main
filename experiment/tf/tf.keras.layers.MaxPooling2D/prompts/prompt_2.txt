`tf.keras.layers.MaxPooling2D` is a pooling layer for 2D spatial data (e.g., images) that downsamples input by taking the maximum value within a sliding window.

**Key Parameters:**
- `pool_size`: Size of the pooling window (default: `(2, 2)`).
- `strides`: Step size of the window (defaults to `pool_size` if not set).
- `padding`: `"valid"` (no padding) or `"same"` (padding to preserve input dimensions).
- `data_format`: `"channels_last"` (default, shape: `[batch, height, width, channels]`) or `"channels_first"`.

**Functionality:**
- Reduces spatial dimensions (height, width) while preserving depth (channels).
- Output size depends on `padding`:
  - `"valid"`: `output_size = floor((input_size - pool_size) / strides) + 1`
  - `"same"`: Output matches input spatial dimensions when `strides=1`.

**Example:**
```python
layer = MaxPooling2D(pool_size=(2, 2), strides=(1, 1), padding="valid")
input = tf.constant([[[[1., 2., 3.], [4., 5., 6.], [7., 8., 9.]]]])
output = layer(input)  # Downsamples via max pooling
```
