`tf.reduce_all` (or `tf.math.reduce_all`) performs a logical AND reduction across specified dimensions of a boolean tensor. It returns `True` only if all elements in the reduced dimensions are `True`. 

**Key parameters:**
- `input_tensor`: The boolean tensor to reduce.
- `axis`: Dimensions to reduce; if `None`, reduces all dimensions.
- `keepdims`: If `True`, retains reduced dimensions with size 1.
- `name`: Optional operation name.

**Example:**
```python
x = tf.constant([[True, True], [False, False]])
tf.reduce_all(x)               # False
tf.reduce_all(x, axis=0)       # [False, False]
tf.reduce_all(x, axis=1)       # [True, False]
```

**Note:** Equivalent to NumPy's `np.all`.
