`tf.keras.optimizers.schedules.CosineDecayRestarts` implements a cosine annealing learning rate schedule with periodic warm restarts, based on the SGDR (Stochastic Gradient Descent with Warm Restarts) paper.  

**Key features:**  
- **Cosine decay with restarts:** The learning rate follows a cosine curve from the initial value down to a minimum (`alpha`), then restarts to a higher value at the end of each cycle.  
- **Configurable cycles:** Each restart cycle can be longer (`t_mul`) and start with a different initial learning rate (`m_mul`).  
- **Usage:** Pass the schedule directly to a Keras optimizer as the learning rate argument. It computes the decayed learning rate based on the current training step.  

**Main parameters:**  
- `initial_learning_rate`: Starting learning rate.  
- `first_decay_steps`: Steps in the first decay cycle.  
- `t_mul`: Multiplier for the length of each subsequent cycle.  
- `m_mul`: Multiplier for the learning rate at the start of each new cycle.  
- `alpha`: Minimum learning rate as a fraction of the initial rate.  

**Example:**  
```python
schedule = CosineDecayRestarts(initial_learning_rate=0.1, first_decay_steps=1000)
optimizer = tf.keras.optimizers.Adam(learning_rate=schedule)
```
