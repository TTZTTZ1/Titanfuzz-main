`tf.keras.optimizers.schedules.CosineDecayRestarts` implements a cosine annealing learning rate schedule with periodic warm restarts, based on the SGDR (Stochastic Gradient Descent with Warm Restarts) paper.  

**Key features:**  
- **Cosine decay with restarts:** The learning rate follows a cosine curve that decays from the initial value to a minimum (`alpha`), then restarts with a potentially higher learning rate and longer cycle.  
- **Adjustable cycles:**  
  - `first_decay_steps`: Steps in the first decay cycle.  
  - `t_mul`: Multiplier to extend each subsequent cycle’s length.  
  - `m_mul`: Multiplier to adjust the learning rate after each restart.  
- **Minimum learning rate:** `alpha` sets the lower bound as a fraction of `initial_learning_rate`.  

**Usage:**  
Pass the schedule directly to a Keras optimizer. It computes the decayed learning rate based on the current step, supporting serialization via `serialize`/`deserialize`.  

**Example:**  
```python
schedule = CosineDecayRestarts(
    initial_learning_rate=0.1,
    first_decay_steps=1000,
    t_mul=2.0,
    m_mul=0.9,
    alpha=0.01
)
optimizer = tf.keras.optimizers.Adam(learning_rate=schedule)
```
