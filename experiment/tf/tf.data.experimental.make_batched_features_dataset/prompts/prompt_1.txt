**Key functionality:**
- Reads `Example` records from TFRecord files (default reader)
- Parses features into dictionaries of tensors or sparse tensors
- Batches, shuffles, and prefetches data for efficient training
- Optionally extracts labels if `label_key` is specified

**Main parameters:**
- `file_pattern`: File path(s) to TFRecord files
- `batch_size`: Number of records per batch
- `features`: Dictionary defining feature names and types (FixedLenFeature/VarLenFeature)
- `label_key`: Optional key to separate labels from features
- `shuffle`, `shuffle_buffer_size`: Control data shuffling
- `num_epochs`: Number of dataset repetitions (None = infinite)

**Returns:** A Dataset yielding batched feature dictionaries (or feature-label tuples if `label_key` is provided).

**Common use case:** Efficient input pipelines for training models on TFRecord data with minimal boilerplate code.
