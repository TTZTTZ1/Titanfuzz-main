`tf.data.experimental.make_batched_features_dataset` creates a batched TensorFlow Dataset from TFRecord files containing `Example` protos. - Reads TFRecord files matching a pattern.
- Parses `Example` protos into feature dictionaries using `FixedLenFeature` or `VarLenFeature`.
- Batches, shuffles, and prefetches data for efficient training.
- Supports multi-threaded reading and parsing.

**Main parameters:**
- `file_pattern`: File path(s) or glob pattern for input data.
- `batch_size`: Number of examples per batch.
- `features`: Dictionary defining how to parse each feature.
- `label_key`: Optional key to extract labels into a separate tensor.
- `shuffle`, `shuffle_buffer_size`: Control data shuffling.
- `num_epochs`: Number of times to iterate through the dataset.

**Returns:** A `Dataset` yielding batched feature dictionaries (or tuples of features and labels if `label_key` is provided).
