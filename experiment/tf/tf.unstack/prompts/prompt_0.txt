**Key features:**
- **Input:** A rank-R tensor (R > 0).
- **Output:** A list of rank-(R-1) tensors. The number of tensors equals the size of the input along the chosen `axis`.
- **Axis:** Default is 0 (first dimension). Supports negative indexing.
- **Parameter `num`:** Optional; specifies the length of the target axis. Required in graph mode (`@tf.function`) if the axis length is unknown.

**Common use cases:**
1. Converting tensor slices into separate tensors (e.g., splitting batch data or channel dimensions).
2. Required inside `@tf.function` because symbolic tensors are not iterable—Python unpacking (e.g., `a, b, c = t`) fails here.

**Example:**
```python
x = tf.constant([[1, 2], [3, 4]])  # shape (2, 2)
a, b = tf.unstack(x)  # a = [1, 2], b = [3, 4]; each shape (2,)
```

**Note:** If the axis length is unknown in graph mode, provide `num` explicitly or use `tf.TensorArray` for dynamic unstacking.
