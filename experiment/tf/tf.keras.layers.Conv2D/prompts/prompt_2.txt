The `tf.keras.layers.Conv2D` layer performs 2D convolution on input data, commonly used in image processing and computer vision models. It applies a set of learnable filters to extract spatial features from input tensors.

**Key Parameters:**
- `filters`: Number of output channels (dimensionality of output space)
- `kernel_size`: Height and width of the convolution window (int or tuple)
- `strides`: Step size for moving the convolution window (default: (1,1))
- `padding`: Either "valid" (no padding) or "same" (padding to maintain spatial dimensions when strides=1)
- `activation`: Optional activation function applied to outputs

**Input/Output Shapes:**
- Input: 4D tensor `(batch, height, width, channels)` for channels_last format
- Output: 4D tensor with same batch size, new spatial dimensions, and `filters` channels

**Core Functionality:**
- Creates convolution kernel convolved with layer input
- Optionally adds bias vector and applies activation function
- Supports dilation, groups, regularization, and constraints
- Raises ValueError if both `strides > 1` and `dilation_rate > 1`

**Example:**
```python
# Creates 32 filters with 3x3 kernel and ReLU activation
layer = Conv2D(32, 3, activation='relu')
# Reduces spatial dimensions from 10x10 to 8x8 with valid padding
```
