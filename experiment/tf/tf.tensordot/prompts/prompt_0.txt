`tf.tensordot` performs tensor contraction between two tensors `a` and `b` along specified axes, generalizing operations like matrix multiplication and outer product.

**Key functionality:**
- Sums products of elements from `a` and `b` over contracted axes.
- Equivalent to `numpy.tensordot`.
- Generalizes:  
  - `axes=1` for matrices → matrix multiplication.  
  - `axes=0` → outer product.  
  - Custom axes via list/`Tensor` specify contraction dimensions.

**Parameters:**
- `a`, `b`: Tensors of same type (`float32`/`float64`).
- `axes`:  
  - Scalar `N`: Contracts last `N` axes of `a` with first `N` axes of `b`.  
  - List/`Tensor` shape `[2, k]`: Specifies exact axes to contract for `a` (row 0) and `b` (row 1).
- `name`: Optional operation name.

**Returns:**  
Tensor with same type as `a`. Output order = `order(a) + order(b) - 2 * len(axes[0])`.

**Raises:**  
`ValueError` for incompatible shapes; `IndexError` if axes exceed tensor ranks.
