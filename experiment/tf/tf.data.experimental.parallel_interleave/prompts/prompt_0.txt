It applies `map_func` to input elements to create nested datasets, then interleaves their outputs.

**Key parameters:**
- `map_func`: Function that creates a dataset from an input element.
- `cycle_length`: Number of datasets to process in parallel.
- `block_length`: Number of consecutive elements taken from each dataset before switching.
- `sloppy`: Allows out-of-order element production for performance, sacrificing determinism.

dataset = filenames.apply(

        lambda f: tf.data.TFRecordDataset(f),
        cycle_length=4))

**Note:** This function is deprecated. Use `tf.data.Dataset.interleave(..., num_parallel_calls=tf.data.AUTOTUNE)` with `tf.data.Options.deterministic` for sloppy execution instead.
