`tf.lite.experimental.QuantizationDebugOptions` configures a debugger for analyzing quantization errors in TensorFlow Lite models. It allows you to define custom metrics for evaluating differences between the original float model and its quantized version at both the layer and model levels.

**Key parameters:**
- `layer_debug_metrics`: Functions to compare float vs. dequantized layer outputs.
- `model_debug_metrics`: Functions to compare overall model outputs (e.g., accuracy).
- `layer_direct_compare_metrics`: Functions for direct comparison using scale/zero-point.
- `denylisted_ops`/`denylisted_nodes`: Exclude specific ops or nodes from quantization.
- `fully_quantize`: If `True`, quantizes both the model body and input/output tensors.

This tool helps identify and diagnose quantization-related performance degradation.
