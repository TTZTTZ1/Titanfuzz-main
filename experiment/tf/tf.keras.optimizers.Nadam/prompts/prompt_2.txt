The `tf.keras.optimizers.Nadam` optimizer combines Adam (adaptive moment estimation) with Nesterov momentum, offering adaptive learning rates with improved convergence properties. Key features include:

- **Adaptive Learning Rates**: Uses moving averages of gradients (momentum) and squared gradients (velocity) for parameter updates.
- **Nesterov Momentum**: Incorporates lookahead momentum for more stable updates.
- **Configurable Hyperparameters**: 
  - `learning_rate`: Base step size (default 0.001)
  - `beta_1`: Exponential decay for momentum (default 0.9)
  - `beta_2`: Exponential decay for velocity (default 0.999)
  - `epsilon`: Numerical stability constant (default 1e-7)
- **Advanced Options**: Supports weight decay, gradient clipping, exponential moving averages (EMA), loss scaling for mixed precision, and gradient accumulation.

**Typical Usage**:
```python
optimizer = tf.keras.optimizers.Nadam(learning_rate=0.002, beta_1=0.9, beta_2=0.999)
model.compile(optimizer=optimizer, loss='categorical_crossentropy')
```

The optimizer automatically tracks momentum and velocity variables for each trainable parameter during training. It's particularly effective for problems where standard Adam shows oscillatory behavior or slow convergence.
