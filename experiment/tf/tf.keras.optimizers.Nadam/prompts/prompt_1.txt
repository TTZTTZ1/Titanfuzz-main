The `tf.keras.optimizers.Nadam` optimizer combines **Adam** with **Nesterov momentum**, enhancing gradient-based optimization for neural networks. It adapts learning rates per parameter using moving averages of gradients (first moment) and squared gradients (second moment), while incorporating Nesterov momentum for improved convergence.

### Key Features:
- **Adaptive Learning Rates**: Uses exponential moving averages of gradients (`beta_1`) and squared gradients (`beta_2`) to adjust per-parameter learning rates.
- **Nesterov Momentum**: Integrates lookahead momentum for more stable updates.
- **Numerical Stability**: Includes an epsilon term to prevent division by zero.
- **Additional Optimizations**: Supports weight decay, gradient clipping, exponential moving averages (EMA), loss scaling for mixed precision, and gradient accumulation.

### Typical Usage:
```python
optimizer = tf.keras.optimizers.Nadam(
    learning_rate=0.001,
    beta_1=0.9,
    beta_2=0.999,
    epsilon=1e-07
)
model.compile(optimizer=optimizer, ...)
```

### Common Applications:
- Training deep neural networks where adaptive learning rates and momentum are beneficial.
- Tasks requiring stable convergence, such as image classification, NLP, and reinforcement learning.

Nadam is particularly effective for problems where Adam performs well but can benefit from the added stability of Nesterov momentum.
