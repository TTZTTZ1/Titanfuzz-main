---GENERAL SUMMARY START---
`tf.GradientTape` is a TensorFlow context manager for automatic differentiation. It records operations performed on "watched" tensors, enabling gradient computation via its `gradient()` method.

**Key Features:**
- **Automatic variable tracking:** By default, it watches all trainable variables (e.g., `tf.Variable`).
- **Manual tensor watching:** Use `watch(tensor)` to track non-variable tensors.
- **Nested tapes:** Support for higher-order derivatives.
- **Persistent mode:** Allows multiple gradient calls when `persistent=True`.
- **Fine-grained control:** Disable auto-tracking with `watch_accessed_variables=False`.

**Core Methods:**
- `gradient(target, sources)`: Computes gradients.
- `jacobian(target, sources)`: Computes Jacobian matrices.
- `batch_jacobian(target, source)`: Efficient per-example Jacobians.
- `reset()`: Clears all recorded operations.
- `stop_recording()`: Temporarily pauses recording.

**Common Use Case:**
```python
with tf.GradientTape() as tape:
    tape.watch(x)
    y = x**2
dy_dx = tape.gradient(y, x)  # Returns 2x
```

**Notes:**
- Non-persistent tapes are single-use (one `gradient()` call).
- For better performance, wrap tape operations in `@tf.function`.
- Only supports real/complex dtypes.
