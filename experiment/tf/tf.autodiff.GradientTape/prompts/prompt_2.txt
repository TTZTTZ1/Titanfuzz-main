`tf.GradientTape` is a TensorFlow context manager for automatic differentiation, enabling gradient computation by recording operations on watched tensors.

**Key Features:**
- **Automatic Tracking:** By default, it watches all trainable variables (`tf.Variable` with `trainable=True`).
- **Manual Control:** Use `watch()` to track specific tensors or disable automatic tracking via `watch_accessed_variables=False`.
- **Persistent Mode:** Set `persistent=True` to compute multiple gradients from the same tape (released when garbage collected).
- **Higher-Order Gradients:** Supports nested tapes for higher-order derivatives.

**Core Methods:**
- `gradient(target, sources)`: Computes gradients of `target` with respect to `sources`.
- `jacobian(target, sources)`: Computes the Jacobian matrix.
- `batch_jacobian(target, source)`: Efficiently computes per-example Jacobians under independence assumptions.
- `reset()`: Clears all recorded operations.
- `stop_recording()`: Temporarily pauses recording to reduce memory usage.

**Usage Example:**
```python
x = tf.constant(3.0)
with tf.GradientTape() as tape:
    tape.watch(x)
    y = x * x
dy_dx = tape.gradient(y, x)  # Returns 6.0
```

**Notes:**
- Non-persistent tapes can only compute gradients once.
- For better performance with `jacobian`/`batch_jacobian`, wrap the tape code in `@tf.function`.
- Only tensors with real or complex dtypes are differentiable.
---GENERAL SUMMARY END---
