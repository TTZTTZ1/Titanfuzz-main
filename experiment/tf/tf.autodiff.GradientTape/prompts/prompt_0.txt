`tf.GradientTape` is a TensorFlow context manager for automatic differentiation, enabling gradient computation of operations recorded within its scope.

**Key Features:**
- **Automatic Tracking:** By default, it watches all trainable variables (`tf.Variable` with `trainable=True`). Manual tracking via `watch()` is also supported.
- **Gradient Computation:** Use `gradient(target, sources)` to compute gradients of a target with respect to specified sources.
- **Higher-Order Derivatives:** Nested tapes allow computation of higher-order derivatives.
- **Persistent Mode:** Set `persistent=True` to compute multiple gradients from the same tape (released upon garbage collection).
- **Jacobian Support:** Methods `jacobian()` and `batch_jacobian()` compute Jacobian matrices efficiently.
- **Control Over Tracking:** Disable automatic variable tracking with `watch_accessed_variables=False` for fine-grained control.

**Common Use Cases:**
- Custom training loops
- Gradient-based optimization
- Computing higher-order derivatives (e.g., for Hessians)
- Gradient analysis in neural networks

**Important Notes:**
- By default, a non-persistent tape can only compute gradients once.
- Use `@tf.function` for better performance with Jacobian computations.
- Only tensors with real or complex dtypes are differentiable.
