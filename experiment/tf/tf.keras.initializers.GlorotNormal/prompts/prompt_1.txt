The `tf.keras.initializers.GlorotNormal` (also known as Xavier normal initializer) is a weight initializer that draws samples from a truncated normal distribution centered at 0. It is designed to maintain stable variance across layers in neural networks, particularly for activation functions like sigmoid or tanh.

**Key Features:**
- **Distribution:** Uses a truncated normal distribution with standard deviation `sqrt(2 / (fan_in + fan_out))`, where `fan_in` is the number of input units and `fan_out` is the number of output units in the weight tensor.
- **Purpose:** Helps mitigate vanishing or exploding gradients during training by scaling initial weights appropriately.
- **Usage:** Can be used standalone to generate initial values or as a `kernel_initializer` in Keras layers (e.g., `Dense`, `Conv2D`).
- **Seed Control:** Accepts a `seed` argument for deterministic initialization.

**Example:**
```python
initializer = tf.keras.initializers.GlorotNormal()
values = initializer(shape=(2, 2))  # Standalone usage
layer = tf.keras.layers.Dense(3, kernel_initializer=initializer)  # Layer usage
```
