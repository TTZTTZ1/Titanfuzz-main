`tf.keras.layers.SimpleRNNCell` is a recurrent neural network cell that processes a single timestep within a sequence. It is designed to be used with `keras.layers.RNN` to handle sequential data step by step.

**Key Features:**
- **Single-step processing**: Unlike `SimpleRNN` (which processes entire sequences), this cell operates on one timestep at a time.
- **Core computation**: It combines the current input and previous hidden state to produce a new hidden state using:
  - A linear transformation of inputs (via `kernel` weights)
  - A linear transformation of the recurrent state (via `recurrent_kernel` weights)
  - An optional bias and activation function (default: `tanh`)

**Main Parameters:**
- `units`: Dimensionality of the output/hidden state.
- `activation`: Activation function applied to the output (default: `tanh`).
- `dropout` / `recurrent_dropout`: Optional dropout rates for input and recurrent connections.
- Initializers, regularizers, and constraints for weights and bias.

**Call Arguments:**
- `sequence`: Input for the current timestep (shape: `[batch, features]`).
- `states`: Previous hidden state (shape: `[batch, units]`).
- `training`: Boolean to toggle dropout during training/inference.

**Example Usage:**
```python
cell = keras.layers.SimpleRNNCell(units=4)
rnn_layer = keras.layers.RNN(cell)
output = rnn_layer(input_sequence)  # Processes full sequence
```

**Use Case:**  
Ideal for custom RNN implementations where fine-grained control over each timestep is needed, or when building stacked/conditional RNN architectures.
