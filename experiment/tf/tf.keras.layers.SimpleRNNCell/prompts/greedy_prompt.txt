**Key Features:**
- **Purpose**: Implements the basic RNN operation: `output = activation(dot(input, kernel) + dot(state, recurrent_kernel) + bias)`.
- **Input/Output**: Takes a 2D tensor `(batch, features)` per timestep and a previous state `(batch, units)`, returning a new state of the same shape.
- **Parameters**: Includes standard RNN options like activation functions, bias usage, weight initializers/regularizers/constraints, and dropout for inputs/recurrent connections.
- **Usage**: Typically wrapped in `keras.layers.RNN` to process full sequences, supporting both single-output and sequence-output modes.

**Example:**
```python

```

This cell is foundational for building custom RNN architectures with fine-grained control over each timestep's computation.
