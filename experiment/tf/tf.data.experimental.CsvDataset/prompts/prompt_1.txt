**Key Features:**
- Reads one or more CSV files.
- Parses each row into a tuple of tensors based on specified column types and defaults.
- Supports optional compression, custom delimiters, quoting, missing value handling, and column selection/exclusion.
- Follows RFC 4180 CSV format standards.

**Basic Usage:**
```python

    filenames=['data.csv'],
    record_defaults=[tf.float32, tf.int32, tf.string],
    header=True,
    select_cols=[0, 1, 2]  # Select specific columns

```

**Parameters Overview:**
- `filenames`: Path(s) to CSV file(s).

- `header`: Whether to skip the first line as a header.
- `select_cols`/`exclude_cols`: Choose specific columns to include or exclude.
- `compression_type`: Supports "GZIP" or "ZLIB".
- `field_delim`: Custom field delimiter (default: comma).
- `na_value`: String to treat as missing/NaN.

**Example Output:**
For a CSV with rows like `"1.5,10,apple"`, the dataset yields elements like `(1.5, 10, b'apple')`.

**Note:** For higher-level features (automatic type inference, batching, shuffling), consider `tf.data.experimental.make_csv_dataset`.
