`tf.keras.optimizers.Adam` is an adaptive stochastic gradient descent optimizer that implements the Adam algorithm. It adjusts learning rates per parameter using estimates of first and second moments of gradients, making it efficient and well-suited for large-scale problems.

**Key Parameters:**
- `learning_rate`: Step size (default 0.001)
- `beta_1`, `beta_2`: Exponential decay rates for moment estimates (defaults 0.9, 0.999)
- `epsilon`: Numerical stability constant (default 1e-7)
- `amsgrad`: Whether to use AMSGrad variant (default False)
- `weight_decay`: Optional L2 regularization
- Gradient clipping options: `clipnorm`, `clipvalue`, `global_clipnorm`
- EMA options: `use_ema`, `ema_momentum` for weight smoothing
- `loss_scale_factor`: For mixed precision training
- `gradient_accumulation_steps`: For small batch training

**Core Functionality:**
- Maintains moving averages of gradients (momentum) and squared gradients (velocity)
- Computes adaptive learning rates for each parameter
- Supports advanced features like gradient clipping, weight decay, and exponential moving averages

**Typical Usage:**
```python
optimizer = tf.keras.optimizers.Adam(learning_rate=0.001)
model.compile(optimizer=optimizer, ...)
```

The optimizer automatically handles variable updates during training and includes methods for saving/loading state, configuration serialization, and various optimization enhancements.
