`tf.keras.optimizers.Adam` is a widely used stochastic gradient descent optimizer that adapts learning rates based on estimates of first-order (momentum) and second-order moments of gradients. It is efficient, memory-friendly, and suitable for large-scale problems.

**Key Parameters:**
- `learning_rate`: Step size for updates (default 0.001).
- `beta_1`, `beta_2`: Exponential decay rates for moment estimates (defaults 0.9, 0.999).
- `epsilon`: Small constant for numerical stability (default 1e-7).
- `amsgrad`: Whether to use the AMSGrad variant (default False).
- `weight_decay`: Optional L2 regularization on weights.
- Gradient clipping options: `clipnorm`, `clipvalue`, `global_clipnorm`.
- EMA (Exponential Moving Average) support: `use_ema`, `ema_momentum`, `ema_overwrite_frequency`.
- Mixed precision training: `loss_scale_factor`.
- `gradient_accumulation_steps`: For accumulating gradients over multiple steps.

**Main Methods:**
- `apply(grads)`: Updates model variables using computed gradients.
- `build(var_list)`: Initializes optimizer variables (momentum, velocity).
- `update_step(gradient, variable, learning_rate)`: Core update logic.
- `finalize_variable_values(var_list)`: Applies EMA or final updates if enabled.
- `get_config()` / `from_config()`: Serialization and deserialization.

**Typical Usage:**
```python
optimizer = tf.keras.optimizers.Adam(learning_rate=0.001, beta_1=0.9, beta_2=0.999)
optimizer.apply_gradients(zip(grads, model.trainable_variables))
```

Adam combines momentum (accelerated convergence) and adaptive learning rates (per-parameter scaling), making it robust and widely applicable in deep learning.
