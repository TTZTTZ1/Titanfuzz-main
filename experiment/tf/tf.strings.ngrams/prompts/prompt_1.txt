---GENERAL SUMMARY START---
`tf.strings.ngrams` generates n-grams from string sequences. It joins adjacent strings within a sequence to form n-grams of specified widths, with options for custom separators, padding, and handling of short sequences.

**Key parameters:**
- `data`: Input tensor or ragged tensor of strings.
- `ngram_width`: Integer or list of integers defining n-gram sizes.
- `separator`: String used to join n-gram elements (default: space).
- `pad_values`: Optional padding for sequence start/end.
- `padding_width`: Number of padding elements added to each side.
- `preserve_short_sequences`: Ensures at least one n-gram for short sequences.

**Returns:** A ragged tensor of n-grams.

**Example:**
```python
tf.strings.ngrams(["A", "B", "C", "D"], 2) 
# Output: [b'A B', b'B C', b'C D']
```
