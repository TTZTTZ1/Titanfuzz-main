`tf.data.experimental.load` loads a previously saved TensorFlow dataset from disk. It is deprecated; use `tf.data.Dataset.load()` instead.

**Key parameters:**
- `path`: Directory where the dataset was saved.
- `element_spec`: Optional structure of `tf.TypeSpec` matching the dataset elements; required in graph mode.
- `compression`: Optional decompression algorithm (`GZIP` or `NONE`).
- `reader_func`: Optional function to customize how sharded data is read and ordered.

**Functionality:**
- Restores a dataset saved with `tf.data.experimental.save`.
- Preserves element order if the dataset was sharded.
- Allows custom loading logic (e.g., shuffling shards) via `reader_func`.

**Example:**
```python
dataset = tf.data.experimental.load(path)
for elem in dataset:
    print(elem)
```

**Errors raised:**
- `FileNotFoundError` if `element_spec` is missing and saved type specs cannot be found.
- `ValueError` if `element_spec` is missing in graph mode.
