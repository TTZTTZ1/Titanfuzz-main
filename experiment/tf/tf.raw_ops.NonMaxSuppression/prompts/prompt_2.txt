**Key Points:**  
- **Inputs:**  
  - `boxes`: 2D float tensor `[num_boxes, 4]` in `[y1, x1, y2, x2]` format (normalized or absolute coordinates).  
  - `scores`: 1D float tensor `[num_boxes]` for each box’s confidence.  
  - `max_output_size`: Scalar integer for the maximum number of selected boxes.  
  - `iou_threshold` (optional, default=0.5): IOU threshold to suppress overlapping boxes.  

- **Functionality:**  
  - Selects boxes in descending score order, removing those with high IOU overlap with higher-scoring selections.  
  - Algorithm is invariant to coordinate system translations/reflections.  

- **Output:**  
  - 1D integer tensor of selected box indices (size ≤ `max_output_size`). Use `tf.gather` to retrieve the corresponding box coordinates.  

**Example Usage:**  
```python
selected_indices = tf.raw_ops.NonMaxSuppression(boxes, scores, max_output_size, iou_threshold)
selected_boxes = tf.gather(boxes, selected_indices)
```
