`tf.keras.models.clone_model` creates a copy of a Keras Functional or Sequential model with newly instantiated layers and weights, rather than sharing existing ones. It is useful for replicating a model's architecture while resetting its parameters.

**Key Parameters:**
- `model`: The original Functional or Sequential model to clone.
- `input_tensors`: Optional new inputs; if not provided, new `Input` layers are created.
- `clone_function`: Customizes how each layer is cloned (default uses layer configs).
- `call_function`: Customizes layer calls (Functional models only).
- `recursive`: If `True`, recursively clones inner models.

**Important Notes:**
- Shared objects in the original model become separate in the clone.
- Subclassed models are not directly supported; use `model.__class__.from_config(model.get_config())` instead.
- Custom `clone_function` or `call_function` can modify layer behavior (e.g., adding dropout, setting random seeds).

**Example:**
```python
new_model = clone_model(model)  # Basic clone with new weights
```
