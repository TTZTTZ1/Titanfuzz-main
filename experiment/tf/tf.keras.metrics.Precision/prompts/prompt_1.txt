`tf.keras.metrics.Precision` computes the precision metric for classification tasks, measuring the proportion of true positive predictions among all positive predictions. It supports customizable thresholds, top-k predictions, and per-class evaluation.

**Key Features:**
- **Calculation**: Precision = True Positives / (True Positives + False Positives)
- **Parameters**:
  - `thresholds`: Decision threshold(s) for binarizing predictions (default=0.5)
  - `top_k`: Consider only top-k predicted classes
  - `class_id`: Evaluate precision for a specific class only
- **Usage**: Works with both binary and multi-class classification, supports sample weighting, and integrates seamlessly with `model.compile()`.

**Example:**
```python
m = tf.keras.metrics.Precision()
m.update_state([0, 1, 1, 1], [1, 0, 1, 1])  # y_true, y_pred
m.result().numpy()  # Returns 0.6666667
```

**Common Use Cases:**
- Imbalanced data classification
- Model evaluation during training
- Federated learning scenarios with sparse aggregation
