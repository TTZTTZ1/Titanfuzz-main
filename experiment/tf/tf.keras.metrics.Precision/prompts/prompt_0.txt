`tf.keras.metrics.Precision` is a metric that calculates the precision of predictions relative to true labels. It tracks true positives and false positives to compute precision as:

\[
\text{precision} = \frac{\text{true\_positives}}{\text{true\_positives} + \text{false\_positives}}
\]

**Key Parameters:**
- `thresholds`: (Optional) Threshold(s) in [0, 1] to binarize predictions. Default is 0.5 if not set.
- `top_k`: (Optional) If set, considers only the top-k predicted classes for each sample.
- `class_id`: (Optional) If specified, computes precision for a specific class only.
- `name` / `dtype`: (Optional) Instance name and data type.

**Usage Example:**
```python

```

```python

```

**Key Methods:**
- `update_state()`: Accumulates TP/FP statistics.
- `result()`: Computes and returns current precision.
- `reset_state()`: Resets all metric variables.

**Notes:**
- Supports sample weighting.
- For models with `from_logits=True`, set `thresholds=0`.
