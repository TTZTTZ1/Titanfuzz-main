`tf.keras.layers.SimpleRNN` is a fully-connected recurrent neural network layer where the output is fed back as input for the next timestep. It is commonly used for processing sequential data.

**Key Features:**
- **Core Functionality:** Processes sequences by maintaining a hidden state updated at each timestep via `new_state = activation(input * kernel + state * recurrent_kernel + bias)`.
- **Input/Output Shapes:** Expects a 3D input `(batch, timesteps, features)`. Output shape depends on `return_sequences`:
  - `False` (default): Returns last output only `(batch, units)`.
  - `True`: Returns full sequence `(batch, timesteps, units)`.
- **Key Parameters:**
  - `units`: Dimensionality of the output space.
  - `activation`: Activation function (default: `tanh`).
  - `dropout` / `recurrent_dropout`: Fraction of units to drop for regularization.
  - `return_sequences`, `return_state`: Control output format.
  - `stateful`: If `True`, maintains state across batches.
- **Use Case:** Suitable for basic sequence modeling tasks (e.g., time-series prediction, text processing) but may struggle with long-term dependencies due to vanishing gradients.

**Example:**
```python
# Return last output only
simple_rnn = SimpleRNN(units=4)
output = simple_rnn(inputs)  # Shape: (batch, 4)

# Return full sequence and final state
simple_rnn = SimpleRNN(units=4, return_sequences=True, return_state=True)
whole_sequence, final_state = simple_rnn(inputs)  # Shapes: (batch, timesteps, 4), (batch, 4)
```
