**tf.keras.layers.SimpleRNN** is a basic recurrent neural network layer where the output from the previous time step is fed back as input to the current step. It is a fully-connected RNN suitable for processing sequential data.

### **Key Features:**
- **Input/Output Shape:** Expects a 3D input `(batch_size, timesteps, features)`. Output shape depends on `return_sequences`:
  - `False` (default): Returns last output only → `(batch_size, units)`.
  - `True`: Returns full sequence → `(batch_size, timesteps, units)`.
- **Core Parameters:**
  - `units`: Dimensionality of the output space.
  - `activation`: Activation function (default: `tanh`).
  - `dropout` & `recurrent_dropout`: Regularization via dropout on input/recurrent connections.
  - `return_state`: If `True`, returns both output and final hidden state.
- **Use Cases:** Suitable for simple sequence modeling tasks (e.g., time series prediction, text processing) but limited by vanishing gradient problems for long sequences.

### **Example Usage:**
```python
# Return last output only
simple_rnn = SimpleRNN(units=4)
output = simple_rnn(inputs)  # Shape: (batch, 4)

# Return full sequence and final state
simple_rnn = SimpleRNN(units=4, return_sequences=True, return_state=True)
whole_sequence, final_state = simple_rnn(inputs)  # Shapes: (batch, timesteps, 4), (batch, 4)
```

### **Notes:**
- Inherits from `RNN` base class, supporting features like masking, statefulness, and unrolling.
- For longer sequences, consider **LSTM** or **GRU** layers to mitigate vanishing gradients.
- Dropout is applied only during training (`training=True`).
