---GENERAL SUMMARY START---
`tf.keras.layers.SimpleRNN` is a fully-connected recurrent neural network (RNN) layer where the output from the previous timestep is fed back as input for the next timestep. It is suitable for processing sequential data.

**Key Features:**
- **Input:** 3D tensor of shape `[batch, timesteps, feature]`.
- **Output:** By default, returns the final output (shape `[batch, units]`). If `return_sequences=True`, returns all timestep outputs (shape `[batch, timesteps, units]`).
- **Core Parameters:**
  - `units`: Dimensionality of the output space.
  - `activation`: Activation function (default `tanh`).
  - `dropout` / `recurrent_dropout`: Dropout rates for input and recurrent connections.
  - `return_sequences`: Whether to return the full output sequence or just the last output.
  - `return_state`: Whether to return the final state in addition to the output.
  - `stateful`: If `True`, the final state from one batch is used as the initial state for the next.

**Common Use Case:**  
Processing sequential data like time series or text, where each output depends on both the current input and previous computations.

**Example:**
```python
simple_rnn = keras.layers.SimpleRNN(4, return_sequences=True)
output = simple_rnn(inputs)  # Returns sequence of outputs for each timestep
```
