`tf.keras.layers.AvgPool2D` (also called `AveragePooling2D`) performs 2D average pooling to downsample spatial dimensions (height and width) by computing the average value within a sliding window for each input channel.

**Key Parameters:**
- **`pool_size`**: Size of the pooling window (e.g., `(2, 2)`).
- **`strides`**: Step size for sliding the window; defaults to `pool_size` if not set.
- **`padding`**: `"valid"` (no padding) or `"same"` (padding to preserve input dimensions).
- **`data_format`**: `"channels_last"` (default, shape: `(batch, height, width, channels)`) or `"channels_first"` (shape: `(batch, channels, height, width)`).

**Functionality:**
- Reduces spatial dimensions while retaining channel depth.
- Output size depends on `padding`:
  - `"valid"`: `output_shape = floor((input_shape - pool_size) / strides) + 1`
  - `"same"`: `output_shape = floor((input_shape - 1) / strides) + 1`

**Use Case:** Commonly used in convolutional neural networks (CNNs) to progressively reduce feature map size, decrease computation, and emphasize dominant features.
