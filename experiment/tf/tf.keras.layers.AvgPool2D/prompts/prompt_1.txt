- **`pool_size`**: Size of the pooling window (e.g., `(2, 2)`).
- **`strides`**: Step size for sliding the window; defaults to `pool_size` if not set.
- **`padding`**: `"valid"` (no padding) or `"same"` (padding to preserve input dimensions).
- **`data_format`**: `"channels_last"` (default, shape: `(batch, height, width, channels)`) or `"channels_first"`.

**Input/Output Shapes:**
- **Input**: 4D tensor with shape depending on `data_format`.
- **Output**: 4D tensor with downsampled spatial dimensions (height and width reduced based on `pool_size`, `strides`, and `padding`).

**Example:**
```python
avg_pool = tf.keras.layers.AveragePooling2D(pool_size=(2, 2), strides=(1, 1), padding="valid")
```
This layer reduces feature map size while retaining channel depth, commonly used in CNNs to decrease computational load and control overfitting.
