`tf.scan` is a higher-order TensorFlow function that sequentially applies a callable `fn` to elements of `elems` along their first dimension, accumulating results. It is similar to a fold/reduce operation but returns the intermediate accumulated values at each step rather than just the final result.

**Key Features:**
- **Inputs:** `fn` (callable), `elems` (tensor or nested sequence), optional `initializer` (initial accumulator value).
- **Process:** `fn` receives the current accumulator and the current slice of `elems`, returning the updated accumulator. The sequence of all accumulators is returned.
- **Direction:** By default, scanning proceeds first to last; set `reverse=True` to scan last to first.
- **Multi-arity Support:** Both `elems` and `initializer` can be nested structures (lists/tuples of tensors), with `fn`'s arguments and outputs matching those structures.
- **Deprecated:** `back_prop=False` is deprecated; use `tf.stop_gradient` on the output instead.

**Common Use Cases:**
- Cumulative operations (e.g., cumulative sum).
- Sequential models where each step depends on previous outputs (e.g., Fibonacci sequence generation).
- Custom iterative computations over tensor sequences.

**Example:**
```python
elems = tf.constant([1, 2, 3, 4, 5])
cumulative_sum = tf.scan(lambda a, x: a + x, elems)
# Result: [1, 3, 6, 10, 15]
```
