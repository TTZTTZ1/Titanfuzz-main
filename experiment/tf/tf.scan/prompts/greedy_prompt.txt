`tf.scan` is a TensorFlow function that sequentially applies a callable `fn` to elements of a tensor (or nested tensors) along their first dimension, accumulating results. It is similar to a fold/reduce operation but returns all intermediate accumulator values rather than just the final result.

**Key features:**
- **Inputs:** `fn` (callable), `elems` (tensor or nested tensors), optional `initializer` (starting accumulator value).
- **Process:** `fn` receives the current accumulator and the current slice of `elems`, returning an updated accumulator.
- **Output:** A tensor (or nested tensors) containing the accumulated results at each step.
- **Options:** Supports reverse scanning, GPU-CPU memory swapping, and control over parallel iterations.
- **Deprecated:** `back_prop=False` is deprecated; use `tf.stop_gradient` on the output instead.

**Example:**
```python
import tensorflow as tf
elems = tf.constant([1, 2, 3, 4, 5])
sums = tf.scan(lambda a, x: a + x, elems)  # Result: [1, 3, 6, 10, 15]
```

It is useful for iterative computations where intermediate states are needed, such as cumulative sums or sequences like Fibonacci numbers.
