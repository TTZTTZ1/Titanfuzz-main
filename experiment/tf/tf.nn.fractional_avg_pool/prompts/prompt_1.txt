`tf.nn.fractional_avg_pool` performs fractional average pooling on a 4D input tensor. Unlike standard pooling with fixed window sizes, it uses random or pseudorandom sequences to determine variable-sized pooling regions, reducing the spatial dimensions by a specified pooling ratio.

**Key Parameters:**
- `value`: 4D input tensor of shape `[batch, height, width, channels]`.
- `pooling_ratio`: List of floats (≥4 elements) specifying the pooling factor for each dimension. Only height and width ratios (indices 1 and 2) can be >1.0; batch and channel ratios must be 1.0 (e.g., `[1.0, 1.44, 1.73, 1.0]`).
- `pseudo_random`: Controls whether the pooling sequence is pseudorandom or fully random.
- `overlapping`: If `True`, boundary values can be shared between adjacent pooling cells.

**Returns:**  
A tuple containing:
1. `output`: The pooled tensor (same type as input).
2. `row_pooling_sequence` and `col_pooling_sequence`: Integer tensors describing the pooling regions.

**Note:** This operation is the average-pooling variant of fractional max pooling, with pooling region generation following the same randomized approach described in the [Fractional Max-Pooling paper](https://arxiv.org/abs/1412.6071).
