`tf.nn.fractional_avg_pool` performs fractional average pooling on a 4D input tensor. It reduces spatial dimensions by averaging values within pooling regions determined by a pseudo-random sequence.

**Key Parameters:**
- `value`: 4D input tensor of shape `[batch, height, width, channels]`.
- `pooling_ratio`: List of floats (≥4 elements) specifying pooling ratios for each dimension. Only height and width ratios (indices 1 and 2) can be >1.0; batch and channel ratios must be 1.0.
- `pseudo_random`: Controls whether pooling sequence is pseudorandom or random.
- `overlapping`: If `True`, boundary values are shared between adjacent pooling cells.

**Returns:**  
A tuple containing:
1. Pooled output tensor (same type as input).
2. Row pooling sequence (`int64` tensor).
3. Column pooling sequence (`int64` tensor).

**Note:** This operation is analogous to fractional max pooling but uses averaging instead of maximization within each pooling region.
