\text{RMSE} = \sqrt{\text{mean}((y\_\text{pred} - y\_\text{true})^2)}
\]

- Inherits from `Mean` and `Metric` base classes.


**Main Methods:**
- `update_state()`: Accumulates error statistics.
- `result()`: Computes and returns the current RMSE.
- `reset_state()`: Clears the metric's state (e.g., between epochs).

RMSE is commonly used in **regression tasks** to measure prediction accuracy, with lower values indicating better model performance.
