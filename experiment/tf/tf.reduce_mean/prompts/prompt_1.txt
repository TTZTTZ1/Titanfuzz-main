`tf.reduce_mean` computes the mean of elements across specified dimensions of a tensor.  

**Key features:**  
- **Reduction axis:** Use `axis` to specify dimensions to average over; if `axis=None`, reduces all dimensions.  
- **Shape handling:** By default, reduced dimensions are removed unless `keepdims=True` (keeps them as size 1).  
- **Data type:** Output type is inferred from the input tensor (unlike NumPy’s `np.mean`, which defaults to `float64`).  

**Example:**  
```python

tf.reduce_mean(x)               # 1.5
tf.reduce_mean(x, axis=0)       # [1.5, 1.5]
tf.reduce_mean(x, axis=1)       # [1., 2.]
```

**Use cases:**  
Commonly used in loss computation, feature aggregation, and dimensionality reduction in machine learning models.
