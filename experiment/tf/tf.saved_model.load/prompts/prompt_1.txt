**Key features:**
- **Signatures:** Loaded models expose a `.signatures` dictionary mapping signature names (e.g., `"serving_default"`) to callable functions.
- **Trackable objects:** Attributes such as variables, functions, or other objects saved with `tf.saved_model.save` are accessible on the returned object.
- **Keras models:** While loadable, the returned object is not a full Keras model (use `tf.keras.models.load_model` for full Keras functionality).
- **TensorFlow 1.x support:** For 1.x SavedModels, it provides `.signatures`, `.variables`, `.graph`, and a `.prune()` method for subgraph extraction.
- **Async loading:** To safely load asynchronously saved models, check for `saved_model.pb` file completion before loading.

**Arguments:**
- `export_dir`: Path to the SavedModel directory.
- `tags` (optional): MetaGraph tags to load (required if multiple MetaGraphs exist).
- `options` (optional): `tf.saved_model.LoadOptions` for advanced loading configurations.

**Returns:** A trackable object with loaded signatures and saved attributes.
