`tf.Module` is TensorFlow's base class for creating reusable, stateful components (like neural network layers or models). It provides automatic tracking of variables and submodules, hierarchical naming for TensorBoard, and a structured way to organize trainable parameters.

**Key features:**
- **Variable/Module tracking:** Automatically collects `tf.Variable` and nested `tf.Module` instances via `.variables`, `.trainable_variables`, and `.submodules` properties.
- **Name scoping:** Each module has a `name_scope` for organized operation/variable naming in TensorBoard graphs.
- **Decorator support:** Use `@tf.Module.with_name_scope` to automatically wrap methods in the module's name scope.

**Example usage:**
```python
class Dense(tf.Module):
    def __init__(self, in_dim, out_dim):
        super().__init__()
        self.w = tf.Variable(tf.random.normal([in_dim, out_dim]))
        self.b = tf.Variable(tf.zeros([out_dim]))
    
    def __call__(self, x):
        return tf.matmul(x, self.w) + self.b
```

**Benefits:**  
- Simplifies parameter management and reuse  
- Enables clean model architecture through nesting  
- Facilitates debugging with structured naming
