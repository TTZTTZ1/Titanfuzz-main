- **Purpose**: Converts serialized `tf.Example` features into ragged tensors, handling nested variable-length data.
- **Core Parameters**:

  - `partitions`: A list defining how to split values into rows/dimensions (e.g., `RowSplits`, `RowLengths`, `UniformRowLength`).
  - `dtype`: Data type of the values (e.g., `int64`, `float32`, `string`).
- **Output Structure**:
  - Without partitions: Results in a 1D tensor (single example) or 2D `RaggedTensor` (batch).
  - With partitions: Creates higher-dimensional ragged tensors, where each partition adds a ragged dimension.
  - If the innermost partition is `UniformRowLength`, the output becomes a dense tensor.
- **Use Case**: Ideal for parsing sparse or irregular data like sequences of varying lengths, nested lists, or segmented features.

### Example:
```python

    value_key="values",
    dtype=tf.int64,
```
