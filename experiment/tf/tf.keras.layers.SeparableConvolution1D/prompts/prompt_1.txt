**Key Parameters:**
- `filters`: Number of output filters.
- `kernel_size`: Size of the depthwise convolution window.
- `strides`: Stride length (default 1).
- `padding`: `"valid"` (no padding) or `"same"` (padding to preserve input length when `strides=1`).
- `data_format`: `"channels_last"` (default, shape `[batch, steps, channels]`) or `"channels_first"` (shape `[batch, channels, steps]`).
- `depth_multiplier`: Number of depthwise output channels per input channel (default 1).
- `activation`: Activation function applied to the output (optional).

**Input/Output Shape:**
- Input: 3D tensor `(batch, steps, channels)` (channels_last) or `(batch, channels, steps)` (channels_first).
- Output: 3D tensor with `steps` dimension adjusted by convolution and `channels` replaced by `filters`.

**Example:**
```python
layer = SeparableConv1D(filters=3, kernel_size=4, strides=2, activation='relu')
output = layer(input_tensor)  # Reduces steps, outputs 3 channels
```

This layer is useful for building lightweight models, especially in scenarios like time-series analysis or 1D signal processing.
