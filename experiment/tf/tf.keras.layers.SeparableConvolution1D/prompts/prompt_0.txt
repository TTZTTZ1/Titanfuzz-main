- `depth_multiplier`: Number of depthwise output channels per input channel.
- `activation`: Optional activation function applied to the output.

3D tensor with shape adjusted by convolution parameters.

**Example:**

This layer is efficient for processing 1D sequences (e.g., time-series or text data) while maintaining representational capacity.
