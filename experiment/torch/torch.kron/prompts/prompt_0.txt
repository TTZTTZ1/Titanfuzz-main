`torch.kron` computes the Kronecker product (⊗) of two tensors. It expands each element of the first tensor by multiplying it with the entire second tensor, producing a block-structured output.

**Key details:**
- Inputs can have any number of dimensions; tensors with fewer dimensions are automatically unsqueezed to match.
- Output dimensions are the elementwise product of the input dimensions: `(a₀*b₀ × a₁*b₁ × … × aₙ*bₙ)`.
- Works with both real and complex-valued tensors.
- For matrices, it creates a block matrix where each block is a scaled copy of the second matrix.

**Example:**
```python
>>> torch.kron(torch.eye(2), torch.ones(2,2))
tensor([[1., 1., 0., 0.],
        [1., 1., 0., 0.],
        [0., 0., 1., 1.],
        [0., 0., 1., 1.]])
```
