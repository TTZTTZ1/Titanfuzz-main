The `GRUCell` is a single gated recurrent unit (GRU) cell that processes one time step at a time. It takes an input tensor and a hidden state, applies GRU operations (reset gate, update gate, and new gate), and outputs the updated hidden state for the next step.

**Key details:**
- **Parameters:** `input_size` (input feature dimension), `hidden_size` (hidden state dimension), `bias` (optional bias usage).
- **Inputs:** `input` (current input) and `hidden` (previous hidden state, defaults to zero).
- **Output:** `h'` (next hidden state).
- **Shape:** Input and hidden can be batched `(N, size)` or single `(size)`; output matches hidden shape.
- **Weights:** Includes `weight_ih`, `weight_hh`, and optional biases, initialized uniformly.
- **Use case:** Suitable for custom sequential processing where manual iteration over time steps is needed.

**Example usage:**
```python
rnn = nn.GRUCell(10, 20)
hx = torch.randn(3, 20)
for i in range(6):
    hx = rnn(input[i], hx)
```
