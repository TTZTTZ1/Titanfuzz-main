`torch.nn.Sequential` is a container that chains multiple neural network modules in sequence. It allows you to treat the entire chain as a single module, where the output of each layer is automatically passed as input to the next.

**Key Features:**
- **Ordered Execution:** Modules are executed in the order they are provided.
- **Flexible Construction:** Can be initialized with modules directly or via an `OrderedDict` for named layers.
- **Container Methods:** Supports `append()`, `extend()`, `insert()`, and `pop()` to modify the sequence dynamically.
- **Simplified Forward Pass:** Automatically forwards input through all layers without manual intervention.

**Example:**
```python
model = nn.Sequential(
    nn.Linear(10, 20),
    nn.ReLU(),
    nn.Linear(20, 5)
)
```
This is ideal for straightforward, linear architectures where data flows sequentially through layers.
