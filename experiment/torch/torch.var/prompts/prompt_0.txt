`torch.var` computes the variance of a tensor along specified dimensions.  

**Key parameters:**  
- `input`: Input tensor.  
- `dim`: Dimension(s) to reduce; if `None`, variance is computed over all elements.  
- `correction`: Adjusts the denominator as `N − correction` (default is 1, corresponding to Bessel’s correction for sample variance).  
- `keepdim`: If `True`, retains reduced dimensions as size 1.  

**Formula:**  
\[
\sigma^2 = \frac{1}{\max(0, N - \text{correction})} \sum_{i=0}^{N-1} (x_i - \bar{x})^2
\]  
where \( N \) is the number of samples and \( \bar{x} \) is the sample mean.  

**Example:**  
```python
>>> torch.var(a, dim=1, keepdim=True)
```  
Computes variance along each row, keeping the reduced dimension.
