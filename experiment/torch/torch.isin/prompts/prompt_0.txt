**Key parameters:**

- `test_elements`: Tensor or scalar containing values to test against.
- `assume_unique`: If True, assumes both inputs have unique elements for faster computation.
- `invert`: If True, returns True for elements *not* in `test_elements`.

**Note:** Only one of `elements` or `test_elements` can be a scalar, not both.

**Example:**
```python
>>> torch.isin(torch.tensor([[1, 2], [3, 4]]), torch.tensor([2, 3]))
tensor([[False,  True],
        [ True, False]])
```
