## Core Concepts
- **Parameterizable Distributions**: Includes common distributions (Normal, Bernoulli, Categorical, etc.) with parameter validation via constraints.
- **Gradient Estimation**: Supports two main methods for backpropagation through random samples:
  - **Score Function (REINFORCE)**: Uses `sample()` and `log_prob()` for policy gradients
  - **Pathwise Derivative (Reparameterization)**: Uses `rsample()` for differentiable sampling via reparameterization trick

## Main Components

### 1. **Distribution Base Class**
Abstract base class with essential methods:
- `sample()` / `rsample()`: Generate samples (with/without reparameterization)
- `log_prob()`: Compute log probability density/mass
- `entropy()`: Calculate distribution entropy
- `mean`, `variance`, `mode`: Distribution statistics
- `support`: Defines valid domain of the distribution

### 2. **Key Distributions**
- **Univariate**: Bernoulli, Beta, Binomial, Categorical, Cauchy, Exponential, Gamma, Geometric, Laplace, Normal, Poisson, Uniform, etc.
- **Multivariate**: Dirichlet, Multinomial, MultivariateNormal, LowRankMultivariateNormal
- **Transformed**: LogNormal, HalfNormal, Gumbel, Pareto, Weibull (via `TransformedDistribution`)
- **Specialized**: LKJCholesky (correlation matrices), Wishart, VonMises (circular), MixtureSameFamily

### 3. **Transforms & Constraints**
- **Transforms**: Bijective transformations (Affine, Exp, Sigmoid, Softmax, StickBreaking, etc.) for reparameterization
- **Constraints**: Define valid parameter ranges (positive, simplex, interval, etc.)
- **Registry System**: `transform_to()` and `biject_to()` for mapping constraints to transforms

### 4. **KL Divergence**
- `kl_divergence()`: Computes KL divergence between distributions
- Extensible via `register_kl()` decorator for custom distribution pairs

## Practical Usage Examples

### REINFORCE (Policy Gradient)
```python
probs = policy_network(state)

loss.backward()
```

### Reparameterization Trick (VAEs)

- **Independent Wrapper**: Reinterprets batch dimensions as event dimensions
- **Transformed Distributions**: Compose base distributions with transforms

The package is designed for both machine learning applications (VAEs, reinforcement learning) and general probabilistic modeling, providing efficient, differentiable operations on GPU/CPU.
