`torch.std` computes the standard deviation of a tensor along specified dimensions.  

**Key parameters:**  
- `dim`: dimension(s) to reduce; if `None`, reduces all dimensions.  
- `correction`: adjusts the denominator as \( N - \text{correction} \) (default is 1, applying Bessel’s correction for sample standard deviation).  
- `keepdim`: if `True`, retains reduced dimensions as size 1.  

**Formula:**  
\[
\sigma = \sqrt{\frac{1}{\max(0, N - \text{correction})} \sum (x_i - \bar{x})^2}
\]  
where \( N \) is the number of samples.  

**Example:**  
```python
torch.std(a, dim=1, keepdim=True)
```  
calculates the standard deviation along each row, keeping the dimension.
