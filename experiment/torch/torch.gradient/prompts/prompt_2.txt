`torch.gradient` estimates the gradient of a scalar-valued function sampled on a grid using second-order central differences for interior points and first- or second-order approximations at boundaries.

**Key parameters:**
- `input`: Tensor of function samples.
- `spacing` (optional): Modifies index-to-coordinate mapping (scalar, list of scalars, or list of 1D tensors).
- `dim` (optional): Specific dimension(s) to compute gradient over.
- `edge_order` (optional): Boundary approximation order (1 or 2).

**Functionality:**
- Computes partial derivatives for each specified dimension.
- Default `spacing=1` treats tensor indices as coordinates.
- Supports real and complex-valued functions.
- Requires at least `edge_order + 1` samples per dimension.

**Example use:**
```python
# Gradient of f(x)=x^2 at given points
values = torch.tensor([4., 1., 1., 16.])
coords = (torch.tensor([-2., -1., 1., 4.]),)
torch.gradient(values, spacing=coords)  # Returns (tensor([-3., -2., 2., 5.]),)
```
