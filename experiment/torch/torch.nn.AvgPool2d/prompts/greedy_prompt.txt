---GENERAL SUMMARY START---
`AvgPool2d` performs 2D average pooling over an input tensor. It slides a fixed-size window across the input, computing the average of values within each window to produce a downsampled output.

**Key parameters:**
- `kernel_size`: Size of the pooling window (int or tuple).
- `stride`: Step size for sliding the window (defaults to `kernel_size`).
- `padding`: Adds zero-padding to the input before pooling.
- `ceil_mode`: Uses ceiling (instead of floor) for output size calculation.
- `count_include_pad`: Includes padding in average computation when `True`.
- `divisor_override`: Optional custom divisor instead of the pool region size.

**Input/Output shape:**
- **Input:** `(N, C, H_in, W_in)` or `(C, H_in, W_in)`
- **Output:** `(N, C, H_out, W_out)` or `(C, H_out, W_out)`

**Output size formula:**
\[
H_{out} = \left\lfloor \frac{H_{in} + 2 \times \text{padding}[0] - \text{kernel\_size}[0]}{\text{stride}[0]} + 1 \right\rfloor
\]
(same for width dimension)

**Example:**
```python
m = nn.AvgPool2d(kernel_size=3, stride=2)  # Square window
output = m(input_tensor)  # Reduces spatial dimensions via averaging
```
