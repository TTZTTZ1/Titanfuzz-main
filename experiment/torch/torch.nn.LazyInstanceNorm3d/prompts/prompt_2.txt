`LazyInstanceNorm3d` is a PyTorch module for 3D instance normalization that automatically infers the number of input channels (`num_features`) from the first batch of data. It defers initialization of parameters (weight, bias, running_mean, running_var) until the first forward pass, based on `input.size(1)`.  

**Key features:**

- Supports both affine learnable parameters and tracking of running statistics (configurable).  
- Input shape: `(N, C, D, H, W)` or `(C, D, H, W)`; output shape is identical.  

**Use case:**  
Useful when the input channel dimension is unknown at model definition time, simplifying architecture prototyping.
