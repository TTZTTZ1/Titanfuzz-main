**Key Features:**
- **Inlines constants:** Parameters and attributes become fixed values in the computation graph.
- **Preserves specified attributes:** Use `preserved_attrs` to keep specific attributes mutable (e.g., version counters).
- **Requires eval mode:** The input module must be in evaluation mode.
- **Optimizes numerics:** Optional passes for further performance gains (may affect numerical precision).

**Use Cases:**
- Deploying trained models for faster inference.
- Reducing overhead by eliminating parameter management.

**Limitations:**
- Frozen modules lose the ability to use `to()` for device/dtype changes.
- Only supports ScriptModules in eval mode.
