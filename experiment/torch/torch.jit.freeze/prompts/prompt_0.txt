`torch.jit.freeze` converts a PyTorch `ScriptModule` into a frozen version by inlining submodules, parameters, and attributes as constants in the TorchScript IR graph. This optimizes the model for inference, removing mutable parameters and applying generic speed improvements.

**Key Points:**
- Only works with ScriptModules in `eval()` mode.
- Preserves the `forward` method and any attributes listed in `preserved_attrs`.
- Automatically preserves attributes modified within preserved methods.
- After freezing, run `optimize_for_inference` for further hardware-specific optimizations.
- Removes module hierarchy and parameter mutability, so device/dtype methods like `.to()` no longer function directly.

**Example Usage:**
```python
scripted = torch.jit.script(MyModule().eval())
frozen = torch.jit.freeze(scripted, preserved_attrs=["custom_attr"])
# Parameters become constants; preserved attributes remain mutable.
```
