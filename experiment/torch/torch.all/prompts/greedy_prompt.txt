- Returns a boolean tensor (except for `uint8` input, where output is also `uint8`).  
- Can operate over the entire tensor or along specified dimensions.  

**Usage:**  
1. **Without dimension:** Returns a single boolean value for the whole tensor.  
   ```python
   torch.all(input)
   ```
2. **With dimension:** Reduces along the given dimension(s), returning `True` for rows/columns where all elements are `True`.  
   ```python
   torch.all(input, dim, keepdim=False)
   ```

**Example:**  
```python
>>> a = torch.tensor([[True, True], [True, False]])
>>> torch.all(a, dim=1)

```
