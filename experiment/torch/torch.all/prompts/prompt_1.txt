`torch.all` tests whether all elements in a tensor evaluate to `True`. It has two main forms:

1. **Global check**: Returns a single boolean tensor indicating if every element in the input is `True`.
2. **Dimension-wise check**: When a dimension `dim` is specified, it reduces over that dimension, returning `True` for each slice where all elements are `True`. The `keepdim` argument controls whether the reduced dimension is retained.

**Key notes:**
- Output dtype is `bool` for most types, except `uint8` (output remains `uint8`).
- If `dim=None`, all dimensions are reduced (global check).
- Works with both boolean and numeric tensors (non-zero values are considered `True`).

**Examples:**
- `torch.all(tensor([0, 1, 2]))` → `False` (since 0 is `False`)
- `torch.all(bool_tensor, dim=1)` → Reduces over rows, returning a 1D boolean tensor.
