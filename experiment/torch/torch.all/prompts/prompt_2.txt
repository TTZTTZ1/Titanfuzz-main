`torch.all` tests whether all elements in a tensor evaluate to `True`.  

**Key features:**  
- Returns a boolean tensor (except for `uint8` input, where output is also `uint8`).  
- Can operate over the entire tensor or along specified dimensions.  

**Two main usage forms:**  

1. **Overall check** – tests all elements:  
   ```python
   torch.all(input) → bool tensor
   ```

2. **Reduction along dimension(s)** – checks each slice in `dim`:  
   ```python
   torch.all(input, dim, keepdim=False) → reduced tensor
   ```
   - If `keepdim=True`, keeps reduced dimension(s) with size 1.  

**Examples:**  
- `torch.all(tensor([0, 1, 2])) → False` (0 is `False`)  
- For a 2D boolean tensor, `torch.all(a, dim=1)` checks each row.
