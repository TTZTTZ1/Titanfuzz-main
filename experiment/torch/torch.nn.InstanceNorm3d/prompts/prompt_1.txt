---GENERAL SUMMARY START---
`InstanceNorm3d` applies instance normalization to 5D inputs (e.g., 3D images with channels). It normalizes each channel independently for each sample in a batch using the sample's own mean and variance, following the formula:

\[
y = \frac{x - \mathrm{E}[x]}{\sqrt{\mathrm{Var}[x] + \epsilon}} \cdot \gamma + \beta
\]

- **Key parameters**:  
  - `num_features`: Number of channels \(C\) in the input.  
  - `affine`: If `True`, adds learnable scale (\(\gamma\)) and shift (\(\beta\)) parameters per channel.  
  - `track_running_stats`: If `True`, tracks running mean/variance during training for evaluation (uncommon in instance norm). Default is `False`.  

- **Input shape**: \((N, C, D, H, W)\) or \((C, D, H, W)\).  
- **Output shape**: Same as input.  

- **Typical use**:  
  - Stylization, 3D vision tasks.  
  - Often used without affine parameters (`affine=False`).  

- **Example**:  
  ```python
  m = nn.InstanceNorm3d(100, affine=True)
  output = m(input)  # input shape: (20, 100, 35, 45, 10)
  ```
---GENERAL SUMMARY END---
