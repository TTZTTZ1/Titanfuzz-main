**Purpose:**  
Enables custom pruning of PyTorch model parameters using a user-defined mask, allowing on-the-fly pruning and reparameterization.

**Key Functionality:**
- **`apply(module, name, mask)`**: Attaches a forward pre-hook to a tensor within a module to apply pruning dynamically. Reparameterizes the tensor into an original version (`name+'_orig'`) and a pruning mask (`name+'_mask'`).
- **`apply_mask(module)`**: Retrieves the mask and original tensor from the module and returns the pruned tensor (original tensor multiplied by the mask).
- **`prune(t, default_mask, importance_scores)`**: Computes and returns a pruned version of input tensor `t` based on the mask. If no mask is provided, it uses the tensor itself as importance scores.
- **`remove(module)`**: Removes the pruning reparameterization hooks and buffers, leaving the parameter permanently pruned (original tensor and mask are deleted).

**Usage Flow:**
1. Create a `CustomFromMask` instance with a predefined mask.
2. Use `apply()` to attach pruning to a specific tensor in a module.
3. The mask is applied automatically during forward passes via `apply_mask()`.
4. Optionally, use `prune()` to compute pruned tensors directly.
5. Use `remove()` to clean up the pruning setup while keeping the tensor pruned.

**Note:** Pruning is irreversible after `remove()` is called.
