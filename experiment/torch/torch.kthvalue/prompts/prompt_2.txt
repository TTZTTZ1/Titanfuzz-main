`torch.kthvalue` returns the `k`-th smallest element and its index along a specified dimension. By default, it operates on the last dimension and reduces it unless `keepdim=True`. On CUDA, indices may be nondeterministic when multiple equal values exist.

**Parameters:**
- `input`: Input tensor.
- `k`: Rank of the smallest element to find (1-based).
- `dim`: Dimension to operate on (default: last dimension).
- `keepdim`: If `True`, retains the reduced dimension.

**Returns:** A namedtuple `(values, indices)`.

**Example:**
```python
>>> x = torch.tensor([[1., 2., 3.], [4., 5., 6.]])
>>> torch.kthvalue(x, 2, dim=0, keepdim=True)
values=tensor([[4., 5., 6.]]), indices=tensor([[1, 1, 1]])
```
