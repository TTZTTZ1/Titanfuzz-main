**BatchNorm2d** is a PyTorch module that applies batch normalization to 4D inputs (batch size × channels × height × width). It normalizes each channel across the batch and spatial dimensions to stabilize and accelerate training.

**Key Features:**
- Normalizes input using per-channel mean and variance computed over the mini-batch.
- Includes learnable scale (γ) and shift (β) parameters when `affine=True` (default).
- Maintains running estimates of mean/variance during training for consistent evaluation.
- Momentum updates running statistics (default: 0.1), distinct from optimizer momentum.

**Parameters:**
- `num_features`: Number of channels `C` in input.
- `eps`: Small constant for numerical stability (default: 1e-5).
- `momentum`: Weight for updating running statistics; `None` uses simple average.
- `affine`: Enables learnable γ and β (default: `True`).
- `track_running_stats`: Toggles using running vs. batch statistics in eval mode (default: `True`).

**Usage:**
```python
m = nn.BatchNorm2d(num_channels)  # With learnable parameters
output = m(input)  # Input shape: (N, C, H, W)
```
Preserves input shape; commonly used in CNNs for improved convergence and regularization.
