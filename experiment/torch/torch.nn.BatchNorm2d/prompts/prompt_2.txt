**BatchNorm2d** is a PyTorch module that applies batch normalization to 4D input tensors of shape `(N, C, H, W)`, where `N` is batch size, `C` is number of channels, and `H, W` are spatial dimensions. It normalizes each channel across the batch and spatial dimensions to stabilize and accelerate training.

**Key Features:**
- **Normalization Formula:**  
  \( y = \frac{x - \mathrm{E}[x]}{\sqrt{\mathrm{Var}[x] + \epsilon}} \cdot \gamma + \beta \)
- **Learnable Parameters:**  
  Scale (\(\gamma\)) and shift (\(\beta\)) per channel (enabled by default via `affine=True`).
- **Running Statistics:**  
  During training, maintains exponential moving averages of mean and variance (momentum default: 0.1) for evaluation, unless `track_running_stats=False`.
- **Usage:**  
  Primarily used in convolutional neural networks (CNNs) to reduce internal covariate shift.

**Parameters:**
- `num_features`: Number of channels `C` in input.
- `eps`: Small constant for numerical stability (default: 1e-5).
- `momentum`: Weight for updating running statistics (default: 0.1).
- `affine`: If `True`, enables learnable \(\gamma\) and \(\beta\).
- `track_running_stats`: If `True`, tracks running statistics; if `False`, uses batch statistics in both training and eval modes.

**Example:**
```python
m = nn.BatchNorm2d(100)  # With learnable parameters
input = torch.randn(20, 100, 35, 45)
output = m(input)  # Same shape as input
```
