`BatchNorm2d` applies batch normalization to 4D inputs (N, C, H, W), normalizing each channel across spatial dimensions and mini-batches. It stabilizes training by reducing internal covariate shift.

**Key features:**
- Normalizes using per-channel mean/variance over mini-batches:  
  `y = (x - mean) / sqrt(var + eps) * γ + β`
- Maintains running statistics (mean/variance) during training for evaluation (default momentum=0.1)
- Learnable affine parameters γ and β are enabled by default (`affine=True`)
- Set `track_running_stats=False` to always use batch statistics

**Parameters:**
- `num_features`: Number of channels C
- `eps`: Small constant for numerical stability (default 1e-5)
- `momentum`: Weight for updating running statistics (default 0.1)
- `affine`: Enable learnable γ, β (default True)
- `track_running_stats`: Track running statistics (default True)

**Usage:**
```python
m = nn.BatchNorm2d(num_features=100)  # With learnable parameters
output = m(input)  # Input shape: (N, 100, H, W)
```
