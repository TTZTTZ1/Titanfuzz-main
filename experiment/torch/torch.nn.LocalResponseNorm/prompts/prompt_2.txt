**LocalResponseNorm** applies channel-wise normalization to an input tensor using neighboring channels. It adjusts each channel’s value based on the squared values of nearby channels, enhancing contrast and suppressing large uniform responses.

**Key parameters:**
- `size`: number of adjacent channels used for normalization (centered on each channel)
- `alpha`: scaling factor (default: 0.0001)
- `beta`: exponent (default: 0.75)
- `k`: additive constant (default: 1)

**Formula:**  
For each channel \( c \), output \( b_c \) is computed as:
\[
b_c = a_c \left( k + \frac{\alpha}{n} \sum_{c'} a_{c'}^2 \right)^{-\beta}
\]
where the sum runs over \( n \) neighboring channels.

**Input/Output shape:**  
- Input: `(N, C, *)`  
- Output: `(N, C, *)` (same shape)

**Example:**
```python
lrn = nn.LocalResponseNorm(size=2)
output = lrn(torch.randn(32, 5, 24, 24))
```
