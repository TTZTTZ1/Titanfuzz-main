**Summary:**  
`torch.nn.utils.spectral_norm` applies spectral normalization to a specified weight parameter in a PyTorch module. It rescales the weight matrix by its spectral norm (the largest singular value) using the power iteration method, which helps stabilize GAN training by constraining the Lipschitz constant of the discriminator. The normalization is applied via a hook before each `forward()` call.

**Key Details:**
- **Purpose:** Stabilize GAN discriminator/critic training.
- **Mechanism:** Weight tensor is divided by its spectral norm \(\sigma(\mathbf{W})\).
- **Parameters:**  
  - `module`: The PyTorch module containing the weight to normalize.  
  - `name`: Name of the weight parameter (default: `'weight'`).  
  - `n_power_iterations`: Number of power iterations to estimate \(\sigma(\mathbf{W})\) (default: 1).  
  - `eps`: Numerical stability epsilon (default: 1e-12).  
  - `dim`: Dimension corresponding to outputs (default: 0, or 1 for ConvTranspose layers).
- **Returns:** The same module with a spectral norm hook attached.
- **Note:** This version is legacy; the recommended replacement is `torch.nn.utils.parametrizations.spectral_norm()`.

**Example:**
```python

# m.weight is now spectrally normalized before each forward pass.
```
