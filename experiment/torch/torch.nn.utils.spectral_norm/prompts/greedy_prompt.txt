**Summary:**  
`torch.nn.utils.spectral_norm` applies spectral normalization to a specified weight parameter in a PyTorch module. It rescales the weight matrix by its spectral norm (the largest singular value), which helps stabilize GAN training by constraining the Lipschitz constant of the discriminator. The normalization is performed via a hook that runs before each forward pass, using power iteration to estimate the spectral norm.

**Key Points:**
- **Purpose:** Stabilize GAN discriminators/critics.
- **Mechanism:** Weight is divided by its estimated spectral norm:  
  \( W_{SN} = W / \sigma(W) \), where \( \sigma(W) \) is computed via power iteration.
- **Parameters:**  
  - `module`: The PyTorch module containing the weight to normalize.  
  - `name`: Name of the weight parameter (default: `'weight'`).  
  - `n_power_iterations`: Number of power iterations for estimating the spectral norm.  
  - `eps`: Small value for numerical stability.  
  - `dim`: Dimension corresponding to outputs (default varies by layer type).
- **Returns:** The same module with a spectral normalization hook attached.
- **Note:** This version is legacy; the recommended replacement is `torch.nn.utils.parametrizations.spectral_norm()`.

**Example:**
