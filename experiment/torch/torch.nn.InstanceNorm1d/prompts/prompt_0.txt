`torch.nn.InstanceNorm1d` applies instance normalization to 2D (unbatched) or 3D (batched) inputs, normalizing each channel independently per sample. It computes mean and variance per instance in the batch and normalizes using these statistics.

- `affine`: If `True`, adds learnable scale (γ) and shift (β) parameters (default: `False`).
- `track_running_stats`: If `True`, tracks running mean/variance during training for evaluation use (default: `False`).

**Input/Output shape:**  
- Input: `(N, C, L)` (batched) or `(C, L)` (unbatched)
- Output: Same shape as input
