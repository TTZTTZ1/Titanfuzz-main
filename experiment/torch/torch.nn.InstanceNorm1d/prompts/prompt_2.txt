---GENERAL SUMMARY START---
**InstanceNorm1d** applies instance normalization to 2D (unbatched) or 3D (batched) inputs, normalizing each channel independently across each sample in a batch. It is commonly used for tasks like style transfer.

**Key Features:**
- Computes mean and variance per channel for each sample.
- Normalizes using:  
  \( y = \frac{x - E[x]}{\sqrt{\text{Var}[x] + \epsilon}} \times \gamma + \beta \)
- Learnable parameters \(\gamma\) and \(\beta\) are optional (`affine=True`).
- By default, uses instance statistics in both training and evaluation.
- If `track_running_stats=True`, maintains running statistics during training for evaluation use.

**Parameters:**
- `num_features`: Number of channels \(C\).
- `eps`: Small constant for numerical stability (default: 1e-5).
- `momentum`: Weight for updating running statistics (default: 0.1).
- `affine`: Enables learnable scaling/shifting (default: `False`).
- `track_running_stats`: Tracks running mean/variance if `True`.

**Input/Output Shapes:**
- Input: \((N, C, L)\) or \((C, L)\)
- Output: Same shape as input

**Example:**
```python
m = nn.InstanceNorm1d(100, affine=True)  # With learnable parameters
output = m(input)  # input shape: (20, 100, 40)
```
