- **Operation:**  
  `output = beta * input + alpha * (batch1 @ batch2)`  

- **Special behavior:**

  - `out_dtype` optional for CUDA with float16/bfloat16 inputs.  

**Example:**  
```python
>>> M = torch.randn(10, 3, 5)
>>> batch1 = torch.randn(10, 3, 4)
```
