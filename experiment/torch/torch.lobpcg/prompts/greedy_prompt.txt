`torch.lobpcg` computes the `k` largest or smallest eigenvalues and eigenvectors for a symmetric positive definite generalized eigenvalue problem using the LOBPCG algorithm. It supports dense, sparse, and batched matrices.

**Key parameters:**
- `A`: Input matrix (∗, m, m).
- `k`: Number of eigenpairs to compute (default: 1 or based on `X`).
- `B`: Optional matrix for generalized eigenvalue problems (default: identity).
- `X`: Initial eigenvector approximation (dense tensor).
- `largest`: If `True`, compute largest eigenvalues; otherwise smallest (default: `True`).
- `method`: Choose `"ortho"` (robust, default) or `"basic"` (faster but less stable).

**Returns:**
- `E`: Eigenvalues (∗, k).
- `X`: Eigenvectors (∗, m, k).

**Important notes:**
- Backward pass does not support sparse/complex inputs or when `B` is provided.
- `A` is symmetrized automatically if it requires gradients.
- Use `tracker` for iteration monitoring and custom stopping.
- Not applicable if matrix rows < 3 × `k`.
