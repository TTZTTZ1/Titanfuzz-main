**GroupNorm** is a normalization layer that divides input channels into groups and normalizes each group separately. It computes mean and variance per group, applying a learnable affine transformation if enabled. Unlike batch normalization, it uses input statistics in both training and evaluation modes.

**Key parameters:**
- `num_groups`: Number of groups (must divide `num_channels`)
- `num_channels`: Total channels in input
- `eps`: Small constant for numerical stability
- `affine`: Enables learnable scale (γ) and shift (β) parameters

**Usage:**
- Input shape: `(N, C, *)`
- Output shape: Same as input
- Special cases:
  - `num_groups = 1` → equivalent to LayerNorm
  - `num_groups = num_channels` → equivalent to InstanceNorm

**Example:**
```python
m = nn.GroupNorm(3, 6)  # 6 channels divided into 3 groups
output = m(input)
```
