**GroupNorm** is a PyTorch normalization layer that divides input channels into groups and normalizes each group separately. It stabilizes training by computing mean and variance per group, independent of batch size.

**Key Points:**
- **Input:** `(N, C, *)` where `C` must be divisible by `num_groups`.
- **Operation:** Normalizes each group using group-specific statistics:  
  \( y = \frac{x - E[x]}{\sqrt{Var[x] + \epsilon}} \cdot \gamma + \beta \)
- **Parameters:**
  - `num_groups`: Number of channel groups.
  - `num_channels`: Total input channels.
  - `affine`: If `True`, adds learnable scaling (`γ`) and shifting (`β`) per channel.
- **Special Cases:**
  - `num_groups = 1`: Equivalent to **LayerNorm** (normalizes all channels together).
  - `num_groups = num_channels`: Equivalent to **InstanceNorm** (normalizes each channel separately).
- **Usage:** Works in both training and evaluation modes, using input statistics dynamically.

**Example:**
```python
m = nn.GroupNorm(3, 6)  # 6 channels split into 3 groups
output = m(input)       # Same shape as input
```
