**PReLU (Parametric Rectified Linear Unit)** is a learnable activation function that introduces a small slope for negative inputs, defined as:

\[
\text{PReLU}(x) = \max(0, x) + a \cdot \min(0, x)
\]

**Key Features:**
- **Learnable parameter `a`**: Controls the slope for negative values (default initial value: 0.25).
- **Channel-wise or shared**:  
  - `nn.PReLU()`: Single `a` for all input channels.  
  - `nn.PReLU(nChannels)`: Separate `a` per input channel (channel dim is the 2nd input dimension).
- **Weight decay caution**: Avoid using weight decay on `a` for optimal performance.
- **Shape preservation**: Output shape matches input shape.

**Parameters:**
- `num_parameters` (int): Number of `a` to learn (1 or number of input channels). Default: 1.
- `init` (float): Initial value of `a`. Default: 0.25.

**Example:**
```python
m = nn.PReLU()
input = torch.randn(2)
output = m(input)  # Applies element-wise PReLU
```
