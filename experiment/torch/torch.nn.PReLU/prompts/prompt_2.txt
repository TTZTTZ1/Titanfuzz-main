`torch.nn.PReLU` is a learnable activation function that applies a parametric rectified linear unit (PReLU) element-wise. It is defined as:

- **PReLU(x) = max(0, x) + a * min(0, x)**, where `a` is a learnable parameter.

**Key Features:**
- Allows a small negative slope (`a`) for inputs < 0, unlike standard ReLU.
- `a` is initialized to 0.25 by default but can be set via `init`.
- Supports either a single shared `a` across all input channels (default) or separate `a` per channel if `num_parameters` equals the number of input channels.
- **Note:** Do not use weight decay when learning `a` for optimal performance.

**Parameters:**
- `num_parameters` (int): Number of `a` values to learn (1 or number of input channels). Default: 1.
- `init` (float): Initial value of `a`. Default: 0.25.

**Input/Output Shape:**  
- Input: `(*)` (any shape).
- Output: `(*)` (same as input).

**Example:**
```python
m = nn.PReLU()
input = torch.randn(2)
output = m(input)
```
