**PReLU (Parametric Rectified Linear Unit)**  

**Function:**  
Element-wise activation defined as:  
- \( \text{PReLU}(x) = \max(0, x) + a \cdot \min(0, x) \)  
- Equivalent to: \( \text{PReLU}(x) = \begin{cases} x, & \text{if } x \ge 0 \\ a \cdot x, & \text{otherwise} \end{cases} \)  
where \( a \) is a learnable parameter controlling the slope for negative inputs.

**Key Features:**  
- **Learnable parameter \( a \):** Initialized to 0.25 by default.  
- **Channel-wise or shared:**  
  - `nn.PReLU()` → single \( a \) for all input channels.  
  - `nn.PReLU(nChannels)` → separate \( a \) per input channel (channel dim = 2nd dim of input).  
- **Weight decay:** Should not be applied to \( a \) for optimal performance.  
- **Shape:** Input and output have the same shape.

**Parameters:**  
- `num_parameters` (int): Number of \( a \) to learn (1 or number of input channels).  
- `init` (float): Initial value of \( a \).  

**Example:**  
```python
m = nn.PReLU()
input = torch.randn(2)
output = m(input)
```
