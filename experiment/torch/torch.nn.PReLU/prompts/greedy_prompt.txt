**PReLU** is a learnable activation function in PyTorch that applies a parametric rectified linear unit.  

**Key features:**  
- **Function:** `PReLU(x) = max(0, x) + a * min(0, x)`  
  where `a` is a learnable parameter for negative inputs.  
- **Parameters:**  
  - `num_parameters`: Number of `a` values to learn (1 or equal to input channels).  
  - `init`: Initial value of `a` (default 0.25).  
- **Usage:**  
  - Single shared `a` across all channels if `num_parameters=1`.  
  - Separate `a` per channel if `num_parameters` equals input channels.  
- **Notes:**  
  - Avoid weight decay when learning `a`.  
  - Channel dimension is the second dimension of input (if input dim < 2, channels = 1).  
- **Shape:** Input and output shapes are identical.  

**Example:**  
```python
m = nn.PReLU()
output = m(input)  # input shape preserved
```
