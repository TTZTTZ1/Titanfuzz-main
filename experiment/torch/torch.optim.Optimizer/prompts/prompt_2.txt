### **Core Concepts**
- **Optimizer Object**: Holds the current state and updates model parameters based on computed gradients.
- **Parameter Groups**: Supports both global and per-parameter optimization settings (e.g., layer-specific learning rates, weight decay).
- **Step Methods**: 
  - Standard `optimizer.step()` after gradient computation.

### **Supported Algorithms**

- **`lr_scheduler` Module**: Provides schedulers (e.g., `StepLR`, `CosineAnnealingLR`, `ReduceLROnPlateau`) to adjust learning rates during training.
- **Usage Note**: Schedulers should be called **after** `optimizer.step()` (post-PyTorch 1.1.0).

### **Advanced Features**
1. **Weight Averaging**:
   - **SWA (Stochastic Weight Averaging)**: Averages model weights post-training for better generalization.

2. **State Dict Handling**:
   - Supports saving/loading optimizer state (`state_dict`/`load_state_dict`).
