`torch.quantile` computes specified quantiles of a tensor along a given dimension.  

**Key parameters:**  
- `input`: Input tensor.  
- `q`: Quantile value(s) in [0, 1] (scalar or 1D tensor).  
- `dim`: Dimension to reduce (default flattens tensor).  
- `interpolation`: Method when quantile lies between two points (`'linear'`, `'lower'`, `'higher'`, `'midpoint'`, `'nearest'`).  

**Returns:**  
Tensor with quantiles, where the first dimension corresponds to `q` if `q` is a 1D tensor.  

**Example:**  
```python
>>> torch.quantile(a, [0.25, 0.5, 0.75], dim=1)
```
Computes the 25th, 50th, and 75th percentiles along dimension 1.
