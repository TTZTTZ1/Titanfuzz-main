**Key Parameters:**
- `d_model`: Input feature dimension
- `nhead`: Number of attention heads
- `dim_feedforward`: Hidden dimension of feedforward network (default: 2048)
- `dropout`: Dropout probability (default: 0.1)
- `batch_first`: If True, input shape is (batch, seq, feature); otherwise (seq, batch, feature)
- `norm_first`: If True, applies layer norm before attention/feedforward operations

- Supports both regular tensors and NestedTensor inputs (for efficient padding)
- Includes optimized FlashAttention implementation when specific conditions are met (inference mode, batched inputs, compatible activation functions)

- Supports causal masking through the `is_causal` parameter

**Usage Example:**
```python
encoder_layer = nn.TransformerEncoderLayer(d_model=512, nhead=8)
src = torch.rand(10, 32, 512)  # (seq_len, batch, features)
out = encoder_layer(src)
```

The layer serves as a foundational reference implementation, with PyTorch recommending exploration of newer transformer architectures for production use cases.
