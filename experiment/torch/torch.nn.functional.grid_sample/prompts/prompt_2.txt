`torch.nn.functional.grid_sample` performs spatial sampling of an input tensor using a grid of normalized coordinates. It supports 4‑D (spatial) and 5‑D (volumetric) inputs.

**Key behavior:**
- For each output location, the grid provides normalized `(x, y)` (or `(x, y, z)`) coordinates in the range `[-1, 1]`, where `(-1, -1)` corresponds to the top‑left corner and `(1, 1)` to the bottom‑right corner of the input.

- Interpolation can be `'nearest'`, `'bilinear'` (trilinear for 5‑D), or `'bicubic'` (4‑D only).
- The `align_corners` parameter controls whether `-1` and `1` align with pixel centers (`True`) or pixel corners (`False`). Default is `False`.

**Common use:**  
Often paired with `affine_grid` to implement Spatial Transformer Networks. When using CUDA, backward pass may be nondeterministic.
