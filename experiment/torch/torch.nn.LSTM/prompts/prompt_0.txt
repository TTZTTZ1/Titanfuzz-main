The `torch.nn.LSTM` module implements a multi-layer Long Short-Term Memory (LSTM) recurrent neural network. It processes input sequences by applying LSTM cells across time steps and layers.

**Key Parameters:**
- `input_size`: Number of features in input `x`
- `hidden_size`: Number of features in hidden state `h`
- `num_layers`: Number of stacked LSTM layers (default: 1)
- `batch_first`: If `True`, input/output shape is `(batch, seq, feature)` instead of `(seq, batch, feature)`
- `bidirectional`: If `True`, becomes bidirectional LSTM (default: `False`)
- `proj_size`: If >0, uses LSTM with projections (must be < `hidden_size`)
- `dropout`: Dropout probability between layers (except last layer)

**Input/Output Shapes:**
- **Input**: `(L, N, H_in)` or `(N, L, H_in)` if `batch_first=True`
- **Hidden states**: `(D*num_layers, N, H_out)` for `h_n`, `(D*num_layers, N, H_cell)` for `c_n`
- **Output**: `(L, N, D*H_out)` or `(N, L, D*H_out)` if `batch_first=True`

Where:
- `N`: batch size, `L`: sequence length
- `D`: 2 if bidirectional else 1
- `H_in` = `input_size`
- `H_cell` = `hidden_size`
- `H_out` = `proj_size` if >0 else `hidden_size`

**Key Features:**
- Supports packed sequences for variable-length inputs
- Includes optional dropout between layers
- Provides projection capability (when `proj_size > 0`)
- Initializes weights from uniform distribution `U(-√k, √k)` where `k = 1/hidden_size`
- Has known non-determinism issues on CUDA/cuDNN (environment variables can enforce determinism)

**Example:**
```python
rnn = nn.LSTM(10, 20, 2)  # 10 input features, 20 hidden, 2 layers
input = torch.randn(5, 3, 10)  # (seq_len, batch, input_size)
h0 = torch.randn(2, 3, 20)  # (num_layers, batch, hidden_size)
c0 = torch.randn(2, 3, 20)
output, (hn, cn) = rnn(input, (h0, c0))
```
