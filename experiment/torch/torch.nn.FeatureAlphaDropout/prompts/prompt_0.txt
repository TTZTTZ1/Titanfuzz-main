**FeatureAlphaDropout** is a PyTorch module that randomly masks entire channels (feature maps) during training to regularize neural networks. Unlike standard dropout, it sets masked activations to the negative saturation value of the SELU activation function, preserving the self-normalizing properties of networks using SELU. This approach helps maintain zero mean and unit variance in the activations.

**Key Points:**
- **Purpose:** Promotes independence between feature maps, especially useful in convolutional layers where adjacent pixels are strongly correlated.
- **Parameters:** 
  - `p` (float, default 0.5): Probability of masking a channel.
  - `inplace` (bool, default False): If True, performs the operation in-place.
- **Input/Output Shape:** Supports multi-dimensional tensors (e.g., `(N, C, D, H, W)` or `(C, D, H, W)`), returning the same shape as input.
- **Use Case:** Typically applied after `nn.AlphaDropout` or in networks using SELU activations, as described in *Self-Normalizing Neural Networks*.

**Example:**
```python
m = nn.FeatureAlphaDropout(p=0.2)
input = torch.randn(20, 16, 4, 32, 32)
output = m(input)
```
