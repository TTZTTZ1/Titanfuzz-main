**Dropout3d** is a PyTorch module that randomly zeroes out entire 3D feature map channels during training to prevent overfitting.  

**Key points:**  
- **Purpose:** Promotes independence between feature maps in 3D convolutional layers (e.g., from `nn.Conv3d`), unlike standard dropout which acts elementwise.  
- **Input shape:** `(N, C, D, H, W)` or `(C, D, H, W)`, where `C` is the channel dimension.  
- **Parameters:**  
  - `p` (float, default=0.5): Probability of zeroing a channel.  
  - `inplace` (bool): If `True`, modifies input tensor directly.  
- **Use case:** Recommended for early 3D convolution layers where spatial correlations are strong, to avoid merely reducing the effective learning rate.  

**Example:**  
```python
m = nn.Dropout3d(p=0.2)
input = torch.randn(20, 16, 4, 32, 32)  # (batch, channels, depth, height, width)
output = m(input)  # Random channels are zeroed
```
