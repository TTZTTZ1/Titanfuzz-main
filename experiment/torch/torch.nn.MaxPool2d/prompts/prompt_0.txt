**MaxPool2d** is a 2D max pooling layer that reduces spatial dimensions by selecting the maximum value from each sliding window over the input.

**Key Parameters:**
- `kernel_size`: Size of the pooling window (int or tuple).
- `stride`: Step size of the window (defaults to `kernel_size`).
- `padding`: Adds implicit negative infinity padding (default 0).
- `dilation`: Controls spacing between window elements.
- `return_indices`: If `True`, returns max indices for use with `MaxUnpool2d`.
- `ceil_mode`: Uses ceiling instead of floor for output shape calculation.

**Input/Output Shape:**
- **Input:** `(N, C, H_in, W_in)` or `(C, H_in, W_in)`
- **Output:** `(N, C, H_out, W_out)` or `(C, H_out, W_out)`

**Output Dimension Formula:**
\[
H_{out} = \left\lfloor \frac{H_{in} + 2 \times \text{padding}[0] - \text{dilation}[0] \times (\text{kernel\_size}[0] - 1) - 1}{\text{stride}[0]} + 1 \right\rfloor
\]
(Same for width dimension.)

**Usage Example:**
```python
m = nn.MaxPool2d(3, stride=2)  # Square window
output = m(input_tensor)  # Reduces spatial dimensions
```

**Note:** Useful for downsampling in CNNs while preserving the most prominent features.
