- Includes optional class weighting for imbalanced datasets.
- Allows ignoring specific target indices and label smoothing.

**Parameters:**

- `ignore_index`: Target value to ignore in loss computation.
- `reduction`: Specifies loss aggregation (`'none'`, `'mean'`, `'sum'`).
- `label_smoothing`: Smoothing factor between 0 and 1.

- **Input**: `(C)`, `(N, C)`, or `(N, C, d1, d2, ..., dK)` for K-dimensional data.
- **Target**: For class indices: `()`, `(N)`, or `(N, d1, ..., dK)`; for probabilities: same shape as input.
- **Output**: Loss per element if `reduction='none'`, otherwise a scalar.

**Important Notes:**
- When using class probabilities as targets, ensure they form valid probability distributions (values in [0,1], sum to 1). PyTorch does not enforce this automatically.
- Performance is optimized for class indices; use probabilities only when necessary (e.g., label smoothing).
