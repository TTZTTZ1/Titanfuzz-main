`MultiMarginLoss` is a PyTorch loss function for multi-class classification using a hinge loss (margin-based). It penalizes predictions where the correct class score is not sufficiently higher than incorrect class scores by a specified margin.

**Key parameters:**
- `p` (int): Exponent for the loss (1 or 2, default 1).
- `margin` (float): Desired margin between correct and incorrect scores (default 1.0).
- `weight` (Tensor, optional): Per-class weighting.
- `reduction` (str): How to aggregate loss (`'none'`, `'mean'`, `'sum'`).

**Formula (for each sample):**
\[
\text{loss} = \frac{\sum_{i \neq y} \max(0, \text{margin} - x[y] + x[i])^p}{\text{num\_classes}}
\]
If `weight` is provided, multiply by `weight[y]`.

**Shapes:**
- Input: `(N, C)` or `(C)`
- Target: `(N)` or `()` (class indices)
- Output: Scalar (or matching target shape if `reduction='none'`)

**Example:**
```python
loss_fn = nn.MultiMarginLoss()
x = torch.tensor([[0.1, 0.2, 0.4, 0.8]])  # 4 classes
y = torch.tensor([3])                     # target class index
loss = loss_fn(x, y)  # Computes hinge loss
```
