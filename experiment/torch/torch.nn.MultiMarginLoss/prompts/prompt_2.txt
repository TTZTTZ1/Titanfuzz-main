`MultiMarginLoss` is a PyTorch loss function for multi-class classification using a hinge loss (margin-based) approach. It penalizes predictions where the target class score is not sufficiently higher than other class scores by a specified margin.

**Key Parameters:**
- `p` (int): Exponent for the loss (1 or 2, default 1).
- `margin` (float): Desired separation margin (default 1.0).
- `weight` (Tensor, optional): Per-class weighting tensor of size C.
- `reduction` (str): Loss reduction method (`'none'`, `'mean'`, `'sum'`; default `'mean'`).

**Loss Formula:**
For each sample, the loss is computed as:
\[
\text{loss} = \frac{\sum_{i \neq y} \max(0, \text{margin} - x[y] + x[i])^p}{\text{batch size}}
\]
where \( x \) is the input tensor, \( y \) is the target class index, and \( i \) iterates over all non-target classes. If `weight` is provided, each term is scaled by `weight[y]`.

**Input/Output Shapes:**
- **Input:** `(N, C)` or `(C)`, where N is batch size and C is number of classes.
- **Target:** `(N)` or `()`, containing class indices in range `[0, C-1]`.
- **Output:** Scalar (or same as target if `reduction='none'`).

**Example:**
```python
loss_fn = nn.MultiMarginLoss()
x = torch.tensor([[0.1, 0.2, 0.4, 0.8]])  # 1 sample, 4 classes
y = torch.tensor([3])                     # target class index 3
loss = loss_fn(x, y)  # Computes hinge loss
```
