`MultiMarginLoss` is a PyTorch loss function for multi-class classification using a hinge loss (margin-based) approach. It penalizes predictions where the target class score is not sufficiently higher than other class scores by a specified margin.

**Key features:**
- **Formula**: For each sample, loss = Σᵢ max(0, margin − x[y] + x[i])ᵖ / N, where i ≠ target class y
- **Parameters**:
  - `p` (1 or 2): Power for the loss calculation (default: 1)
  - `margin`: Desired separation between target and other classes (default: 1.0)
  - `weight`: Optional per-class weighting tensor
  - `reduction`: How to aggregate loss (`'none'`, `'mean'`, `'sum'`)

**Input/Output shapes:**
- **Input**: (N, C) where N = batch size, C = number of classes
- **Target**: (N) containing class indices (0 ≤ target ≤ C-1)
- **Output**: Scalar loss (or per-element if `reduction='none'`)

**Example**:  
```python
loss_fn = nn.MultiMarginLoss()
x = torch.tensor([[0.1, 0.2, 0.4, 0.8]])  # 4 classes
y = torch.tensor([3])                     # Target class 3
loss = loss_fn(x, y)  # Computes hinge loss
```
