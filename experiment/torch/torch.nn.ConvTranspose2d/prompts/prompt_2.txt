`torch.nn.ConvTranspose2d` is a 2D transposed convolution layer used for upsampling feature maps in neural networks, often in tasks like image generation or segmentation. It applies a learnable kernel to expand spatial dimensions by inserting zeros between input elements (when stride > 1) and convolving.

**Key Parameters:**
- `in_channels`, `out_channels`: Input/output channel counts.
- `kernel_size`: Size of the convolutional filter.
- `stride`: Controls upsampling factor (inserts zeros when >1).
- `padding`: Adjusts implicit zero padding to align input/output shapes with corresponding `Conv2d`.
- `output_padding`: Resolves ambiguity in output size when stride > 1.
- `groups`: Splits input/output channels into independent groups.
- `dilation`: Spacing between kernel elements.
- `bias`: Whether to include a learnable bias term.

**Output Shape:**

```

```
(same for width dimension).

**Notes:**
- Often called "deconvolution," though it is not a true inverse convolution.
- With matching parameters, it can invert the spatial transformation of a `Conv2d` layer.
- Supports optional `output_size` in the forward pass to explicitly define output dimensions.

**Example:**
