- `kernel_size`: Size of the convolutional kernel.
- `stride`: Controls upsampling factor; values >1 insert zeros between input elements.
- `padding`: Adjusts implicit zero padding to align input/output dimensions with corresponding `Conv2d`.
- `output_padding`: Resolves ambiguity when `stride > 1` by slightly increasing output size.
- `dilation`: Spacing between kernel elements.
- `groups`: Controls input-output channel groupings (e.g., depthwise separable convolutions at `groups=in_channels`).

**Note:** It is not a true deconvolution but a transposed convolution, useful for learnable upsampling in neural networks.
