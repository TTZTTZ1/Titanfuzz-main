**Dropout** is a regularization layer that randomly sets a fraction of input elements to zero during training to prevent overfitting.  

**Key points:**  
- **Probability `p`**: Each element has probability `p` of being zeroed (default 0.5).  
- **Scaling**: During training, outputs are scaled by `1/(1-p)` to maintain expected value; during evaluation, it acts as identity.  
- **Independent sampling**: Different elements are zeroed each forward pass.  
- **In-place option**: Can modify input tensor directly if `inplace=True`.  
- **Shape**: Input and output have the same shape.  

**Example:**  
```python
m = nn.Dropout(p=0.2)
output = m(input)  # Randomly zeros ~20% of elements during training
```
