**Dropout** is a regularization layer that randomly sets a fraction of input elements to zero during training (with probability `p`), sampled independently each forward pass. This helps prevent overfitting by reducing co-adaptation of neurons. During training, the remaining elements are scaled by \( \frac{1}{1-p} \) to maintain overall activation magnitude, while at evaluation it acts as an identity function (no dropout applied).

**Key parameters:**
- `p` (float, default 0.5): Probability of zeroing an element.
- `inplace` (bool, default False): If True, performs operation in-place.

**Shape:**  
- Input: Any shape `(*)`
- Output: Same shape as input

**Example:**
```python
m = nn.Dropout(p=0.2)
output = m(input)  # Randomly zeros ~20% of elements during training
```
