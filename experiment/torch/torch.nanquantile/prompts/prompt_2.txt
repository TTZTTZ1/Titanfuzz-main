`torch.nanquantile` computes quantiles while ignoring `NaN` values in the input tensor. It behaves like `torch.quantile` but excludes `NaN` from calculations. If all values in a reduced dimension are `NaN`, the result is `NaN`.  

**Parameters:**  
- `input`: Input tensor.  
- `q`: Quantile value(s) in [0, 1] (scalar or 1D tensor).  
- `dim` (optional): Dimension to reduce; defaults to all dimensions.  
- `keepdim` (optional): Whether to retain the reduced dimension.  

**Keyword arguments:**  
- `interpolation`: Method for quantile interpolation (`linear`, `lower`, `higher`, `midpoint`, `nearest`).  
- `out` (optional): Output tensor.  

**Example:**  
```python
>>> t = torch.tensor([float('nan'), 1, 2])
>>> t.nanquantile(0.5)
```
