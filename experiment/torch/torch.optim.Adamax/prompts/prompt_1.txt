- **Parameters**:

  - `lr` (default: 2e-3): Learning rate.
  - `betas` (default: (0.9, 0.999)): Coefficients for gradient and infinity norm averaging.
  - `eps` (default: 1e-8): Numerical stability term.

  - `foreach` (optional): Uses a foreach implementation for performance (CUDA preferred).
  - `maximize` (default: False): Maximizes the objective instead of minimizing.
  - `differentiable` (default: False): Allows autograd through the optimizer step (may impair performance).
  - `capturable` (default: False): Safe for CUDA graphs/torch.compile (may affect ungraphed performance).

### **Core Methods:**
1. **`step(closure=None)`**: Performs a single optimization step.
2. **`zero_grad(set_to_none=True)`**: Resets gradients; setting to `None` reduces memory usage.
3. **`state_dict()` / `load_state_dict(state_dict)`**: Save/load optimizer state.
4. **`add_param_group(param_group)`**: Adds a parameter group dynamically.

### **Additional Functionality:**

- **State Management**: The state dict includes parameter-specific state and group metadata (e.g., learning rates, parameter names if using `named_parameters`).

### **Usage Example:**

This optimizer is effective for training neural networks with adaptive, stable updates, particularly when handling sparse gradients.
