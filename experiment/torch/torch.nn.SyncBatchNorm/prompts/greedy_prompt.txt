`SyncBatchNorm` is a PyTorch module that extends standard batch normalization to synchronize batch statistics across multiple GPUs during distributed training. It computes mean and variance across all mini-batches within a specified process group, ensuring consistent normalization. Key features include:

- **Usage**: Designed for `DistributedDataParallel` with one GPU per process. Convert existing `BatchNorm*D` layers using `SyncBatchNorm.convert_sync_batchnorm()` before wrapping with DDP.
- **Parameters**: Similar to `BatchNorm`, with added `process_group` to define synchronization scope (defaults to global).
- **Behavior**: Synchronization only occurs during training (`model.train()`); evaluation uses local or running statistics.
- **Example**:
