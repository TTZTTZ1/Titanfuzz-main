`torch.diagonal` extracts the diagonal elements from a tensor along specified dimensions. It returns a view (not a copy) with the diagonal appended as a new last dimension.

**Key parameters:**
- `offset`: selects diagonals above (`>0`), below (`<0`), or on the main diagonal (`0`)
- `dim1` & `dim2`: the two dimensions defining the plane from which diagonals are taken (default: 0 and 1)

**Main features:**
- Works on tensors with at least 2 dimensions
- For batch diagonals, use `dim1=-2, dim2=-1`
- Can be reversed with `torch.diag_embed()` (with explicit dimension specification)

**Example:**
```python
>>> a = torch.randn(3, 3)
>>> torch.diagonal(a, offset=1)  # Elements above main diagonal
```
