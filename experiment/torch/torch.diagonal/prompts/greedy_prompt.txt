---GENERAL SUMMARY START---
`torch.diagonal` extracts the diagonal elements from a tensor along specified dimensions. It returns a view (not a copy) with the diagonal appended as a new dimension at the end.

**Key parameters:**
- `offset`: selects diagonals above (>0), below (<0), or on (0) the main diagonal.
- `dim1` and `dim2`: the two dimensions defining the plane from which the diagonal is taken (default: 0 and 1).

**Main features:**
- Works on tensors with at least 2 dimensions.
- For batch diagonals, use `dim1=-2, dim2=-1`.
- Combined with `torch.diag_embed()` (with explicit dimensions), it can reconstruct a diagonal matrix.

**Example:**
```python
>>> a = torch.randn(3, 3)
>>> torch.diagonal(a, offset=1)
tensor([element above main diagonal])
```
