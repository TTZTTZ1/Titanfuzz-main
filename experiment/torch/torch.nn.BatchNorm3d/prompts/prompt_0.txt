`BatchNorm3d` applies batch normalization to 5D inputs (N, C, D, H, W), normalizing across the batch and spatial dimensions (N, D, H, W) for each channel C. It stabilizes training by adjusting inputs using per-channel mean and variance, with optional learnable scaling (γ) and shifting (β) parameters.

**Key features:**
- Normalizes as:  
  \( y = \frac{x - E[x]}{\sqrt{Var[x] + \epsilon}} \cdot \gamma + \beta \)
- Tracks running statistics during training (by default) for consistent evaluation behavior.
- `affine=True` enables learnable γ and β; `track_running_stats=False` uses batch statistics in both training and evaluation.

**Parameters:**
- `num_features`: Number of channels C.
- `eps`: Small constant for numerical stability (default 1e-5).
- `momentum`: Weight for updating running statistics (default 0.1).
- `affine`: Enables learnable γ and β (default True).
- `track_running_stats`: Tracks running mean/variance (default True).

**Example:**
```python
m = nn.BatchNorm3d(100)  # With learnable parameters
input = torch.randn(20, 100, 35, 45, 10)
output = m(input)  # Same shape as input
```
