The `torch.linalg.pinv` function computes the **Moore-Penrose pseudoinverse** of a matrix, which generalizes the matrix inverse to non‑square or rank‑deficient matrices.  

**Key features:**
- Supports **float, double, cfloat, cdouble** dtypes and **batched inputs**.
- Can use **SVD** (default) or **eigendecomposition** (if `hermitian=True`).
- Singular values below `max(atol, σ₁·rtol)` are discarded (σ₁ is the largest singular value).
- Default `rtol = max(m, n) * eps` if not specified.
- **Recommended** to use `torch.linalg.lstsq` instead for solving linear systems, as it is faster and more stable.

**Parameters:**
- `A`: input tensor of shape `(*, m, n)`.
- `atol`, `rtol`: tolerance values (scalar or tensor).
- `hermitian`: if `True`, assumes `A` is Hermitian (complex) or symmetric (real) and uses `eigh`.
- `out`: optional output tensor.

**Example use:**
```python
import torch
A = torch.randn(3, 5)
A_pinv = torch.linalg.pinv(A)  # shape (5, 3)
```
