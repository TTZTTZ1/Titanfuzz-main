`torch.unique` returns the distinct elements from an input tensor, optionally sorting them and providing additional information.

**Key parameters:**
- `input`: Input tensor.
- `sorted`: If `True` (default), unique elements are returned in ascending order.
- `return_inverse`: If `True`, returns indices mapping original elements to unique output.
- `return_counts`: If `True`, returns counts of each unique element.
- `dim`: Dimension along which to find unique sub-tensors; if `None`, operates on flattened tensor.

**Returns:**
- `output`: Tensor of unique elements.
- `inverse_indices` (optional): Index tensor mapping input to output.
- `counts` (optional): Counts of each unique element.

**Notes:**
- Removes both consecutive and non-consecutive duplicates.
- Currently sorts internally regardless of `sorted` argument; use `torch.unique_consecutive` for pre-sorted inputs to avoid sorting overhead.

**Example:**
```python
>>> torch.unique(torch.tensor([1, 3, 2, 3]))
tensor([1, 2, 3])
```
