**LeakyReLU** is a PyTorch activation function that applies a small, non-zero gradient for negative inputs, preventing "dead neurons" in neural networks.

**Key Features:**
- **Function:** `LeakyReLU(x) = max(0, x) + negative_slope * min(0, x)`
- **Default slope:** `negative_slope = 0.01` (1e-2)
- **In-place option:** `inplace=False` by default (avoids modifying input tensor)
- **Shape:** Output maintains the same shape as input

**Usage:**
```python
import torch.nn as nn
m = nn.LeakyReLU(negative_slope=0.1)  # Custom slope
output = m(input_tensor)
```

**Purpose:** Provides a small gradient for negative values (unlike standard ReLU), helping maintain learning during backpropagation when inputs are negative.
