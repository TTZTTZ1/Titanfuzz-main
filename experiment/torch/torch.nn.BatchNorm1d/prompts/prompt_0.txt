`BatchNorm1d` applies batch normalization to 2D or 3D inputs by normalizing each feature/channel across mini-batches. It stabilizes training by reducing internal covariate shift.

**Key parameters:**
- `num_features`: Number of features/channels `C`
- `eps`: Small constant for numerical stability (default: 1e-5)
- `momentum`: Weight for updating running statistics (default: 0.1)
- `affine`: Whether to learn scaling (`γ`) and shifting (`β`) parameters (default: `True`)
- `track_running_stats`: Whether to track running mean/variance for evaluation (default: `True`)

**Operation:**
- Normalizes as:  
  \( y = \frac{x - \mathrm{E}[x]}{\sqrt{\mathrm{Var}[x] + \epsilon}} \times \gamma + \beta \)
- During training: Uses batch statistics; updates running estimates
- During evaluation: Uses running statistics (if `track_running_stats=True`)

**Input/Output shapes:**
- 2D: `(N, C)` → `(N, C)`
- 3D: `(N, C, L)` → `(N, C, L)`

**Example:**
```python
m = nn.BatchNorm1d(100)  # With learnable parameters
input = torch.randn(20, 100)
output = m(input)
```
