`BatchNorm1d` applies batch normalization to 2D or 3D inputs, normalizing across the batch dimension for each feature/channel. It stabilizes training by reducing internal covariate shift.

**Key features:**
- Normalizes using per-feature mean and variance computed over mini-batches
- Includes learnable scale (γ) and shift (β) parameters when `affine=True`
- Maintains running statistics (mean/variance) during training for evaluation use (configurable via `track_running_stats`)
- Works on shapes `(N, C)` or `(N, C, L)` where N=batch size, C=features/channels, L=sequence length

**Parameters:**
- `num_features`: Number of input features/channels (C)
- `eps`: Small constant for numerical stability (default: 1e-5)
- `momentum`: Weight for updating running statistics (default: 0.1)
- `affine`: Enable learnable γ and β (default: True)
- `track_running_stats`: Track running statistics (default: True)

**Example:**
```python
m = nn.BatchNorm1d(100)  # 100 features
output = m(input)  # Same shape as input
```
