## Core Concepts
- **Parameterizable distributions**: Create distributions with learnable parameters (e.g., `Normal(loc, scale)`)
- **Two gradient estimation methods**:
  - **Score function/REINFORCE**: Uses `sample()` and `log_prob()` for policy gradients
  - **Pathwise derivative/Reparameterization**: Uses `rsample()` for differentiable sampling (e.g., VAEs)

## Main Components

### 1. **Distribution Classes**
- **Base class**: `Distribution` (abstract) with methods like `sample()`, `log_prob()`, `entropy()`, `mean`, `variance`
- **Common distributions**: 
  - Continuous: `Normal`, `Beta`, `Gamma`, `Exponential`, `Dirichlet`, etc.
  - Discrete: `Bernoulli`, `Categorical`, `Binomial`, `Poisson`, etc.
  - Multivariate: `MultivariateNormal`, `LowRankMultivariateNormal`
  - Transformed: `LogNormal`, `HalfNormal`, `Gumbel`, etc.

### 2. **Key Methods**
- `sample()` / `rsample()`: Generate samples (latter is reparameterizable)
- `log_prob(value)`: Compute log probability density/mass
- `entropy()`: Compute distribution entropy
- `cdf()` / `icdf()`: Cumulative distribution and inverse functions

### 3. **Specialized Classes**
- `ExponentialFamily`: Base for exponential family distributions
- `TransformedDistribution`: Apply transforms to base distributions

### 4. **Transforms & Constraints**
- **Transforms**: `AffineTransform`, `ExpTransform`, `SigmoidTransform`, etc.
- **Constraints**: Define valid parameter ranges (e.g., `positive`, `simplex`, `interval`)
- **Constraint registries**: `biject_to()` and `transform_to()` for unconstrained optimization

### 5. **Utilities**
- **KL Divergence**: `kl_divergence(p, q)` with registered implementations for many distribution pairs
- **Validation**: Automatic argument validation with `validate_args`
