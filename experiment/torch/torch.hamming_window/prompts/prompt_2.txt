\( w[n] = \alpha - \beta \cos\left( \frac{2\pi n}{N-1} \right) \), where \( N \) is the full window size.

- Other standard tensor arguments (`dtype`, `device`, etc.) control the output tensor properties.

**Notes:**
- If `window_length = 1`, the window is `[1]`.
- `torch.hamming_window(L, periodic=True)` equals `torch.hamming_window(L+1, periodic=False)[:-1]`.
- This is a generalized version of `torch.hann_window`.

**Returns:** A 1‑D tensor of shape `(window_length,)` containing the window values.
