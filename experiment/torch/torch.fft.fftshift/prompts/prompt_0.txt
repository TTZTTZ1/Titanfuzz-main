`torch.fft.fftshift` reorders FFT output so that negative frequencies appear first, shifting the zero-frequency component to the center of the tensor. It is commonly used to visualize FFT results with frequencies arranged symmetrically around zero.

**Key points:**
- Moves the origin `(0, ..., 0)` to the tensor center along specified dimensions.
- By default, operates on all dimensions; optional `dim` parameter allows selective shifting.
- For even-sized dimensions, the Nyquist frequency is placed at index 0.
- Often paired with `ifftshift` for converting between centered and uncentered grid conventions in spatial/spectral data processing.

**Example:**
```python
f = torch.fft.fftfreq(4)  # [0.0, 0.25, -0.5, -0.25]
torch.fft.fftshift(f)     # [-0.5, -0.25, 0.0, 0.25]
```
