`torch.nn.Upsample` is a PyTorch module for increasing the spatial resolution of 1D, 2D, or 3D data. It supports nearest-neighbor and linear interpolation modes (linear, bilinear, bicubic, trilinear) depending on input dimensions.

**Key parameters:**
- `size` (optional): Target output dimensions.
- `scale_factor` (optional): Multiplier for spatial size (cannot be used with `size`).
- `mode`: Interpolation algorithm (`'nearest'`, `'linear'`, `'bilinear'`, `'bicubic'`, `'trilinear'`).
- `align_corners`: If `True`, aligns input and output corner pixels for linear modes (default: `False`).

**Usage:**
- Input tensors should follow the format: `(batch, channels, [depth], [height], width)`.
- Output size is determined by either `size` or `scale_factor`.
- For downsampling or general resizing, use `torch.nn.functional.interpolate()` instead.

**Example:**
```python
upsample = nn.Upsample(scale_factor=2, mode='bilinear')
output = upsample(input_tensor)
```
