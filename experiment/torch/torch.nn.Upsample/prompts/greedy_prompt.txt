`torch.nn.Upsample` is a PyTorch module for increasing the spatial resolution of 1D, 2D, or 3D data. It supports nearest-neighbor and linear interpolation methods (linear, bilinear, bicubic, trilinear) depending on the input dimensions.

**Key Parameters:**
- `size` or `scale_factor`: Specify the output size or scaling multiplier (only one can be used).
- `mode`: Interpolation algorithm (`'nearest'`, `'linear'`, `'bilinear'`, `'bicubic'`, `'trilinear'`).
- `align_corners`: If `True`, aligns input and output corner pixels for linear interpolations, affecting output values based on input size.

**Usage Notes:**
- Input tensors should be 3D, 4D, or 5D (minibatch × channels × spatial dimensions).
- For general resizing (including downsampling), use `torch.nn.functional.interpolate()` instead.
- The `align_corners` setting changes interpolation behavior, especially noticeable near edges.

**Example:**
```python
upsample = nn.Upsample(scale_factor=2, mode='bilinear')
output = upsample(input)  # Doubles spatial dimensions
```
