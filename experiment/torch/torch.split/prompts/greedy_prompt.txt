`torch.split` divides a tensor into multiple chunks along a specified dimension. It returns a tuple of views of the original tensor.

**Parameters:**
- `tensor`: The input tensor to split.
- `split_size_or_sections`:  
  - If an **integer**, the tensor is split into equally sized chunks of that size (last chunk may be smaller if not divisible).  
  - If a **list of integers**, the tensor is split into chunks with the specified sizes along the dimension.
- `dim`: The dimension along which to split (default is 0).

**Returns:** A tuple of tensors (chunks).

**Example:**
```python
>>> a = torch.arange(10).reshape(5, 2)
>>> torch.split(a, 2)  # Split into chunks of size 2 along dim 0
>>> torch.split(a, [1, 4])  # Split into chunks of sizes 1 and 4
```
