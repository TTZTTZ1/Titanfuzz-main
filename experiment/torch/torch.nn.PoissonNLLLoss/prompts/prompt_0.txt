`PoissonNLLLoss` computes the negative log-likelihood loss for targets following a Poisson distribution. It is used when modeling count data where the target is Poisson-distributed with the input as its rate parameter.

**Key Parameters:**
- `log_input`: If `True`, the input is expected to be in log-space (`exp(input) - target * input`). If `False`, the loss is `input - target * log(input + eps)`.
- `full`: If `True`, includes the Stirling approximation term for the factorial, improving accuracy for larger target values.
- `eps`: Small constant to prevent log(0) when `log_input=False`.
- `reduction`: Specifies how to aggregate the loss (`'none'`, `'mean'`, or `'sum'`).

**Loss Formula (simplified):**  
`loss = exp(input) - target * input` (if `log_input=True`)  
or  
`loss = input - target * log(input + eps)` (if `log_input=False`).  
When `full=True`, an additional Stirling approximation term is added for `target > 1`.

**Usage Example:**
```python
loss_fn = nn.PoissonNLLLoss()
log_input = torch.randn(5, 2, requires_grad=True)
target = torch.randn(5, 2)
output = loss_fn(log_input, target)
output.backward()
```

**Shape:**  
- Input and target must have the same shape `(*)`.  
- Output is a scalar by default (or same shape as input if `reduction='none'`).
