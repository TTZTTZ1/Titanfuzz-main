**PoissonNLLLoss** is a PyTorch loss function for modeling count data where the target follows a Poisson distribution.  

**Key behavior:**  
- Computes negative log-likelihood loss between input and target under the assumption `target ~ Poisson(input)`.  
- If `log_input=True` (default), the loss is `exp(input) - target * input`.  
- If `log_input=False`, the loss is `input - target * log(input + eps)`.  
- The optional `full` parameter adds a Stirling approximation term for the factorial in the loss.  

**Parameters of note:**  
- `log_input`: Whether the input is already in log space.  
- `full`: Whether to include the Stirling term for better accuracy.  
- `eps`: Small constant to avoid log(0) when `log_input=False`.  
- `reduction`: How to aggregate loss (`'none'`, `'mean'`, `'sum'`).  

**Example usage:**  
```python
loss_fn = nn.PoissonNLLLoss()
loss = loss_fn(log_input, target)  # log_input and target same shape
loss.backward()
```

- Input: `(*)`  
- Target: `(*)` (same shape as input)  
- Output: Scalar if reduction is `'mean'` or `'sum'`; otherwise same shape as input.
