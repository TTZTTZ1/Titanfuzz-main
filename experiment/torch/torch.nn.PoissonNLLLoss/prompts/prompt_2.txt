`PoissonNLLLoss` computes the negative log-likelihood loss for targets following a Poisson distribution. It is used when modeling count data.

**Key Parameters:**
- `log_input`: If `True`, the input is expected to be the logarithm of the mean (loss = exp(input) - target * input). If `False`, input is the mean directly (loss = input - target * log(input + eps)).
- `full`: If `True`, includes a Stirling approximation term for the factorial, improving accuracy for larger target values.
- `eps`: Small constant to prevent log(0) when `log_input=False`.
- `reduction`: Controls output reduction (`'none'`, `'mean'`, `'sum'`).

**Loss Formula (simplified):**  
`loss = input - target * log(input) + log(target!)`  
The factorial term is omitted unless `full=True`.

**Example Usage:**
```python

log_input = torch.randn(5, 2, requires_grad=True)
target = torch.randn(5, 2)
output = loss_fn(log_input, target)  # Scalar loss
```

**Shape:**  
- Input and target must have the same shape.
- Output is a scalar if reduction is `'mean'` or `'sum'`, otherwise same shape as input.
