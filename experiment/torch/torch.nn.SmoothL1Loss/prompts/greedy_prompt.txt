**Key behavior:**
- If the absolute error `|x - y| < beta`, it uses a **quadratic (L2) term**: `0.5 * (x - y)^2 / beta`
- Otherwise, it uses an **L1 term**: `|x - y| - 0.5 * beta`

- `reduction` (`'none'`, `'mean'`, `'sum'`): Specifies how to aggregate the loss.
- `beta` (float, default=1.0): Threshold for switching between L1 and L2 loss. Must be ≥ 0.

**Notes:**
- Equivalent to `L1Loss` when `beta = 0`.
- Related to `HuberLoss` but differs in scaling and slope behavior.
- Commonly used in regression tasks like object detection (e.g., Fast R-CNN).

**Shape:**

- Output is a scalar if reduction is applied, otherwise same shape as input.
