`torch.linalg.cholesky` computes the Cholesky decomposition of a Hermitian (complex) or symmetric (real) positive-definite matrix \( A \), returning a lower-triangular matrix \( L \) such that \( A = LL^H \) (or \( A = LL^T \) for real inputs).  

**Key features:**  
- Supports real (`float`, `double`) and complex (`cfloat`, `cdouble`) tensors.  
- Handles batched inputs: `A` shape `(*, n, n)` → output shape `(*, n, n)`.  
- Optional `upper=True` returns an upper-triangular factor (conjugate transpose of the lower-triangular one).  
- Throws an error if any matrix in the batch is not positive-definite.  
- On CUDA, synchronizes device with CPU; use `torch.linalg.cholesky_ex` for a non-synchronizing version.  

**Example use:**  
```python
L = torch.linalg.cholesky(A)          # Lower-triangular Cholesky factor
A_reconstructed = L @ L.T.conj()      # Reconstruct original matrix (approx.)
```
