`torch.linalg.cholesky` computes the Cholesky decomposition of a Hermitian (complex) or symmetric (real) positive-definite matrix **A**, returning a lower-triangular matrix **L** such that **A = L @ Lᴴ** (where ᴴ denotes conjugate transpose).  

**Key features:**  
- Supports real (`float`, `double`) and complex (`cfloat`, `cdouble`) tensors.  
- Handles batched inputs: shape `(*, n, n)` where `*` are batch dimensions.  
- Optional `upper=True` returns an upper-triangular factor (conjugate transpose of the lower-triangular one).  
- Synchronizes CPU with CUDA device; use `torch.linalg.cholesky_ex` for a non-synchronizing version.  
- Raises `RuntimeError` if any matrix in the batch is not positive-definite.  

**Example use:**  
```python
L = torch.linalg.cholesky(A)  # A is positive-definite
```  
This is commonly used for solving linear systems, sampling from multivariate distributions, and matrix inversion.
