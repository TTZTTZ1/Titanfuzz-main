**StepLR** is a PyTorch learning rate scheduler that reduces the learning rate by a fixed factor (`gamma`) at regular intervals (`step_size` epochs).

**Key Features:**
- **Decay Schedule:** Multiplies the learning rate by `gamma` every `step_size` epochs.
- **Integration:** Works alongside other learning rate adjustments.
- **Initialization:** When `last_epoch=-1`, it starts with the optimizer's current learning rate.

**Parameters:**
- `optimizer`: The optimizer to schedule.
- `step_size` (int): Number of epochs between each decay.
- `gamma` (float, default=0.1): Factor by which the learning rate is multiplied.
- `last_epoch` (int, default=-1): Starting epoch index.

**Example:**
```python
scheduler = StepLR(optimizer, step_size=30, gamma=0.1)
for epoch in range(100):
    train()
    validate()
    scheduler.step()  # Called after optimizer.step()
```
With an initial learning rate of 0.05:
- Epochs 0–29: 0.05
- Epochs 30–59: 0.005
- Epochs 60–89: 0.0005, etc.

**Main Methods:**
- `step()`: Updates the learning rate (call after optimizer.step()).
- `get_last_lr()`: Returns the most recent learning rates.
- `get_lr()`: Computes the next learning rates (internal use).
- `state_dict()` / `load_state_dict()`: Save and restore scheduler state.
