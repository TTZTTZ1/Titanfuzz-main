`StepLR` is a PyTorch learning rate scheduler that reduces the learning rate by a multiplicative factor (`gamma`) at fixed epoch intervals (`step_size`).  

**Key points:**  
- **Purpose:** Decays the learning rate every `step_size` epochs.  
- **Parameters:**  
  - `optimizer`: The optimizer to wrap.  
  - `step_size`: Number of epochs between each decay.  
  - `gamma`: Factor by which the learning rate is multiplied (default: 0.1).  
  - `last_epoch`: Index of the last epoch (default: -1, starts from initial LR).  

**Usage example:**  
```python
scheduler = StepLR(optimizer, step_size=30, gamma=0.1)
for epoch in range(100):
    train(...)
    validate(...)
    scheduler.step()  # Call after optimizer.step()
```

**Important methods:**  
- `step()`: Updates the learning rate (call once per epoch).  
- `get_last_lr()`: Returns the current learning rate(s).  
- `state_dict()` / `load_state_dict()`: Save and restore scheduler state.  

**Note:** Works alongside other LR adjustments; decay occurs when `epoch % step_size == 0`.
