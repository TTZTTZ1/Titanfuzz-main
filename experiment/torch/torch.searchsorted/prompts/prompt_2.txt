`torch.searchsorted` finds the insertion indices for `values` into a `sorted_sequence` such that the sorted order is preserved. It returns a tensor of indices with the same shape as `values`.

**Key parameters:**
- `sorted_sequence`: A 1-D or N-D tensor with monotonically increasing innermost dimension (unless `sorter` is provided).
- `values`: The search values (tensor or scalar).
- `side` (or `right`): Controls whether to return the first (`side='left'`, default) or last (`side='right'`) suitable index.
- `sorter`: Optional indices to sort an unsorted sequence.
- `out_int32`: Output dtype (int32 if True, otherwise int64).

**Example:**
```python
>>> sorted_seq = torch.tensor([1, 3, 5, 7, 9])
>>> values = torch.tensor([3, 6, 9])
>>> torch.searchsorted(sorted_seq, values)
tensor([1, 3, 4])  # Indices where values would be inserted to maintain order
```
