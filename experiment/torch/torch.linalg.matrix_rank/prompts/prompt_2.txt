- Synchronizes CUDA devices with the CPU.

- `atol` (optional): Absolute tolerance (default 0).
- `rtol` (optional): Relative tolerance (default based on matrix dimensions if `atol` is not set).
- `hermitian`: Whether `A` is Hermitian/symmetric (default `False`).
- `out` (optional): Output tensor.

**Note:** The positional argument `tol` (NumPy compatibility) is deprecated; use `atol` and `rtol` instead.
