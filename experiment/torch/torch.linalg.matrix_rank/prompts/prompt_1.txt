`torch.linalg.matrix_rank` computes the numerical rank of a matrix by counting singular values (or eigenvalues for Hermitian/symmetric matrices) exceeding a threshold.  

**Key features:**  
- Supports real and complex tensors, including batched inputs.  
- Rank threshold: `max(atol, largest_singular_value * rtol)`.  
- Default `rtol` is `max(m, n) * eps` if not specified.  
- Uses SVD (default) or eigenvalue decomposition for Hermitian matrices.  
- Backward-compatible with NumPy’s `linalg.matrix_rank` but keyword arguments `atol`/`rtol` are preferred over positional `tol`.  

**Example:**  
```python
>>> A = torch.eye(10)
>>> torch.linalg.matrix_rank(A)
tensor(10)
```
