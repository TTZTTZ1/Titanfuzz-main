`AvgPool3d` performs 3D average pooling over an input tensor of shape `(N, C, D, H, W)` or `(C, D, H, W)`. It slides a 3D window across the depth, height, and width dimensions, computing the average of values inside each window.

**Key parameters:**
- `kernel_size`: Size of the pooling window (int or tuple of 3 ints).
- `stride`: Step size of the window (default = kernel_size).
- `padding`: Zero-padding added to all three sides (default = 0).
- `ceil_mode`: If `True`, uses ceil instead of floor for output shape calculation.
- `count_include_pad`: If `True`, includes zero-padding in average computation.
- `divisor_override`: Optional custom divisor to replace `kernel_size` product.

**Output shape:**  
`(N, C, D_out, H_out, W_out)` or `(C, D_out, H_out, W_out)`, where each dimension is computed as:  
`floor((input_size + 2 × padding - kernel_size) / stride + 1)` (or ceil if `ceil_mode=True`).

**Example:**
```python
m = nn.AvgPool3d(kernel_size=3, stride=2)  # Square window
m = nn.AvgPool3d((3, 2, 2), stride=(2, 1, 2))  # Non-square window
```
