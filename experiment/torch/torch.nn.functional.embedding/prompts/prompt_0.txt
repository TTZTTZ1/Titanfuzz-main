`torch.nn.functional.embedding` is a function that retrieves embeddings from a given embedding matrix based on input indices. It serves as a lookup table, commonly used for tasks like word embedding retrieval.

**Key Parameters:**
- `input`: Tensor of indices to look up.
- `weight`: Embedding matrix of shape `(vocab_size, embedding_dim)`.
- `padding_idx` (optional): Specifies an index whose embedding is fixed (gradients ignored).
- `max_norm` (optional): Renormalizes embeddings exceeding this norm (modifies `weight` in-place).
- `scale_grad_by_freq` (optional): Scales gradients by inverse word frequency.
- `sparse` (optional): If `True`, produces sparse gradients for `weight`.

**Shape:**
- Input: Arbitrary shape containing indices.
- Output: Same shape as input, with an added last dimension of size `embedding_dim`.

**Notes:**
- Unlike `torch.nn.Embedding`, this function does not initialize the `padding_idx` row to zeros.
- Analytical gradients for the `padding_idx` row may differ from numerical gradients.

**Example Usage:**
```python
import torch.nn.functional as F

input = torch.tensor([[1, 2], [3, 4]])          # Indices
weight = torch.randn(10, 5)                     # Embedding matrix
output = F.embedding(input, weight)             # Retrieved embeddings
```
