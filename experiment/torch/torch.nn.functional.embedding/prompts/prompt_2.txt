`torch.nn.functional.embedding` is a function that retrieves embeddings from a fixed-size lookup table using input indices. It is commonly used for word embeddings in NLP tasks.

**Key Parameters:**
- `input`: LongTensor of indices to look up.
- `weight`: Embedding matrix of shape `(V, embedding_dim)`, where `V` is the vocabulary size.
- `padding_idx` (optional): Index where gradients are ignored and embeddings remain fixed.
- `max_norm` (optional): Renormalizes embeddings with norms exceeding this value (modifies `weight` in-place).
- `scale_grad_by_freq` (optional): Scales gradients by inverse word frequency.
- `sparse` (optional): Enables sparse gradient computation for `weight`.

**Input/Output Shape:**
- **Input**: Arbitrary shape containing indices.
- **Output**: Same shape as input, with an added last dimension of size `embedding_dim`.

**Example:**
```python
import torch.nn.functional as F
input = torch.tensor([[1, 2], [3, 4]])  # Shape: (2, 2)
weight = torch.randn(10, 5)             # Shape: (10, 5)
output = F.embedding(input, weight)     # Shape: (2, 2, 5)
```

**Note:** Unlike `torch.nn.Embedding`, this function does not initialize the `padding_idx` row to zeros automatically.
