**ASGD (Averaged Stochastic Gradient Descent)** is an optimizer in PyTorch that improves convergence by averaging parameter values over time, reducing variance in stochastic updates.

**Key Parameters:**
- `lr` (default 0.01): Learning rate
- `lambd` (default 1e-4): Decay term for averaging
- `alpha` (default 0.75): Power for learning rate adjustment
- `t0` (default 1e6): Iteration count when averaging begins
- `weight_decay` (default 0): L2 regularization
- `foreach`: Use foreach implementation for better CUDA performance (more memory)
- `maximize`: Maximize instead of minimize objective
- `differentiable`: Allow autograd through optimizer step (impairs performance)
- `capturable`: Safe for CUDA graphs/torch.compile (impairs ungraphed performance)

**Core Functionality:**
- `step(closure)`: Performs optimization step, optionally using closure for loss recomputation
- `zero_grad(set_to_none=True)`: Resets gradients (None reduces memory vs zero tensor)
- `state_dict()` / `load_state_dict()`: Save/load optimizer state
- `add_param_group()`: Add new parameter groups during training

**Special Features:**
- Pre/post hooks for `step()`, `state_dict()`, and `load_state_dict()` for custom processing
- Memory-performance tradeoff via `foreach` parameter
- Compatible with CUDA graphs and torch.compile via `capturable`

**Use Case:** Particularly useful for problems where stochastic gradient noise hinders convergence, as averaging produces more stable parameter estimates over time.
