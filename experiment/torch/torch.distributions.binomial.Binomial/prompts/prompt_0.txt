### 1. **Distribution Base Class**

- Properties: `mean`, `variance`, `mode`, `support`

### 2. **Gradient Estimation Methods**
Two main approaches for backpropagation through random samples:

**Score Function (REINFORCE)**
- Uses `sample()` and `log_prob()`
- Common in reinforcement learning policy gradients
```python

loss = -m.log_prob(action) * reward

**Pathwise Derivative (Reparameterization Trick)**
- Uses `rsample()` for differentiable sampling

```

### 3. **Available Distributions**

- **Multivariate**: Dirichlet, Multinomial, MultivariateNormal, LowRankMultivariateNormal, Wishart, etc.
- **Transformed**: LogNormal, HalfNormal, HalfCauchy, Gumbel, Pareto, etc.
- **Specialized**: MixtureSameFamily, Independent, TransformedDistribution, Relaxed distributions

### 4. **Transforms System**
- Transform between constrained and unconstrained spaces
- Common transforms: `ExpTransform`, `SigmoidTransform`, `SoftmaxTransform`, `StickBreakingTransform`
- Used with `TransformedDistribution` to create complex distributions

### 5. **Constraints System**
- Defines valid parameter ranges for distributions
- Registry system (`biject_to`, `transform_to`) links constraints to transforms

- Pre-registered KL divergences between many distribution pairs
- Extensible via `register_kl()` decorator
