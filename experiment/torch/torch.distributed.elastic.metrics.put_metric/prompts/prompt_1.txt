     ```python

     def my_function():
         pass
     ```
   - Publishes:
     - `<module_or_class>.success`
     - `<module_or_class>.failure`
     - `<module_or_class>.duration.ms`

### **Built-in Handlers:**
- `ConsoleMetricHandler`: Outputs metrics to the console.
- `NullMetricHandler`: Discards all metrics (default).

### **Custom Handlers:**
Implement the `MetricHandler` interface to send metrics to custom destinations (e.g., databases, monitoring systems).

