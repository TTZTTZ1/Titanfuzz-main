`torch.any` tests whether any element in a tensor evaluates to `True`. It has two main forms:

1. **Global check**: Returns a single boolean tensor indicating if any element in the entire input tensor is `True`.
   ```python
   torch.any(input) → Tensor
   ```

2. **Dimension-wise check**: Reduces over specified dimension(s), returning a boolean tensor indicating if any element is `True` along each slice.
   ```python
   torch.any(input, dim, keepdim=False) → Tensor
   ```

**Key points:**
- Output dtype is `bool` for all supported dtypes except `uint8` (where it remains `uint8`).
- If `dim` is specified, reduction occurs along that dimension; if `dim=None`, all dimensions are reduced.
- `keepdim` controls whether reduced dimensions are retained as size 1 or squeezed out.

**Examples:**
- `torch.any(tensor([0, 1, 2]))` → `tensor(True)`
- For a 2D tensor, `torch.any(a, dim=1)` returns a 1D tensor with `True` for rows containing any `True` value.
