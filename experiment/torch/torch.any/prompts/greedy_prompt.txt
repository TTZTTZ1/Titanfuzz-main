1. **Global check**: Returns a single boolean tensor indicating if any element in the entire input tensor is `True`.
2. **Dimension-wise reduction**: For a specified dimension `dim`, returns a boolean tensor indicating if any element along that dimension is `True`. The `keepdim` argument controls whether the reduced dimension is retained.

**Key notes:**
- Output dtype is `bool` for all supported types except `uint8` (where it remains `uint8`).
- If `dim` is not specified, the operation reduces over all dimensions.
- The function aligns with NumPy’s behavior for boolean output.

**Examples:**
- `torch.any(tensor)` → single boolean value for the whole tensor.
- `torch.any(tensor, dim=1)` → boolean values for each row (or along dimension 1).
