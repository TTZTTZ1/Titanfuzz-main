**Key Parameters:**
- `distance_function`: Customizable non-negative function to measure similarity (default: `nn.PairwiseDistance`).
- `margin`: Minimum required difference between positive and negative distances.
- `swap`: If `True`, uses distance swapping to handle hard negatives.
- `reduction`: Applies `'mean'`, `'sum'`, or `'none'` to the loss output.

**Formula:**  
For each triplet \((a, p, n)\):  
\[
\text{loss} = \max(d(a, p) - d(a, n) + \text{margin}, 0)
\]  
where \(d\) is the distance function.

**Use Cases:**  
- Metric learning, face recognition, recommendation systems.  
- Supports custom distances (e.g., L-infinity, cosine dissimilarity) or built-in functions.

**Example:**
```python
triplet_loss = nn.TripletMarginWithDistanceLoss(
    distance_function=lambda x, y: 1 - F.cosine_similarity(x, y),
    margin=1.0
```
