`TripletMarginWithDistanceLoss` is a PyTorch loss function that learns embeddings by ensuring that anchor samples are closer to positive samples than to negative samples by a specified margin. It generalizes the standard triplet loss by allowing a custom distance function.

**Key Features:**
- **Inputs**: Three tensors (anchor, positive, negative) of shape `(N, *)`.
- **Custom Distance**: Accepts any non-negative, real-valued function (default: `PairwiseDistance`).
- **Margin**: Penalizes cases where negative distance ≤ positive distance + margin.
- **Swap Option**: If enabled, uses the anchor-positive-negative distance swap from the Balntas et al. paper.
- **Reduction**: Supports `'none'`, `'mean'`, or `'sum'`.

**Formula**:  
For each triplet `(a, p, n)` in the batch:  
`loss = max(d(a, p) - d(a, n) + margin, 0)`  
where `d` is the distance function.

**Example Use**:
```python
triplet_loss = nn.TripletMarginWithDistanceLoss(
    distance_function=lambda x, y: 1 - F.cosine_similarity(x, y),
    margin=0.5
)

```
