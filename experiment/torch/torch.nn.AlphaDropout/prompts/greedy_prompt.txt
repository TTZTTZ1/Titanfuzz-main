**AlphaDropout** is a PyTorch dropout variant that preserves the **mean and standard deviation** of inputs with zero mean and unit variance. It is designed to work with **SELU activations** to maintain self-normalizing network properties.

**Key features:**
- Maintains input statistics (mean=0, std=1) during training
- Randomly masks elements with probability `p` (default: 0.5)
- Scales and shifts masked values to preserve normalization
- Acts as identity during evaluation

**Usage:**
```python
import torch.nn as nn
m = nn.AlphaDropout(p=0.2)  # 20% dropout probability
output = m(input_tensor)    # Same shape as input
```

**Parameters:**
- `p`: Dropout probability (float)
- `inplace`: Optional in-place operation (bool)
