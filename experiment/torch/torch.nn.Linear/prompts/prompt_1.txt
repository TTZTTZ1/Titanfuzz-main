**y = xAᵀ + b**, where **A** is the weight matrix and **b** is the bias.

**Key Parameters:**
- `in_features` – size of each input sample
- `out_features` – size of each output sample
- `bias` – if `False`, no bias term is learned (default: `True`)

**Shape:**
- **Input:** `(*, in_features)`
- **Output:** `(*, out_features)` (all dimensions except the last match the input)

**Learnable Parameters:**
- `weight` – shape `(out_features, in_features)`, initialized uniformly from `U(-√k, √k)` with `k = 1 / in_features`
- `bias` – shape `(out_features)`, initialized similarly if `bias=True`

**Example:**
```python
m = nn.Linear(20, 30)   # 20 input features → 30 output features
input = torch.randn(128, 20)
output = m(input)       # shape: (128, 30)
```

**Notes:**
- Supports TensorFloat32 on CUDA.
- On certain ROCm devices with float16 inputs, backward precision may differ.
- Includes methods like `reset_parameters()` to reinitialize weights and biases.
