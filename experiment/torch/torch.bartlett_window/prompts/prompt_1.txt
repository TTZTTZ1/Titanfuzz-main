`torch.bartlett_window` generates a Bartlett (triangular) window of a specified length. It returns a 1D tensor where values follow a triangular shape, peaking at the center.

**Key parameters:**
- `window_length` (int): Size of the window.
- `periodic` (bool, optional): If `True`, creates a periodic window suitable for functions like `torch.stft` by omitting the last duplicate value; effectively uses `window_length + 1` in the formula. Default is `True`.

**Formula:**
For a window of size \( N \) (adjusted if `periodic=True`), the values are:
\[
w[n] = 1 - \left| \frac{2n}{N-1} - 1 \right|
\]
which increases linearly to the midpoint and then decreases.

**Note:** If `window_length = 1`, the window is simply `[1]`.

**Keyword arguments** control the tensor properties (`dtype`, `device`, `requires_grad`, etc.), with only floating-point dtypes supported.
