**RandomStructured** is a PyTorch pruning method that randomly removes entire channels (structured pruning) from a tensor.

### Key Features:
- **Structured Pruning**: Prunes full channels along a specified dimension (`dim`, default = -1).
- **Random Selection**: Channels are selected randomly for pruning.
- **Flexible Pruning Amount**: `amount` can be an integer (absolute number of channels) or a float (fraction of channels, 0.0–1.0).

### Main Methods:
1. **`apply(module, name, amount, dim=-1)`**  
   Attaches pruning to a module parameter, adding a forward pre-hook and mask.

2. **`compute_mask(t, default_mask)`**  
   Generates a random mask that zeros out channels along `dim`.

3. **`prune(t, default_mask=None, importance_scores=None)`**  
   Returns a pruned tensor using the computed mask.

4. **`remove(module)`**  
   Removes pruning reparameterization but keeps the tensor permanently pruned.

### Usage Example:
```python
import torch.nn.utils.prune as prune

# Randomly prune 30% of channels in the last dimension of a convolutional layer
prune.RandomStructured.apply(module, name="weight", amount=0.3, dim=-1)
```

### Notes:
- Pruning is **permanent** after `remove()` is called.
- The original tensor is stored as `weight_orig`, and the mask as `weight_mask`.
- Works on unpruned tensors; respects existing masks if reapplied.
