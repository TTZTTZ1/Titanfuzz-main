`torch.jit.trace` converts a Python function or `nn.Module` into TorchScript by recording tensor operations executed with example inputs. It produces either a `ScriptFunction` (for standalone functions) or a `ScriptModule` (for modules) that can be optimized and run independently of Python.

**Key Points:**
- **Ideal for** deterministic operations on tensors and simple data structures (tuples, lists, dicts of tensors).
- **Limitations:** Cannot capture data-dependent control flow (e.g., conditionals, loops) or dynamic behaviors (e.g., training/eval mode differences). Use `torch.jit.script` for such cases.
- **Inputs:** Provide either `example_inputs` (tuple) or `example_kwarg_inputs` (dict) to trace the function.
- **Options:** Includes validation (`check_trace`), strictness control (`strict`), and tolerance settings for numeric verification.

**Example:**
