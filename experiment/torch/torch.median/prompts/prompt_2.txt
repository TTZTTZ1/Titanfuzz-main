**Function:** `torch.median`  
**Purpose:** Computes the median value(s) of a tensor.

---

### **Two Usage Modes:**

1. **Global Median**  
   `torch.median(input)` → `Tensor`  
   Returns the median of all elements in `input`.  
   - For an even number of elements, returns the **lower median**.  
   - For the average of the two medians, use `torch.quantile(input, q=0.5)`.

2. **Median Along a Dimension**  
   `torch.median(input, dim, keepdim=False, *, out=None)` → `(values, indices)`  
   Returns a namedtuple containing:  
   - `values`: Median of each slice along `dim`.  
   - `indices`: Index of each median value in `dim`.  
   - If `keepdim=True`, output retains the reduced dimension as size 1.

---

### **Key Notes:**
- **Even-sized tensors:** Returns the lower median (not the average).  
- **Non-unique medians:** `indices` may not point to the first occurrence; behavior can vary between CPU/GPU.  
- **Gradients:** Deterministic for global median, but non-deterministic when `dim` is specified.

---

### **Example:**
```python
# Global median
a = torch.tensor([1.5219, -1.5212, 0.2202])
torch.median(a)  # tensor(0.2202)

# Median along dimension
b = torch.randn(4, 5)
torch.median(b, dim=1)  # Returns (values, indices) per row
```
