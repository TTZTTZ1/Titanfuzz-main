`torch.nn.functional.pad` pads an input tensor along specified dimensions. 

**Key points:**
- **Padding size (`pad`)**: Specified as a tuple of even length, starting from the last dimension and moving forward. For example, `(left, right)` pads the last dimension, `(left, right, top, bottom)` pads the last two dimensions.
- **Modes**: Supports `'constant'` (default), `'reflect'`, `'replicate'`, and `'circular'`. Constant padding works for any dimension; other modes are limited to padding the last 1–3 dimensions depending on input dimensionality.
- **Parameters**:
  - `input`: N-dimensional tensor to pad.
  - `pad`: Tuple defining padding amounts per dimension.
  - `mode`: Padding mode.
  - `value`: Fill value for constant padding (default 0).
- **Note**: CUDA backend may introduce nondeterministic behavior in the backward pass.

**Example**:  
`F.pad(tensor, (1, 2, 3, 4), mode='constant', value=0)` pads the last dimension by (1, 2) and the second-to-last by (3, 4).
