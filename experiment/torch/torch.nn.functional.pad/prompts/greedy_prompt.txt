`torch.nn.functional.pad` pads an input tensor along specified dimensions.  

**Key points:**  
- **Padding size (`pad`)**: Specified as a tuple of even length, starting from the last dimension and moving forward. Each pair corresponds to `(padding_left, padding_right)` for a dimension.  
- **Padding modes**:  
  - `'constant'`: Pads with a fixed value (default 0). Works for any number of dimensions.  
  - `'reflect'`, `'replicate'`, `'circular'`: Only supported for padding the last 1, 2, or 3 dimensions depending on input dimensionality.  
- **Parameters**:  
  - `input`: N‑dimensional tensor.  
  - `pad`: m‑tuple where m is even and m/2 ≤ input dimensions.  
  - `mode`: One of `'constant'`, `'reflect'`, `'replicate'`, `'circular'`.  
  - `value`: Fill value for constant padding (default 0).  

**Example**:  
```python
# Pad last dimension by 1 on each side
out = F.pad(tensor, (1, 1), mode='constant', value=0)
```  

**Note**: CUDA backend may introduce nondeterministic behavior in the backward pass.
