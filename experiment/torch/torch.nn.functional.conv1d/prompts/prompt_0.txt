`torch.nn.functional.conv1d` performs a 1D convolution over an input tensor. It applies learnable filters (kernels) to extract features from sequential data, such as time series or audio signals.

**Key parameters:**
- `input`: Shape `(batch, in_channels, width)`
- `weight`: Shape `(out_channels, in_channels/groups, kernel_width)`
- `bias`: Optional, shape `(out_channels)`
- `stride`: Step size for kernel movement
- `padding`: Controls input padding (`'valid'` for no padding, `'same'` for output matching input width when stride=1)
- `dilation`: Spacing between kernel elements
- `groups`: Divides input channels into separate groups for depthwise convolution

**Notes:**
- Supports complex data types and TensorFloat32
- CUDA/CuDNN may use nondeterministic algorithms by default
- `padding='same'` has performance considerations with even-length kernels and odd dilations

**Example:**
