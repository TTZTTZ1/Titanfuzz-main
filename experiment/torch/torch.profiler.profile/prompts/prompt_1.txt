The PyTorch Profiler is a performance analysis tool for collecting metrics during model training and inference. It helps identify expensive operations, analyze input shapes, examine stack traces, and visualize execution traces.

**Key Components:**
- **`torch.profiler.profile`**: Main context manager for profiling, supporting configurable activities (CPU/CUDA/XPU), memory tracking, shape recording, and FLOPs estimation.
- **`schedule()`**: Allows cyclic profiling (wait/warmup/active phases) to reduce overhead in long-running jobs.
- **`tensorboard_trace_handler()`**: Exports traces for visualization in TensorBoard.
- **`_KinetoProfile`**: Lower-level API wrapping autograd profiling (experimental).

```python

    on_trace_ready=tensorboard_trace_handler('./logs')
```

- Tracks operator execution time, memory allocation, and kernel activity

- Exports traces in Chrome JSON format or for TensorBoard
- Includes ITT APIs (`itt.mark()`, `itt.range_push/pop()`) for Intel instrumentation

**Notes:**

- Enabling shape/stack tracing adds overhead
