`torch.flatten` reshapes a tensor into a one-dimensional array or flattens a specified range of dimensions, preserving element order. It returns a view when possible, otherwise a copy.

- `start_dim`: First dimension to flatten (default 0).

**Example:**  
```python
>>> t = torch.tensor([[[1, 2], [3, 4]], [[5, 6], [7, 8]]])
>>> torch.flatten(t)
tensor([1, 2, 3, 4, 5, 6, 7, 8])
>>> torch.flatten(t, start_dim=1)
tensor([[1, 2, 3, 4],
        [5, 6, 7, 8]])
```
