## Core Components

Abstract base class with essential methods:

### 4. **KL Divergence**
- `kl_divergence()`: Computes KL divergence between distributions

## Common Usage Patterns

### REINFORCE (Policy Gradient)
```python
probs = policy_network(state)

action = m.sample()
```

### Reparameterization Trick
```python

loss.backward()
```
