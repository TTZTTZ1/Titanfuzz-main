The `torch.Tensor.to()` method converts a tensor's data type (dtype), device (e.g., CPU/GPU), or both. It returns a new tensor with the specified properties unless the original already matches, in which case it returns the original (unless `copy=True` is set).

**Key features:**
- **Three calling signatures:**
  1. `to(dtype, ...)` – changes only the data type.
  2. `to(device, dtype=None, ...)` – changes device and optionally dtype.
  3. `to(other_tensor, ...)` – copies dtype and device from another tensor.
- **Efficient:** Returns the original tensor if it already meets the target criteria (no copy).
- **Gradient handling:** If converting to an integer dtype, `requires_grad` is automatically set to `False`.
- **Options:** Supports non-blocking transfers (for async device moves) and explicit memory format control.

**Example use:**
```python
tensor.to(torch.float64)          # Convert dtype
tensor.to('cuda')                 # Move to GPU
tensor.to(other_tensor)           # Match another tensor's dtype/device
```
