`torch.Tensor.repeat` creates a new tensor by repeating the original tensor along specified dimensions. It copies the data (unlike `expand`) and requires a sequence of integers (`repeats`) indicating how many times to repeat along each dimension. The output size is determined by multiplying each dimension's size by its corresponding repeat value.

**Key points:**
- Copies tensor data.
- Similar to `numpy.tile`, not `numpy.repeat` (use `torch.repeat_interleave` for that).
- Input: `*repeats` (integers or sequence of integers).
- Output: New tensor with repeated elements.

**Example:**
```python
x = torch.tensor([1, 2, 3])
x.repeat(2, 3)  # Repeat twice along dim 0, three times along dim 1
# tensor([[1, 2, 3, 1, 2, 3, 1, 2, 3],
#         [1, 2, 3, 1, 2, 3, 1, 2, 3]])
```
