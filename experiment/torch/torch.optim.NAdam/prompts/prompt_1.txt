**NAdam** is a PyTorch optimizer that combines **Nesterov momentum** with the **Adam** algorithm. It adapts learning rates per parameter using moving averages of gradients (first moment) and squared gradients (second moment), while incorporating lookahead momentum for improved convergence.

### **Key Features:**
- **Nesterov Momentum Integration**: Uses a momentum schedule (`μ_t`) based on `momentum_decay` for accelerated updates.
- **Adam-like Adaptation**: Maintains bias-corrected first and second moment estimates.
- **Weight Decay Options**:  
  - Standard L2 penalty (added to gradient)  
  - **Decoupled weight decay** (NAdamW variant) applied directly to parameters.
- **Flexible Configuration**: Supports `foreach` (performance), `capturable` (CUDA graphs), `differentiable`, and `maximize` modes.

### **Core Algorithm Steps:**
1. Compute gradient `g_t`.
2. Apply weight decay (coupled or decoupled).
3. Update momentum terms `μ_t`, `μ_{t+1}` using `momentum_decay`.
4. Update first (`m_t`) and second (`v_t`) moment estimates.
5. Compute bias-corrected Nesterov momentum term `m̂_t`.
6. Update parameters:  
   `θ_t = θ_t - γ * m̂_t / (√v̂_t + ε)`

### **Typical Usage:**
```python
optimizer = torch.optim.NAdam(
    model.parameters(),
    lr=0.002,
    betas=(0.9, 0.999),
    weight_decay=0.01,
    momentum_decay=0.004,
    decoupled_weight_decay=False  # Set True for NAdamW
)
```

### **Key Parameters:**
- `lr`: Learning rate (default: 2e-3)
- `betas`: Coefficients for moment estimates
- `momentum_decay`: Controls Nesterov momentum decay (default: 4e-3)
- `decoupled_weight_decay`: Enables AdamW-style weight decay

### **Advantages:**
- Often converges faster than Adam in practice
- Combines benefits of Nesterov momentum and adaptive learning rates
- Compatible with PyTorch's optimizer utilities (schedulers, checkpointing, hooks)

**Reference**: [Incorporating Nesterov Momentum into Adam](https://openreview.net/forum?id=OM0jvwB8jIp57ZJjtNEZ)
