- **Parameters**:

  - `capturable` (default: False): Safe for CUDA graphs/torch.compile (may reduce performance otherwise).
  - `differentiable` (default: False): Allows autograd through the optimizer step (impairs performance).

1. **Initialization**: Tracks first (`m`) and second (`v`) moment estimates (initially zero).
2. **Update Steps**:
   - Computes gradients (negated if `maximize=True`).
   - Applies weight decay (either coupled or decoupled).
   - Updates momentum with Nesterov acceleration using `momentum_decay`.
   - Computes bias-corrected moment estimates.
   - Updates parameters using the adaptive learning rate.

### **Additional Methods:**
- `step(closure)`: Performs a single optimization step.
- `zero_grad(set_to_none=True)`: Resets gradients (set to `None` by default for memory efficiency).
- `state_dict()` / `load_state_dict()`: Save/load optimizer state.
- `add_param_group()`: Add new parameter groups dynamically.
- **Hooks**: Pre/post hooks for `step`, `state_dict`, and `load_state_dict` for custom processing.

### **Use Case:**
Suitable for training deep neural networks where adaptive learning rates and Nesterov momentum may improve convergence speed and stability. Choose `decoupled_weight_decay=True` for NAdamW (often better generalization). Use `foreach=True` on CUDA for performance (if memory permits).
