# First, here is a general summary of the API's functionality and usage:
---GENERAL SUMMARY START---
**CTCLoss** is a PyTorch loss function for sequence-to-sequence tasks where input and output sequences may be unaligned (e.g., speech recognition, handwriting recognition). It computes the loss by summing probabilities over all possible alignments between a continuous input sequence and a discrete target sequence.

---

### **Key Parameters**
- **blank** (int, default=0): Index of the blank label.
- **reduction** (str): `'none'`, `'mean'`, or `'sum'` (default=`'mean'`).
- **zero_infinity** (bool): If `True`, infinite losses (from overly short inputs) are zeroed (default=`False`).

---

### **Input Shapes**
| Argument | Shape | Description |
|----------|-------|-------------|
| `log_probs` | `(T, N, C)` or `(T, C)` | Log-softmax probabilities. `T`=input length, `N`=batch size, `C`=number of classes (including blank). |
| `targets` | `(N, S)` or `(sum(target_lengths),)` | Target sequences (class indices, excluding blank). |
| `input_lengths` | `(N,)` or `()` | Lengths of each input sequence (`≤ T`). |
| `target_lengths` | `(N,)` or `()` | Lengths of each target sequence (`≤ S`). |

---

### **Usage Notes**
- **Target formats**:  
  - **Padded**: Shape `(N, S)` with `target_lengths` specifying valid lengths.  
  - **Unpadded**: Concatenated 1D tensor with `target_lengths` summing to its length.
- **CuDNN requirements**:  
  - Targets in concatenated format.  
  - All `input_lengths = T`.  
  - `target_lengths ≤ 256`, `dtype=torch.int32`, `log_probs.dtype=torch.float32`.
- **Non-determinism**: CUDA+CuDNN may use non-deterministic algorithms; set `torch.backends.cudnn.deterministic = True` for reproducibility.

---

### **Example (Padded Targets)**
```python
ctc_loss = nn.CTCLoss()
loss = ctc_loss(log_probs, targets, input_lengths, target_lengths)
loss.backward()
```
Where `log_probs` is the output of `log_softmax` over classes.

---

### **Reference**
Graves et al., *Connectionist Temporal Classification: Labelling Unsegmented Sequence Data with Recurrent Neural Networks* (ICML 2006).
