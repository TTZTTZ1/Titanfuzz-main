- `n_fft`: FFT size.
- `hop_length`: Step between windows (default: `n_fft // 4`).
- `win_length`: Window length (default: `n_fft`).
- `window`: Window function tensor (e.g., Hann window); if `None`, a rectangular window is used (not recommended).
- `center`: If `True`, pads the signal so frames are centered.
- `onesided`: For real input, returns only positive frequencies (default: `True`).
- `return_complex`: Must be explicitly set; returns complex output (preferred) or real with separate real/imaginary dimensions.
- `normalized`: If `True`, scales output by `(frame_length)^{-0.5}`.

**Returns:**
- Complex tensor of shape `(*, N, T)` if `return_complex=True`.
- Real tensor of shape `(*, N, T, 2)` if `return_complex=False`, where the last dimension holds real/imaginary parts.

**Important warnings:**
- Always specify `return_complex` explicitly (default is transitioning to `True`).
- Provide a tapered window (e.g., `torch.hann_window`) to avoid artifacts.
