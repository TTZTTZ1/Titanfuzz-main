- **n_fft**: Size of the Fourier transform.
- **hop_length**: Step between windows (defaults to `n_fft // 4`).
- **win_length**: Window length (defaults to `n_fft`).
- **window**: Optional window function (e.g., Hann window); if not provided, a rectangular window is used (may cause artifacts).
- **center**: If `True`, pads input so frames are centered.
- **onesided**: For real inputs, returns only non‑redundant frequencies (default `True`).

**Output shape:**

- If `return_complex=False`: `(B?, N, T, 2)` (real + imaginary).

**Important warnings:**
- Always specify `return_complex` for real inputs (default transitioning to `True`).
- Providing a window (e.g., `torch.hann_window`) is recommended to avoid artifacts.
- For real output format, use `torch.view_as_real()` on the complex result instead of `return_complex=False`.
