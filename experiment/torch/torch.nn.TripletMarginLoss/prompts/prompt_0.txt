- **Loss Formula**:  
  \( L(a, p, n) = \max\{d(a_i, p_i) - d(a_i, n_i) + \text{margin}, 0\} \)

  \( d(x_i, y_i) = \lVert x_i - y_i \rVert_p \), with optional small ε for numerical stability.
- **Parameters**:
  - `margin` (default: 1.0): Minimum desired separation.
  - `p` (default: 2): Norm degree for distance (e.g., 2 for Euclidean).
  - `eps` (default: 1e-6): Small constant for stability.
  - `swap` (default: False): Option to use distance swapping as per the referenced paper.
  - `reduction` (default: 'mean'): How to aggregate loss ('none', 'mean', or 'sum').

**Usage**:
- **Input Shape**: `(N, D)` where N is batch size and D is feature dimension.
- **Output**: Scalar if reduction is 'mean'/'sum'; otherwise `(N,)`.
- **Example**:
  ```python
  loss_fn = nn.TripletMarginLoss(margin=1.0, p=2)
  loss = loss_fn(anchor, positive, negative)
  ```

**Purpose**:  
Used in metric learning and embedding tasks to ensure that similar samples are closer in the feature space than dissimilar ones.
