`torch.multinomial` samples indices from a multinomial distribution defined by input probabilities or weights. Each row of the input tensor is treated as a probability distribution, and the function returns a tensor with `num_samples` indices per row, drawn according to those probabilities.

**Key points:**
- Input rows must be non‑negative, finite, and have a non‑zero sum (they don’t need to sum to 1).
- If `replacement=False` (default), sampling is without replacement and `num_samples` must not exceed the number of non‑zero elements in the row.
- If `replacement=True`, sampling is with replacement and any number of samples can be drawn.
- Output shape:  
  - Vector input → vector of length `num_samples`.  
  - Matrix input with `m` rows → `(m, num_samples)`.

**Example:**
```python
weights = torch.tensor([0, 10, 3, 0], dtype=torch.float)
torch.multinomial(weights, 2)                     # e.g., tensor([1, 2])
torch.multinomial(weights, 4, replacement=True)   # e.g., tensor([2, 1, 1, 1])
```
