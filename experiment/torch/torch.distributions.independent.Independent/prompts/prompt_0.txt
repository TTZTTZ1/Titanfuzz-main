## Core Components

### 1. **Distribution Base Class**
Abstract base class with essential methods:
- `sample()` / `rsample()`: Generate samples
- `log_prob()`: Compute log probability
- `entropy()`: Distribution entropy
- `mean`, `variance`, `mode`: Distribution properties
- `support`: Valid value constraints

### 2. **Available Distributions**
Includes common distributions:
- **Univariate**: Bernoulli, Beta, Binomial, Categorical, Cauchy, Exponential, Gamma, Geometric, Laplace, Normal, Poisson, Uniform
- **Multivariate**: Dirichlet, Multinomial, MultivariateNormal, LowRankMultivariateNormal
- **Continuous**: LogNormal, Gumbel, StudentT, Weibull

### 3. **Transforms & Constraints**
- **Transforms**: Bijective transformations (Affine, Exp, Sigmoid, Softmax, etc.) for reparameterization
- **Constraints**: Define valid parameter ranges (simplex, positive definite, interval, etc.)
- **Constraint Registry**: `transform_to()` and `biject_to()` for constrained optimization

### 4. **KL Divergence**
- `kl_divergence()`: Compute KL divergence between distributions
- Extensible via `register_kl()` decorator

## Common Usage Patterns

### REINFORCE (Policy Gradient)
```python
probs = policy_network(state)
m = Categorical(probs)
action = m.sample()
loss = -m.log_prob(action) * reward
loss.backward()
```

### Reparameterization Trick
```python
params = policy_network(state)
m = Normal(*params)
action = m.rsample()  # Differentiable
loss = -reward
loss.backward()
```
