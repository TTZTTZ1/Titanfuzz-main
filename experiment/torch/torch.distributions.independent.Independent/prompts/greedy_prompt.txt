The `torch.distributions` package provides a comprehensive framework for working with probability distributions in PyTorch. Here's a concise summary of its key functionality:

## **Core Purpose**
- Enables construction of stochastic computation graphs with gradient estimators for optimization
- Supports both score function (REINFORCE) and pathwise derivative (reparameterization) gradient estimation methods

## **Key Components**

### **1. Distribution Base Class**
- Abstract base class for all probability distributions
- Common methods: `sample()`, `rsample()`, `log_prob()`, `cdf()`, `icdf()`, `entropy()`
- Properties: `mean`, `variance`, `mode`, `support`

### **2. Available Distributions**
**Continuous**: Normal, Beta, Gamma, Exponential, Dirichlet, Cauchy, Laplace, StudentT, Uniform, etc.
**Discrete**: Bernoulli, Binomial, Categorical, Poisson, Geometric, Multinomial, etc.
**Multivariate**: MultivariateNormal, LowRankMultivariateNormal, Dirichlet
**Transformed**: LogNormal, HalfNormal, Gumbel, Weibull, etc.

### **3. Key Features**
- **Batch/Event Shapes**: Supports batched parameters and samples
- **Parameter Validation**: Automatic constraint checking via `arg_constraints`
- **Gradient Estimation**:
  - `sample()` + `log_prob()` = Score function estimator (REINFORCE)
  - `rsample()` = Pathwise derivative estimator (reparameterization trick)
- **KL Divergence**: Pre-registered KL computations between distribution pairs
- **Transforms**: Bijective transformations for constrained parameter spaces

### **4. Practical Usage Examples**

**REINFORCE (Policy Gradient):**
```python
probs = policy_network(state)
m = Categorical(probs)
action = m.sample()
loss = -m.log_prob(action) * reward
loss.backward()
```

**Reparameterization (VAEs):**
```python
params = policy_network(state)
m = Normal(*params)
action = m.rsample()  # Differentiable sample
loss = -reward
loss.backward()
```

- **Transformed Distributions**: Compose base distributions with transforms
- **Constraint Registry**: `transform_to()` and `biject_to()` for constrained optimization
- **Exponential Family**: Specialized class for distributions in exponential family form

### **6. Design Principles**
- Follows TensorFlow Distributions package design
- Enables stochastic gradient estimators for optimization
- Supports both discrete and continuous distributions
- Provides automatic broadcasting and shape handling

The package is essential for probabilistic modeling, reinforcement learning, variational inference, and any application requiring differentiable sampling from probability distributions.
