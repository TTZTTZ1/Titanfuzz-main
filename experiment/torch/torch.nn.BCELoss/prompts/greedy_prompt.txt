`BCELoss` (Binary Cross Entropy Loss) is a PyTorch loss function used for binary classification tasks. It measures the difference between predicted probabilities and true binary labels (values between 0 and 1).

**Key Features:**
- Computes loss per element:  
  `loss = -[y * log(x) + (1 - y) * log(1 - x)]`
- Supports optional per-element weighting via `weight` tensor.
- Reduction options: `'none'` (no reduction), `'mean'` (default, average loss), or `'sum'` (sum loss).
- Automatically clamps log outputs to ≥ -100 to avoid infinite gradients when predictions are near 0 or 1.

**Usage Example:**
```python

predictions = torch.sigmoid(model_output)  # Ensure values in [0,1]
loss = loss_fn(predictions, target_labels)
```

**Note:** Inputs should be probabilities (e.g., after a sigmoid activation), not raw logits.
