`BCELoss` computes the Binary Cross Entropy between predicted probabilities and target values (both between 0 and 1). It is commonly used for binary classification or reconstruction tasks like auto-encoders.

**Key features:**
- **Formula:** For each element, loss = `-w_n * [y_n * log(x_n) + (1 - y_n) * log(1 - x_n)]`, where `x` is input probability, `y` is target, and `w` is an optional weight.
- **Reduction options:** `'none'` (no reduction), `'mean'` (default, average loss), or `'sum'` (sum loss).
- **Numerical stability:** Clamps log outputs to ≥ -100 to avoid infinite loss/gradients when probabilities approach 0 or 1.
- **Shape:** Input and target must have the same shape; output is a scalar (or same shape if `reduction='none'`).

**Example usage:**
```python
loss_fn = nn.BCELoss()
output = loss_fn(sigmoid(predictions), targets)
```
