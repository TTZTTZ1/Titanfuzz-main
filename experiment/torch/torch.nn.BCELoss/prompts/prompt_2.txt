`BCELoss` (Binary Cross Entropy Loss) is a PyTorch loss function used for binary classification tasks, measuring the error between predicted probabilities and target values (which must be between 0 and 1). It computes:

\[
\ell(x, y) = -w_n [y_n \log x_n + (1 - y_n) \log(1 - x_n)]
\]

where \(x\) are predictions, \(y\) are targets, and \(w\) is an optional weight tensor.

**Key features:**
- **Clamping**: Log outputs are clamped to ≥ -100 to avoid infinite loss/gradients when predictions are near 0 or 1.
- **Reduction options**: `'none'` (no reduction), `'mean'` (default, average loss), or `'sum'` (sum loss).
- **Shape**: Input and target must have the same shape; output is a scalar unless `reduction='none'`.

**Example usage:**
```python
loss_fn = nn.BCELoss()
predictions = torch.sigmoid(model_output)  # Apply sigmoid to get probabilities
loss = loss_fn(predictions, targets)
```

**Note**: Typically used with a sigmoid activation layer to ensure predictions are in [0, 1].
