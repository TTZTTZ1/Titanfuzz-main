`torch.isclose` compares two tensors element-wise and returns a boolean tensor indicating whether corresponding elements are considered "close" based on specified tolerance criteria.

**Key behavior:**
- For finite values, closeness is defined as:  
  `|input - other| ≤ rtol × |other| + atol`
- For non-finite values (inf, NaN), closeness is determined by equality, with an option to treat NaN values as equal via `equal_nan`.
- Default tolerances: `rtol=1e-05`, `atol=1e-08`.

**Parameters:**
- `input`, `other`: tensors to compare
- `rtol`: relative tolerance
- `atol`: absolute tolerance
- `equal_nan`: if `True`, NaN == NaN returns `True`

**Example:**
```python
>>> torch.isclose(torch.tensor([1.0, 2.0]), torch.tensor([1.00001, 2.00002]))
tensor([True, False])
```
