\[
| \text{input} - \text{other} | \leq \text{rtol} \times | \text{other} | + \text{atol}

- `atol`: absolute tolerance (default 1e-08)
- `equal_nan`: if `True`, treats NaN values as equal (default `False`)

**Special cases:**
- Non‑finite values (inf, -inf) are considered close only if they are exactly equal.
- When `equal_nan=True`, NaN values are regarded as equal to each other.

**Example:**
```python
>>> torch.isclose(torch.tensor([1.0, 2.0, 3.0]), torch.tensor([1.0000001, 3.0, 4.0]))
```
