`torch.diag` creates a diagonal matrix from a vector or extracts a diagonal from a matrix.

- `0`: Main diagonal.
- `> 0`: Diagonals above the main diagonal.
- `< 0`: Diagonals below the main diagonal.

**Example uses:**
- `torch.diag(vector)` → Creates a diagonal matrix.
- `torch.diag(matrix, k)` → Extracts the k-th diagonal from the matrix.
