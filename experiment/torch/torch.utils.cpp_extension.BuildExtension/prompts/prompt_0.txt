- **AOT Compilation** (via `setup.py`):
  ```python
  from setuptools import setup
  from torch.utils.cpp_extension import BuildExtension, CppExtension

  setup(

      cmdclass={'build_ext': BuildExtension}
  )
  ```

- **JIT Compilation** (runtime loading):
  ```python

  module = load(name='my_ext', sources=['my_ext.cpp'], verbose=True)
  ```

- **Ninja**: Recommended for faster builds; worker count can be controlled via `MAX_JOBS` environment variable.

This module simplifies integrating custom, high-performance C++/CUDA/SYCL code into PyTorch workflows, enabling both pre-compiled extensions and dynamic runtime compilation.
