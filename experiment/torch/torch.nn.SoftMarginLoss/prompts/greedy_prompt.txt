**Key points:**  
- **Purpose**: Optimizes logistic loss between input `x` and target `y` (values 1 or -1).  
- **Formula**:  
  \[
  \text{loss}(x, y) = \frac{1}{n} \sum_i \log(1 + \exp(-y[i] \cdot x[i]))
  \]
  where \( n \) is the number of elements in `x`.  
- **Parameters**:  
  - `reduction`: `'none'`, `'mean'` (default), or `'sum'`.  
  - `size_average` and `reduce` are deprecated (use `reduction`).  
- **Shapes**:  
  - Input and target can have any shape `(*)`.  
  - Output is scalar if reduction is `'mean'` or `'sum'`; otherwise same shape as input.  

**Usage example**:  
```python

```
