---GENERAL SUMMARY START---
`torch.repeat_interleave` repeats each element of a tensor a specified number of times along a given dimension.  

**Key points:**  
- **Different from `torch.repeat()`** — it repeats individual elements, not whole tensor blocks.  
- **Parameters:**  
  - `input`: Tensor to repeat.  
  - `repeats`: Integer or tensor specifying repetitions per element.  
  - `dim`: Dimension along which to repeat (default flattens tensor).  
  - `output_size`: Optional; pre‑specifies output size to avoid synchronization.  
- **Returns:** Tensor with same shape as input except along `dim`, where size = sum of repeats.  

**Examples:**  
- `x = [1, 2, 3]`, `repeats=2` → `[1, 1, 2, 2, 3, 3]`.  
- Can repeat rows/columns using `dim`.  
- If `repeats` is a tensor, each element repeated accordingly.  

**Special form:** `torch.repeat_interleave(repeats)` generates sequence `[0, 1, 1, 2, 2, 2, …]` based on `repeats`.
---GENERAL SUMMARY END---
