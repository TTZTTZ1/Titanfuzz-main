`torch.repeat_interleave` repeats each element of a tensor a specified number of times along a given dimension.  

**Key points:**  
- **Different from `torch.Tensor.repeat()`** — it repeats individual elements rather than repeating the whole tensor.  
- **Parameters:**  
  - `input`: The tensor to repeat.  
  - `repeats`: Number of repetitions per element (can be a scalar or a tensor).  
  - `dim`: Dimension along which to repeat (default flattens the tensor).  
  - `output_size`: Optional; pre‑specifies output size to avoid synchronization.  
- **Returns:** A tensor with the same shape as `input`, except along `dim`, where size = sum of repeats.  

**Examples:**  
- `torch.tensor([1, 2, 3])` with `repeats=2` → `[1, 1, 2, 2, 3, 3]`.  
- Can repeat rows/columns when `dim` is specified.  
- If `repeats` is a tensor, each element is repeated accordingly.  

**Special form:** `torch.repeat_interleave(repeats)` generates `[0 repeated repeats[0] times, 1 repeated repeats[1] times, …]`.
---GENERAL SUMMARY END---
