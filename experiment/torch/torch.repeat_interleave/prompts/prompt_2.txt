`torch.repeat_interleave` repeats elements of a tensor along a specified dimension, similar to `numpy.repeat`.  

**Key parameters:**  
- `input`: The tensor to repeat.  
- `repeats`: Number of repetitions per element (int or tensor).  
- `dim`: Dimension along which to repeat (default flattens the tensor).  
- `output_size`: Optional; pre‑specifies output size to avoid synchronization overhead.  

**Behavior:**  
- If `repeats` is a tensor, each element is repeated according to the corresponding value.  
- If `dim` is not given, the input is flattened first.  
- The output shape matches the input except along the repeated dimension, where size becomes the sum of repeats.  

**Example highlights:**  
```python
>>> torch.repeat_interleave(torch.tensor([1, 2, 3]), 2)
tensor([1, 1, 2, 2, 3, 3])
```
When `repeats` is the only argument, it generates a sequence where each index `i` is repeated `repeats[i]` times, starting from 0.
