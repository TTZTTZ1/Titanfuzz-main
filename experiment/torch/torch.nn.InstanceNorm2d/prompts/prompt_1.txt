`InstanceNorm2d` applies instance normalization to 4D inputs (e.g., images with shape `(N, C, H, W)`), normalizing each channel separately for each sample in a batch. It computes mean and variance per instance and channel, using the formula:

\[
y = \frac{x - \mathrm{E}[x]}{\sqrt{\mathrm{Var}[x] + \epsilon}} \cdot \gamma + \beta
\]

- **Key parameters**:  
  - `num_features`: Number of channels `C`.  
  - `affine`: If `True`, adds learnable scale (`γ`) and shift (`β`) parameters.  
  - `track_running_stats`: If `True`, tracks running statistics during training for evaluation (uncommon for instance norm).  
  - `eps`: Small constant for numerical stability.  
  - `momentum`: Momentum for updating running statistics (if tracked).

- **Default behavior**:  
  Uses instance-specific statistics in both training and evaluation unless `track_running_stats=True`.

- **Use case**:  
  Commonly used in style transfer and image generation to normalize individual samples independently, preserving stylistic features.

- **Example**:  
  ```python
  m = nn.InstanceNorm2d(100, affine=True)  # With learnable parameters
  output = m(input)  # input shape: (20, 100, 35, 45)
  ```
