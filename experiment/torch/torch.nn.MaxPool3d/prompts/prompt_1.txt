`MaxPool3d` applies 3D max pooling over an input tensor of shape `(N, C, D, H, W)` or `(C, D, H, W)`. It slides a 3D window across the depth, height, and width dimensions, outputting the maximum value within each window.

**Key Parameters:**
- `kernel_size`: Size of the pooling window (int or tuple of 3 ints).
- `stride`: Step size of the window (defaults to `kernel_size`).
- `padding`: Implicit negative infinity padding added to all sides (int or tuple).
- `dilation`: Spacing between kernel elements (int or tuple).
- `return_indices`: If `True`, returns max indices for use with `MaxUnpool3d`.
- `ceil_mode`: Uses ceiling instead of floor for output shape calculation.

**Output Shape:**  
Given input `(N, C, D_in, H_in, W_in)`, output dimensions are computed as:

\[
\text{Dim}_{out} = \left\lfloor \frac{\text{Dim}_{in} + 2 \times \text{padding} - \text{dilation} \times (\text{kernel\_size} - 1) - 1}{\text{stride}} + 1 \right\rfloor
\]

(Applied separately to depth, height, and width.)

**Example:**
```python
m = nn.MaxPool3d(kernel_size=3, stride=2)  # Square window
output = m(input)  # Reduces spatial dimensions via max pooling
```
