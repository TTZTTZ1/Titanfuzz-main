`MaxPool3d` is a PyTorch module that performs 3D max pooling over an input tensor with shape `(N, C, D, H, W)` or `(C, D, H, W)`. It slides a 3D window across the depth, height, and width dimensions, outputting the maximum value within each window.

**Key parameters:**
- `kernel_size`: size of the pooling window (int or tuple of 3 ints)
- `stride`: step size for sliding the window (defaults to `kernel_size`)
- `padding`: implicit negative infinity padding added to all sides (int or tuple)
- `dilation`: spacing between kernel elements (int or tuple)
- `return_indices`: if `True`, returns max value indices for use with `MaxUnpool3d`
- `ceil_mode`: uses ceil (instead of floor) to calculate output shape when `True`

**Output shape:** `(N, C, D_out, H_out, W_out)` or `(C, D_out, H_out, W_out)`, where each dimension is computed as:
```
floor((input_size + 2*padding - dilation*(kernel_size-1) - 1) / stride + 1)
```

**Example:**
```python
m = nn.MaxPool3d(3, stride=2)  # 3x3x3 window, stride 2
output = m(input)  # input shape: (20, 16, 50, 44, 31)
```
