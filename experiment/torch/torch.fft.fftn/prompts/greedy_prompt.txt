`torch.fft.fftn` computes the **N-dimensional discrete Fourier transform** of a tensor. It transforms all specified dimensions into the frequency domain, returning both positive and negative frequencies (full complex output).  

**Key parameters:**  
- `input`: Input tensor (real or complex).  
- `s`: Optional size to pad/trim dimensions before transforming.  
- `dim`: Dimensions to transform (default: all dimensions).  
- `norm`: Normalization mode (`"backward"` (default), `"forward"`, or `"ortho"`).  

**Notes:**  
- For real inputs, the output satisfies Hermitian symmetry (redundant frequencies).  
- Use `rfftn` for real inputs to get compact output (positive frequencies only).  
- Supports half-precision on CUDA (SM53+) with power-of-2 lengths.  

**Example:**

fftn_result = torch.fft.fftn(x)  # Transform all dimensions
