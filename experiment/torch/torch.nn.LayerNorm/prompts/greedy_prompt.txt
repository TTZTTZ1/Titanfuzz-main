`torch.nn.LayerNorm` applies layer normalization over specified dimensions of an input tensor. It normalizes the input by subtracting the mean and dividing by the standard deviation over the last `D` dimensions defined by `normalized_shape`, then scales and shifts with learnable parameters if enabled.

**Key Parameters:**
- `normalized_shape`: Shape of dimensions to normalize (e.g., `embedding_dim` for NLP, `[C, H, W]` for images).
- `eps`: Small constant for numerical stability (default: 1e-5).
- `elementwise_affine`: If `True`, enables learnable scaling (`weight`) and shifting (`bias`) parameters (default: `True`).
- `bias`: If `False`, disables the learnable bias (only when `elementwise_affine=True`).

**Characteristics:**
- Computes statistics from input data in both training and evaluation modes.
- Unlike batch or instance normalization, it applies per-element affine transformations.
- Variance is calculated using the biased estimator (`correction=0`).

**Example Usage:**
```python
# NLP: Normalize over embedding dimension
layer_norm = nn.LayerNorm(embedding_dim)
# Image: Normalize over channel and spatial dimensions
layer_norm = nn.LayerNorm([C, H, W])
```

**Input/Output Shape:** Same shape as input `(N, *)`.
