`torch.nn.LayerNorm` applies **layer normalization** across specified dimensions of an input tensor, normalizing each element independently by subtracting the mean and dividing by the standard deviation over those dimensions. It is commonly used in NLP and vision tasks to stabilize training.

**Key parameters:**
- `normalized_shape`: Shape of the dimensions to normalize (e.g., `embedding_dim` for NLP, `[C, H, W]` for images).
- `eps`: Small constant added to the denominator for numerical stability (default: 1e-5).
- `elementwise_affine`: If `True`, applies learnable per-element scaling (`weight`) and shifting (`bias`).
- `bias`: If `False`, disables the learnable bias (only when `elementwise_affine=True`).

**Operation:**  
For input \( x \), it computes:  
\[
y = \frac{x - \mathrm{E}[x]}{\sqrt{\mathrm{Var}[x] + \epsilon}} \cdot \gamma + \beta
\]
where \( \gamma \) (weight) and \( \beta \) (bias) are learnable if `elementwise_affine=True`. Statistics are computed over the last `len(normalized_shape)` dimensions.

**Usage example:**
```python
# NLP: normalize over embedding dimension
layer_norm = nn.LayerNorm(embedding_dim)
# Vision: normalize over channel and spatial dimensions
layer_norm = nn.LayerNorm([C, H, W])
```
**Note:** Unlike batch normalization, layer normalization uses **input-dependent statistics** in both training and evaluation modes.
