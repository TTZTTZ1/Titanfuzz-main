### **Backend Selection Guide**
- **CUDA GPU training**: Use NCCL (best performance)
- **CPU training**: Use Gloo
- **XPU GPU training**: Use XCCL
- **InfiniBand**: NCCL for GPU, Gloo/MPI for CPU

## **Best Practices**

### **Initialization**
```python
import torch.distributed as dist
dist.init_process_group(backend='nccl', init_method='env://')
```

### **Typical Usage Pattern**
```python
# Each process gets unique rank
rank = dist.get_rank()
world_size = dist.get_world_size()

# Collective operations
tensor = torch.ones(2, 2).cuda()
dist.all_reduce(tensor, op=dist.ReduceOp.SUM)

# Cleanup
dist.destroy_process_group()
```

### **DeviceMesh (for complex parallelism)**
