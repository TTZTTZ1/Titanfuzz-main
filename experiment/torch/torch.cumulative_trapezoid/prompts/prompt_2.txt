`torch.cumulative_trapezoid` computes the cumulative integral of a tensor using the trapezoidal rule along a specified dimension. It returns a cumulative value at each step, similar to how `.cumsum` works for summation.

**Key parameters:**
- `y`: Input tensor of values to integrate.
- `x` (optional): Tensor specifying variable spacing between values along `dim`.
- `dx` (optional): Constant spacing (default is 1).
- `dim`: Dimension along which to integrate (default is -1).

**Behavior:**
- If neither `x` nor `dx` is provided, unit spacing is assumed.
- With `dx`, spacing is constant; with `x`, spacing can be arbitrary.
- Output size along `dim` is one less than `y`’s size along that dimension.

**Example:**
```python
y = torch.tensor([1, 5, 10])
torch.cumulative_trapezoid(y)  # → tensor([3., 10.5])
```
This performs cumulative integration: first trapezoid area = (1+5)/2 = 3, cumulative sum = (1+5)/2 + (5+10)/2 = 10.5.
