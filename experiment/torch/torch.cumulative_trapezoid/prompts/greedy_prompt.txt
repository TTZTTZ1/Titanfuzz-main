`torch.cumulative_trapezoid` computes the cumulative integral of a tensor using the trapezoidal rule along a specified dimension. It returns a cumulative value for each spacing interval, analogous to how `cumsum` relates to `sum`.

**Key parameters:**
- `y`: Input tensor of values to integrate.
- `x` (optional): Tensor specifying variable spacing between `y` values.
- `dx` (optional): Constant spacing (defaults to 1 if neither `x` nor `dx` is provided).
- `dim`: Dimension along which to integrate (default: last dimension).

**Key behavior:**
- When only `y` is given, assumes uniform spacing of 1.
- With `dx`, uses constant spacing (multiplies result by `dx`).
- With `x`, uses variable spacing defined by `x`.
- Output size along `dim` is one less than input size.

**Example:**
```python
y = torch.tensor([1, 5, 10])
torch.cumulative_trapezoid(y)  # → tensor([3., 10.5])
# Equivalent to cumulative sum of trapezoidal areas: 
# First interval: (1+5)/2 = 3, second adds (5+10)/2 = 7.5 → 10.5
```
