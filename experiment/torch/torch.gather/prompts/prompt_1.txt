`torch.gather` selects elements from an input tensor along a specified dimension (`dim`) using indices provided in an `index` tensor. The output tensor has the same shape as `index`, with each element determined by the corresponding index value at that position along the chosen dimension.

**Key points:**
- `input` and `index` must have the same number of dimensions.
- For all dimensions `d` except `dim`, `index.size(d) ≤ input.size(d)`.
- No broadcasting between `input` and `index`.

**Example:**
```python
t = torch.tensor([[1, 2], [3, 4]])
torch.gather(t, 1, torch.tensor([[0, 0], [1, 0]]))
# Output: tensor([[1, 1],
#                 [4, 3]])
```
Here, for dimension 1 (columns), indices `[[0,0], [1,0]]` pick: from row 0 → columns 0 and 0 (`[1,1]`); from row 1 → columns 1 and 0 (`[4,3]`).
