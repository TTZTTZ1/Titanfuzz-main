The output tensor has the same shape as the index tensor, with each element determined by replacing the index value at the corresponding position into the given dimension of the input.

**Key points:**
- `input` and `index` must have the same number of dimensions.
- For all dimensions `d` except `dim`, `index.size(d) ≤ input.size(d)`.
- No broadcasting between `input` and `index`.

**Example:**
```python

torch.gather(t, 1, torch.tensor([[0, 0], [1, 0]]))
```
