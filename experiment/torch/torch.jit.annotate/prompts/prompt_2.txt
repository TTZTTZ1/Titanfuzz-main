`torch.jit.annotate` is a type‑hinting utility for TorchScript that explicitly specifies the type of a value or expression when the compiler cannot infer it correctly. It acts as a pass‑through (no‑op) at runtime outside TorchScript.

**Key use cases:**
- Empty containers (e.g., `[]`, `{}`) that TorchScript would otherwise assume contain `Tensor`.

**Example:**
