- **dilation**: Spacing between kernel elements (single value or tuple)
- **groups**: Divides input channels into separate groups for grouped convolutions

**Notes:**
- Supports complex data types and TensorFloat32
- CUDA/CuDNN may use nondeterministic algorithms by default
- `padding='same'` maintains input dimensions when stride=1 but may impact performance with certain kernel/dilation combinations
