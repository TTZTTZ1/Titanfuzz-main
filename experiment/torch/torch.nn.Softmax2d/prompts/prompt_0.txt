**Key points:**
- **Input shape:** `(N, C, H, W)` or `(C, H, W)`  
- **Output shape:** Same as input, with values normalized to the range `[0, 1]`  
- **Operation:** For each spatial position `(h_i, w_j)`, softmax is computed over all `C` channels, making the channel-wise probabilities sum to 1 at that location.  
- **Use case:** Commonly used in semantic segmentation models to produce per-pixel class probabilities.  

**Example:**
```python

output = m(input)  # same shape, channel values normalized per pixel
```
