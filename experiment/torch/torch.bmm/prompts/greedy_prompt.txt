`torch.bmm` performs a **batch matrix-matrix multiplication** between two 3D tensors.  

**Key points:**  
- Both input tensors must be 3‑D with the same batch size.  
- If `input` is shape `(b, n, m)` and `mat2` is `(b, m, p)`, the output is `(b, n, p)`.  
- Each batch `i` computes: `output[i] = input[i] @ mat2[i]`.  
- **Does not broadcast** — for broadcasting, use `torch.matmul` instead.  
- Supports optional `out_dtype` on CUDA for mixed‑precision inputs.  

**Example:**  
```python
>>> input = torch.randn(10, 3, 4)
>>> mat2 = torch.randn(10, 4, 5)
>>> res = torch.bmm(input, mat2)  # shape: (10, 3, 5)
```
