`torch.bmm` performs a **batch matrix-matrix multiplication** between two 3D tensors.  

**Key details:**
- **Inputs:** Both tensors must be 3D with the same batch size:  
  - `input`: shape `(b, n, m)`  
  - `mat2`: shape `(b, m, p)`  
- **Output:** Shape `(b, n, p)`, where each output `i` is `input[i] @ mat2[i]`.  
- **No broadcasting** — for broadcasting, use `torch.matmul` instead.  
- **Optional:** `out_dtype` (CUDA only) to specify output dtype for float16/bfloat16 inputs.  

**Example:**
```python
>>> input = torch.randn(10, 3, 4)
>>> mat2 = torch.randn(10, 4, 5)
>>> res = torch.bmm(input, mat2)  # shape: (10, 3, 5)
```
