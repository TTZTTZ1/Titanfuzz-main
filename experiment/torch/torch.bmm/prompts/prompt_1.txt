`torch.bmm` performs batch matrix multiplication between two 3D tensors.  
- **Inputs**: Both tensors must have the same batch size (`b`).  
  - `input`: shape `(b, n, m)`  
  - `mat2`: shape `(b, m, p)`  
- **Output**: Shape `(b, n, p)`, where each batch result is `input[i] @ mat2[i]`.  
- **Key points**:  
  - Does **not** broadcast; for broadcasting, use `torch.matmul`.  
  - Supports `out_dtype` on CUDA for mixed-precision inputs.  
- **Example**:  
  ```python
  >>> torch.bmm(torch.randn(10, 3, 4), torch.randn(10, 4, 5)).shape
  torch.Size([10, 3, 5])
  ```
