`torch.meshgrid` generates coordinate grids from one-dimensional input tensors, primarily used for visualization or grid-based computations. It takes a list of 1D tensors and returns a sequence of N-dimensional tensors, each representing a coordinate axis expanded to the combined shape of all inputs.

**Key Points:**
- **Input:** List of scalars or 1D tensors.
- **Output:** Sequence of tensors, each with shape `(S0, ..., SN-1)` where `Si` is the size of the i-th input.
- **Indexing Modes:**
  - `'ij'` (current default): Dimensions correspond to input order.
  - `'xy'`: Swaps the first two dimensions (common for plotting).
- **Note:** The default indexing will change to `'xy'` in the future to align with NumPy.
- **Use Case:** Creating grids for functions (e.g., 3D plotting) or representing Cartesian products.

**Example:**
```python
x = torch.tensor([1, 2, 3])
y = torch.tensor([4, 5, 6])
grid_x, grid_y = torch.meshgrid(x, y, indexing='ij')
# grid_x: [[1,1,1], [2,2,2], [3,3,3]]
# grid_y: [[4,5,6], [4,5,6], [4,5,6]]
```
