`torch.meshgrid` generates coordinate grids from one-dimensional input tensors, primarily for visualization or grid-based computations. It takes N 1D tensors and returns N N-dimensional tensors, each expanded to the combined shape of all inputs.

**Key Parameters:**
- `tensors`: List of scalars or 1D tensors.
- `indexing`: 
  - `'ij'` (current default): Output dimensions match input order.
  - `'xy'`: Swaps the first two dimensions (common for plotting).

**Note:** The default indexing will change from `'ij'` to `'xy'` in a future release to align with NumPy's behavior.

**Example:**
```python
x = torch.tensor([1, 2, 3])
y = torch.tensor([4, 5, 6])
grid_x, grid_y = torch.meshgrid(x, y, indexing='ij')
# grid_x: [[1,1,1], [2,2,2], [3,3,3]]
# grid_y: [[4,5,6], [4,5,6], [4,5,6]]
```
The resulting grids represent all coordinate pairs, equivalent to a Cartesian product.
