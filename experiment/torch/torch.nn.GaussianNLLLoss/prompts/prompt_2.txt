`GaussianNLLLoss` is a PyTorch loss function that models targets as samples from Gaussian distributions, where the neural network predicts both the mean (`input`) and variance (`var`). The loss is computed as the negative log likelihood of the target under this Gaussian distribution.

**Key features:**
- **Loss formula**:  
  `loss = 0.5 * (log(max(var, eps)) + (input - target)² / max(var, eps)) + constant`  
  The constant term is included only if `full=True`.
- **Parameters**:  
  - `full`: Whether to include the constant term (default: `False`).  
  - `eps`: Small value to clamp variance for numerical stability (default: 1e-6).  
  - `reduction`: Reduction method (`'none'`, `'mean'`, or `'sum'`).
- **Input shapes**:  
  - `input` and `target`: Same shape `(N, *)`.  
  - `var`: Same shape as `input`, or broadcastable (e.g., scalar, homoscedastic shape with last dimension 1, or one fewer dimension).
- **Use cases**:  
  - Heteroscedastic uncertainty: `var` varies per prediction.  
  - Homoscedastic uncertainty: `var` is shared across predictions (broadcasted).
- **Note**: Clamping `var` with `eps` does not affect gradients.

**Example usage**:  
```python
loss_fn = nn.GaussianNLLLoss()
loss = loss_fn(input, target, var)  # var can be heteroscedastic or homoscedastic
```
