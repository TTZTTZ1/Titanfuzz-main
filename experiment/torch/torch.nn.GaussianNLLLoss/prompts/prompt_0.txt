`GaussianNLLLoss` is a PyTorch loss function that models targets as samples from Gaussian distributions, where the neural network predicts both the mean (`input`) and variance (`var`). The loss is computed as the negative log likelihood of the target given these predictions.

**Key Features:**
- **Loss Formula**:  
  \( \text{loss} = \frac{1}{2} \left( \log(\max(\text{var}, \text{eps})) + \frac{(\text{input} - \text{target})^2}{\max(\text{var}, \text{eps})} \right) + \text{const.} \)  
  The constant term is included only if `full=True`.
- **Parameters**:
  - `full`: Whether to include the constant term (default: `False`).
  - `eps`: Small value to clamp variance for numerical stability (default: 1e-6).
  - `reduction`: Reduction method (`'none'`, `'mean'`, `'sum'`; default: `'mean'`).
- **Input Shapes**:
  - `input` and `target`: Same shape `(N, *)`.
  - `var`: Can be the same shape as `input`, have a trailing dimension of 1, or one fewer dimension (for broadcasting under homoscedastic assumptions).
- **Use Cases**: Suitable for regression tasks where uncertainty estimation is needed, such as predicting both a value and its confidence.

**Example Usage**:
```python
loss_fn = nn.GaussianNLLLoss()
input = torch.randn(5, 2, requires_grad=True)
target = torch.randn(5, 2)
var = torch.ones(5, 2, requires_grad=True)  # Heteroscedastic variance
output = loss_fn(input, target, var)
output.backward()
```
