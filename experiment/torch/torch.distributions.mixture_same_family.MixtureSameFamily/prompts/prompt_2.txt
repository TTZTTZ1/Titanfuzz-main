1. **Distribution Base Class** (`torch.distributions.Distribution`)
   - Abstract base class for all probability distributions.

     - `entropy()`: Compute distribution entropy.
     - `cdf()` / `icdf()`: Cumulative/inverse cumulative distribution functions.

4. **Transforms** (`torch.distributions.transforms`)
   - Bijective transformations (e.g., `ExpTransform`, `SigmoidTransform`) to map between constrained/unconstrained spaces.
   - Used by `TransformedDistribution` to create complex distributions via transformation chains.

5. **Constraints** (`torch.distributions.constraints`)
   - Define valid parameter domains (e.g., `positive`, `simplex`).
