## **Key Components**

### **1. Distribution Base Class**
Abstract base class with essential methods:
- `sample()` / `rsample()`: Draw samples (with/without reparameterization)
- `log_prob()`: Compute log probability
- `entropy()`: Distribution entropy
- `mean`, `variance`, `mode`: Distribution statistics
- `cdf()` / `icdf()`: Cumulative/inverse cumulative distribution functions

### **2. Available Distributions**
Includes both univariate and multivariate distributions:
- **Continuous**: Normal, Beta, Gamma, Dirichlet, MultivariateNormal, etc.
- **Discrete**: Bernoulli, Categorical, Binomial, Poisson, etc.
- **Transformed**: LogNormal, HalfNormal, Gumbel (via `TransformedDistribution`)
- **Specialized**: LKJCholesky (correlation matrices), MixtureSameFamily, etc.

### **3. Transforms & Constraints**
- **Transforms**: Bijective transformations for reparameterization (Affine, Exp, Sigmoid, StickBreaking, etc.)
- **Constraints**: Define valid parameter ranges (positive, simplex, interval, etc.)
- **Registry System**: `transform_to()` and `biject_to()` for mapping between constrained/unconstrained spaces

### **4. KL Divergence**
- `kl_divergence(p, q)`: Computes KL divergence between distributions
- Extensible via `register_kl()` decorator for custom distribution pairs
