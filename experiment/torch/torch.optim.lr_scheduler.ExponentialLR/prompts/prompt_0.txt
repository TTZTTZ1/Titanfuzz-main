**ExponentialLR** is a PyTorch learning rate scheduler that multiplies the learning rate of each parameter group by a fixed factor (`gamma`) after every epoch.

**Key Features:**
- **Purpose**: Exponentially decays the learning rate each epoch.
- **Formula**: `new_lr = current_lr * gamma`
- **Initialization**: Requires an optimizer and a `gamma` value (e.g., 0.95 for 5% decay per epoch).
- **Usage**: Call `scheduler.step()` after each epoch (typically after validation).

**Example:**
```python

for epoch in range(num_epochs):
    train(...)
    validate(...)
    scheduler.step()  # Decays LR by 0.95
```

**Key Methods:**
- `step()`: Updates learning rates (call once per epoch).
- `get_last_lr()`: Returns the most recent learning rates.
- `state_dict()` / `load_state_dict()`: Save/restore scheduler state.

**Note**: The scheduler should be stepped **after** the optimizer's update (i.e., after training/validation).
