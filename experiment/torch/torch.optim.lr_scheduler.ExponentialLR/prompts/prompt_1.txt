**ExponentialLR** is a PyTorch learning rate scheduler that multiplies the learning rate of each parameter group by a fixed factor (`gamma`) every epoch.

**Key Features:**
- **Decay mechanism**: Learning rate = initial LR × gamma^epoch
- **Usage**: Instantiate with an optimizer and gamma, then call `scheduler.step()` after each epoch
- **Typical application**: Continuous exponential decay throughout training

**Basic Example:**
```python
scheduler = ExponentialLR(optimizer, gamma=0.95)
for epoch in range(num_epochs):
    # Training loop
    optimizer.step()
    scheduler.step()  # Decays LR by 0.95×
```

**Important Methods:**
- `step()`: Updates learning rates (call after optimizer.step())
- `get_last_lr()`: Returns current learning rates
- `state_dict()` / `load_state_dict()`: Save/restore scheduler state

**Note**: The scheduler operates on parameter groups, allowing different learning rates for different model parts if configured in the optimizer.
