**ExponentialLR** is a PyTorch learning rate scheduler that multiplies the learning rate by a fixed factor (`gamma`) after each epoch.

**Key Features:**
- **Purpose**: Exponentially decays the learning rate over time
- **Decay Rule**: `new_lr = current_lr * gamma` (applied every epoch)
- **Initialization**: 
  - `optimizer`: The optimizer whose learning rate will be scheduled
  - `gamma`: Multiplicative decay factor (e.g., 0.95 for 5% decay per epoch)
  - `last_epoch`: Starting epoch index (default: -1, which sets initial LR)

**Basic Usage:**
```python
scheduler = ExponentialLR(optimizer, gamma=0.95)
for epoch in range(num_epochs):
    train(...)
    validate(...)
    scheduler.step()  # Apply decay after each epoch
```

**Important Methods:**
- `step()`: Updates learning rates (call after optimizer.step())
- `get_last_lr()`: Returns current learning rates for all parameter groups
- `state_dict()` / `load_state_dict()`: Save/restore scheduler state

**Note**: The decay is applied uniformly to all parameter groups in the optimizer, and the learning rate decreases exponentially according to: `lr = initial_lr * (gamma^epoch)`
