`torch.linalg.cond` computes the condition number of a matrix (or batch of matrices) with respect to a specified matrix norm, measuring the numerical stability of solving linear systems involving that matrix.

**Key points:**
- **Formula**: For matrix \( A \), condition number \( \kappa(A) = \|A\|_p \|A^{-1}\|_p \).
- **Input**: Tensor `A` of shape `(*, m, n)` for norms 2/-2, or `(*, n, n)` for other norms.
- **Norms supported**:
  - `None` or `2`: largest singular value (default).
  - `-2`: smallest singular value.
  - `'fro'`, `'nuc'`, `inf`, `-inf`, `1`, `-1`: requires square invertible matrices.
- **Output**: Real-valued tensor, even for complex inputs.
- **Batch support**: Processes multiple matrices in batch dimensions.
- **Device note**: For norms `('fro', 'nuc', inf, -inf, 1, -1)`, synchronizes CUDA with CPU.

**Example**:
```python
A = torch.tensor([[1., 0, -1], [0, 1, 0], [1, 0, 1]])
cond = torch.linalg.cond(A, 2)  # Condition number using 2-norm
```
