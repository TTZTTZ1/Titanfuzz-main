`torch.linalg.cond` computes the condition number of a matrix (or batch of matrices) relative to a specified matrix norm, measuring the numerical stability of solving linear systems involving that matrix.

**Key points:**
- **Formula:** For matrix \( A \), condition number \( \kappa(A) = \|A\|_p \|A^{-1}\|_p \).
- **Norms supported:** `p` can be `None` (default, 2-norm), `'fro'`, `'nuc'`, `inf`, `-inf`, `1`, `-1`, `2`, or `-2`.
- **Input shape:** 
  - For `p` in `('fro', 'nuc', inf, -inf, 1, -1)`: input must be square and invertible.
  - For `p` in `(2, -2)`: any shape is allowed.
- **Output:** Returns a real-valued tensor (even for complex inputs).
- **Batch support:** Works on batched inputs, returning condition numbers per matrix in the batch.
- **Device note:** For certain norms (`'fro'`, `'nuc'`, `inf`, `-inf`, `1`, `-1`), CUDA inputs synchronize with the CPU.

**Example:**
```python
A = torch.tensor([[1., 0, -1], [0, 1, 0], [1, 0, 1]])
torch.linalg.cond(A, 2)  # Output: tensor([1.4142])
```
