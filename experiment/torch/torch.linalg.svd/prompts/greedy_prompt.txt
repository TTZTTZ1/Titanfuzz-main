`torch.linalg.svd` computes the singular value decomposition (SVD) of a matrix (or batch of matrices) into three components:  
- **U** (left singular vectors)  
- **S** (singular values, real and in descending order)  
- **Vh** (conjugate transpose of right singular vectors)  

**Key parameters:**  
- `full_matrices` (bool): If `True` (default), returns full-sized U and Vh; if `False`, returns reduced forms.  
- `driver` (str, CUDA only): Algorithm choice (`gesvd`, `gesvdj`, `gesvda`, or `None` for auto-selection).  

**Notes:**  
- Works with float, double, cfloat, and cdouble dtypes, and supports batched inputs.  
- Output is a named tuple `(U, S, Vh)`.  
- Gradients can be unstable for repeated or near-zero singular values.  
- For only singular values, use `torch.linalg.svdvals` instead.
