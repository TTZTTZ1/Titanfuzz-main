---GENERAL SUMMARY START---
`MaxPool1d` performs 1D max pooling over an input tensor, sliding a window across the last dimension to output the maximum value within each window.  

**Key parameters:**  
- `kernel_size`: Window size (>0)  
- `stride`: Step size for sliding the window (default = `kernel_size`)  
- `padding`: Adds implicit negative infinity padding on both sides (0 ≤ padding ≤ kernel_size/2)  
- `dilation`: Spacing between window elements (>0)  
- `return_indices`: If `True`, returns max indices for use with `MaxUnpool1d`  
- `ceil_mode`: Uses ceiling instead of floor for output length calculation  

**Input shape:** `(N, C, L_in)` or `(C, L_in)`  
**Output shape:** `(N, C, L_out)` or `(C, L_out)`  

**Output length formula** depends on `ceil_mode` (see documentation for exact expressions).  

**Example:**  
```python
m = nn.MaxPool1d(3, stride=2)  # window size 3, stride 2
input = torch.randn(20, 16, 50)
output = m(input)  # shape: (20, 16, 24) with default ceil_mode=False
```
