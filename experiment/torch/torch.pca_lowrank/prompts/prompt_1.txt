**Purpose:**  
Performs linear Principal Component Analysis (PCA) on a low-rank matrix, batches of matrices, or a sparse matrix. It returns an approximate singular value decomposition (SVD) of a (centered) data matrix.

**Returns:**  
A namedtuple `(U, S, V)` representing:
- `U`: m × q matrix (left singular vectors)
- `S`: q-length vector (singular values)
- `V`: n × q matrix (right singular vectors / principal directions)

**Key Notes:**
- The decomposition approximates the centered matrix \( A \approx U \text{diag}(S) V^H \).
- `V` contains the principal directions.
- \( S^2 / (m-1) \) gives eigenvalues of the covariance matrix when `center=True`.
- `A @ V[:, :k]` projects data onto the top `k` principal components.
- Output dimensions depend on `q` (estimated rank) rather than full dimensions.

**Parameters:**
- `A`: Input tensor of shape `(*, m, n)`.
- `q`: Overestimated rank (default: `min(6, m, n)`).
- `center`: If `True`, centers the data by subtracting the mean.
- `niter`: Number of subspace iterations (default: 2).

**Use Case:**  
Efficient low-rank PCA approximation for large or sparse datasets, suitable for dimensionality reduction and feature extraction.
