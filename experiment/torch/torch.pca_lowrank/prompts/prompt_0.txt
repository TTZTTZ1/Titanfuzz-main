`torch.pca_lowrank` performs linear Principal Component Analysis (PCA) on a low‑rank matrix, batches of matrices, or a sparse matrix. It returns a namedtuple `(U, S, V)` that approximates the singular value decomposition of a (centered) data matrix \( A \), where:

- **V** contains the principal directions.
- **S** relates to the eigenvalues of the covariance matrix when `center=True`.
- **U** and **V** are sized according to the parameter `q` (an overestimated rank), not the full dimensions of \( A \).

**Key parameters:**
- `A`: input tensor of shape `(*, m, n)`.
- `q`: optional overestimated rank (default `min(6, m, n)`).
- `center`: if `True`, centers the data before decomposition (default).
- `niter`: number of subspace iterations (default 2).

**Use case:**  
Project data to the first \( k \) principal components via `matmul(A, V[:, :k])`.  

**Note:** For reproducible results, set the random seed. The algorithm is based on randomized linear algebra methods (see Halko et al., 2009).
