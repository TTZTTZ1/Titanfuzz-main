The `torch.nn.MultiheadAttention` module implements the multi-head attention mechanism from the Transformer architecture. It allows the model to attend to different parts of the input sequence simultaneously across multiple attention heads.

**Key Features:**
- **Input/Output Shapes:** Supports both batch-first (`batch_first=True`) and sequence-first (`batch_first=False`) formats.
- **Masks:** Accepts `key_padding_mask` for padding tokens and `attn_mask` for attention restrictions (causal or custom masks).
- **Optimized Path:** Uses `scaled_dot_product_attention` for faster inference when conditions are met (e.g., self-attention, no gradients, etc.).
- **Flexible Dimensions:** Allows separate dimensions for keys (`kdim`) and values (`vdim`).

**Main Parameters:**
- `embed_dim`: Total embedding dimension.
- `num_heads`: Number of attention heads (must divide `embed_dim`).
- `dropout`: Dropout probability on attention weights.
- `batch_first`: If `True`, input/output tensors are (batch, seq, feature); default is (seq, batch, feature).

**Forward Pass:**
- Takes `query`, `key`, `value` tensors (often same for self-attention).
- Returns `attn_output` and optionally `attn_output_weights`.
- Use `need_weights=False` for optimized performance.

**Typical Usage:**
```python
attn = nn.MultiheadAttention(embed_dim=512, num_heads=8, batch_first=True)
output, weights = attn(query, key, value)
```
