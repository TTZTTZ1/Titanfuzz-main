The `torch.nn.MultiheadAttention` module implements the multi-head attention mechanism from the Transformer architecture. It allows the model to jointly attend to information from different representation subspaces.

**Key Features:**
- Implements the standard multi-head attention formula: `MultiHead(Q,K,V) = Concat(head₁,...,headₕ)Wᵒ`
- Supports optimized `scaled_dot_product_attention()` for better performance
- Provides fastpath inference under specific conditions (self-attention, batched inputs, no gradients, etc.)
- Accepts optional masks for padding and attention control
- Supports both batched and unbatched inputs with `batch_first` parameter

**Main Parameters:**
- `embed_dim`: Total embedding dimension
- `num_heads`: Number of attention heads (embed_dim is split across heads)
- `batch_first`: Controls input/output tensor shape (batch, seq, feature) vs (seq, batch, feature)

**Usage:**
```python
multihead_attn = nn.MultiheadAttention(embed_dim, num_heads)
attn_output, attn_output_weights = multihead_attn(query, key, value)
```

The module returns attention outputs and optionally attention weights, with shape depending on batching and averaging settings.
