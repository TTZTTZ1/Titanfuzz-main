`torch.transpose` swaps two specified dimensions of a tensor. It returns a transposed view for strided tensors (sharing storage) but a new tensor for sparse tensors. **Parameters:**
- `input` (Tensor): Input tensor.
- `dim0` (int): First dimension to swap.
- `dim1` (int): Second dimension to swap.

**Example:**

>>> x = torch.randn(2, 3)
>>> torch.transpose(x, 0, 1)  # Swaps rows and columns
