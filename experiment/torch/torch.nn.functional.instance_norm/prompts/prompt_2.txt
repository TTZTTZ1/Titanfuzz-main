`torch.nn.functional.instance_norm` applies **instance normalization** independently for each channel in every sample of a batch. It normalizes using per-sample, per-channel statistics and can optionally use learnable affine parameters (`weight` and `bias`).  

**Key parameters:**  
- `input`: Input tensor.  
- `running_mean`, `running_var`: Optional running statistics (used in evaluation mode if `use_input_stats=False`).  
- `weight`, `bias`: Optional learnable affine transformation parameters.  
- `use_input_stats`: If `True`, uses input’s statistics; if `False`, uses provided running statistics.  
- `momentum`: Momentum for updating running statistics.  
- `eps`: Small value for numerical stability.  

**Returns:** Normalized tensor of the same shape as input.  

This function is the functional counterpart to `InstanceNorm1d`, `InstanceNorm2d`, and `InstanceNorm3d`.
