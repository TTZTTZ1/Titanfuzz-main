`torch.column_stack` horizontally concatenates a sequence of tensors. It automatically reshapes any 0-D or 1-D input tensors into columns (size `[n, 1]`) before stacking them side-by-side. This function is equivalent to `torch.hstack`, but with the added reshaping behavior for lower-dimensional inputs.

**Key behavior:**
- Input tensors must have the same number of rows.
- Lower-dimensional tensors are treated as single columns.
- Output is a 2-D tensor combining all inputs horizontally.

**Example:**
```python
>>> a = torch.tensor([1, 2, 3])
>>> b = torch.tensor([4, 5, 6])
>>> torch.column_stack((a, b))
tensor([[1, 4],
        [2, 5],
        [3, 6]])
```
