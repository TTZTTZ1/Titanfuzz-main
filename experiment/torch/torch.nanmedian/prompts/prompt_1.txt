`torch.nanmedian` computes the median of a tensor while ignoring NaN values. It has two forms:

1. **Global median**: Returns the median of all elements, ignoring NaNs. If all values are NaN, returns NaN.
2. **Dimension-wise median**: When a dimension is specified, it returns a namedtuple `(values, indices)` with medians and their indices per slice, ignoring NaNs in each slice.

**Key difference from `torch.median`**:  
- `torch.median` returns NaN if any NaN is present.  
- `torch.nanmedian` ignores NaNs and computes the median from valid numbers.

**Example**:
```python
>>> a = torch.tensor([1, float('nan'), 3, 2])
>>> a.nanmedian()
tensor(2.)
```
