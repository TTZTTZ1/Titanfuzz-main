**ELU (Exponential Linear Unit)** is an activation function in PyTorch that applies element-wise non-linearity, defined as:

- **For positive inputs**: returns the input unchanged (`x`).
- **For non-positive inputs**: returns `α * (exp(x) - 1)`.

**Key parameters:**
- `alpha` (float, default=1.0): controls the value for negative inputs.
- `inplace` (bool, default=False): if True, modifies input directly to save memory.

**Characteristics:**
- Handles any input shape (same input/output shape).
- Helps mitigate vanishing gradients and speeds up learning compared to ReLU.
- Smooth for negative values, unlike ReLU.

**Example usage:**
