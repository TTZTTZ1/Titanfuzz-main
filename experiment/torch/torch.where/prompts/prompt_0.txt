# First, here is a general summary of the API's functionality and usage:
---GENERAL SUMMARY START---
`torch.where` selects elements from two tensors based on a condition. It returns a tensor where each element is taken from `input` if the corresponding `condition` is True, otherwise from `other`. All inputs must be broadcastable.

**Parameters:**
- `condition` (BoolTensor): Determines selection source.
- `input` (Tensor/scalar): Values chosen where `condition` is True.
- `other` (Tensor/scalar): Values chosen where `condition` is False.

**Returns:** A tensor with the broadcasted shape of the inputs.

**Example:**
```python
>>> torch.where(x > 0, x, y)  # Select from x if condition holds, else from y
```

**Special form:** `torch.where(condition)` without `input`/`other` returns the indices where `condition` is True (equivalent to `torch.nonzero(condition, as_tuple=True)`).
---GENERAL SUMMARY END---
