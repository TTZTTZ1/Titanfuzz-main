`torch.where` is a PyTorch function that selects elements from one of two input tensors based on a condition tensor. It operates element-wise: where `condition` is True (nonzero), the output takes the value from `input`; otherwise, it takes the value from `other`. All inputs must be broadcastable to a common shape.

**Key features:**
- **Three-argument form:** `torch.where(condition, input, other)` → returns a tensor with selected values.
- **Single-argument form:** `torch.where(condition)` → equivalent to `torch.nonzero(condition, as_tuple=True)`, returning a tuple of coordinate tensors for True entries.
- **Supports scalars:** Both `input` and `other` can be scalars, which are broadcast to match the condition's shape.

**Example:**
```python
>>> torch.where(x > 0, x, 0.0)  # Keeps positive values, replaces others with 0
```
