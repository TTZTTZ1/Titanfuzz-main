`torch.nn.Conv1d` performs 1D convolution on an input signal with multiple channels. It slides a kernel across the input sequence to extract features, producing an output with potentially different channel count and length.

**Key Parameters:**
- `in_channels`: Number of input channels
- `out_channels`: Number of output channels
- `kernel_size`: Size of the convolution filter
- `stride`: Step size for kernel movement (default: 1)
- `padding`: Added zeros on both sides (default: 0, or 'same'/'valid')
- `dilation`: Spacing between kernel elements (default: 1)
- `groups`: Controls input-output connections (default: 1)
- `bias`: Whether to add a learnable bias (default: True)

**Output Length Formula:**
\[
L_{out} = \left\lfloor\frac{L_{in} + 2 \times \text{padding} - \text{dilation} \times (\text{kernel\_size} - 1) - 1}{\text{stride}} + 1\right\rfloor
\]

**Special Modes:**
- `groups=1`: Standard convolution (all inputs connect to all outputs)
- `groups=in_channels`: Depthwise convolution (each input channel has separate filters)

**Example:**
```python
conv = nn.Conv1d(16, 33, 3, stride=2)  # 16→33 channels, kernel size 3
input = torch.randn(20, 16, 50)        # (batch, channels, length)
output = conv(input)                   # Output shape: (20, 33, 24)
```
