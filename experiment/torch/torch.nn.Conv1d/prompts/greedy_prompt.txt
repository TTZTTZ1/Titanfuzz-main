`torch.nn.Conv1d` performs a 1D convolution over an input signal with multiple channels. It applies a learnable filter to extract features from sequential data, such as time series or audio.

**Key parameters:**
- `in_channels`: Number of input channels
- `out_channels`: Number of output channels (number of filters)
- `kernel_size`: Size of the convolving kernel
- `stride`: Step size for convolution (default: 1)
- `padding`: Amount of padding added to input (default: 0)
- `dilation`: Spacing between kernel elements (default: 1)
- `groups`: Controls input-output connections (default: 1)
- `bias`: Whether to add a learnable bias (default: True)

**Input/Output shape:**
- Input: `(N, C_in, L_in)` or `(C_in, L_in)`
- Output: `(N, C_out, L_out)` or `(C_out, L_out)`
- Output length: `L_out = floor((L_in + 2×padding - dilation×(kernel_size-1) - 1)/stride + 1)`

**Special features:**
- Supports depthwise convolution when `groups = in_channels`
- Multiple padding modes: 'zeros', 'reflect', 'replicate', 'circular'
- Supports complex data types
- 'same' padding maintains input length (stride must be 1)

**Example:**
```python
conv = nn.Conv1d(16, 33, 3, stride=2)  # 16 input channels, 33 output channels, kernel size 3
input = torch.randn(20, 16, 50)        # Batch of 20, 16 channels, length 50
output = conv(input)                   # Output shape: (20, 33, 24)
```
