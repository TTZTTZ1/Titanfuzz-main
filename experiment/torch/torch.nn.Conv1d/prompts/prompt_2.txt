`torch.nn.Conv1d` is a PyTorch module that performs 1D convolution over an input signal with multiple channels. It slides a kernel across the input sequence to extract features, commonly used for time-series or sequence data.

**Key Parameters:**
- `in_channels`: Number of input channels.
- `out_channels`: Number of output channels (number of filters).
- `kernel_size`: Size of the convolving kernel.
- `stride`: Step size of the kernel (default: 1).
- `padding`: Amount of padding added to input sides (default: 0). Can be an integer, tuple, or `'valid'` (no padding) / `'same'` (output length matches input, stride must be 1).
- `dilation`: Spacing between kernel elements (default: 1).
- `groups`: Controls input-output connections (default: 1). If `groups = in_channels`, it becomes a depthwise convolution.
- `bias`: Whether to add a learnable bias (default: `True`).
- `padding_mode`: Type of padding (`'zeros'`, `'reflect'`, `'replicate'`, `'circular'`).

**Output Length Formula:**
\[
L_{\text{out}} = \left\lfloor \frac{L_{\text{in}} + 2 \times \text{padding} - \text{dilation} \times (\text{kernel\_size} - 1) - 1}{\text{stride}} + 1 \right\rfloor
\]

**Example:**
```python
conv = nn.Conv1d(16, 33, kernel_size=3, stride=2)
input = torch.randn(20, 16, 50)  # (batch, channels, length)
output = conv(input)  # Output shape: (20, 33, L_out)
```
