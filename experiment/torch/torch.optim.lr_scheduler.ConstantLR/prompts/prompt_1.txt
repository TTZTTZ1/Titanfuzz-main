**ConstantLR** is a PyTorch learning rate scheduler that multiplies the learning rate by a constant factor for a specified number of initial steps, then restores it to the original value.

**Key Features:**
- **Factor scaling:** Multiplies the learning rate by `factor` for the first `total_iters` steps.
- **Automatic restoration:** After `total_iters` steps, the learning rate returns to its original value (scaled by `1/factor`).
- **Compatibility:** Works alongside other learning rate modifications.

**Parameters:**
- `optimizer`: The optimizer to wrap.
- `factor` (float, default=1/3): Multiplier applied to the learning rate.
- `total_iters` (int, default=5): Number of steps to apply the factor.
- `last_epoch` (int, default=-1): Starting epoch index.

**Usage Example:**
```python
scheduler = ConstantLR(optimizer, factor=0.5, total_iters=40)
for epoch in range(100):
    train(...)
    validate(...)
    scheduler.step()  # Call after optimizer.step()
```

**Behavior:**
- For epochs 0-39: `lr = initial_lr * factor`
- From epoch 40 onward: `lr = initial_lr`

**Common Use Case:** Implements a "warm-up" phase where the learning rate starts lower and gradually increases to its base value over `total_iters` steps.
